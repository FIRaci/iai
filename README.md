# IAI — Cẩm nang AI Tools & Dev Tools

Comprehensive Vietnamese guide to AI coding tools and development tools for Windows 11 users.

**Site:** https://iai-flax-sigma.vercel.app

## Tech Stack

React 19 + TypeScript + Vite + Tailwind CSS v4 + shadcn/ui + MDX

## Content

- **AI Tools** — Claude Code, ClaudeKit, Cursor, GitHub Copilot, Devin, Windsurf, OpenClaw
- **Dev Tools** — Node.js, Git, VS Code
- **Comparisons** — AI coding tools comparison table

## Development

```bash
pnpm install
pnpm dev
pnpm build
```
