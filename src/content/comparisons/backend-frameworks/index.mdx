## Khi nào dùng gì

### Express
- **Mạnh nhất**: Đơn giản, ecosystem lớn nhất
- **Phù hợp**: Quick APIs, legacy projects
- **Điểm yếu**: Không type-safe, callback hell

### FastAPI
- **Mạnh nhất**: Auto docs, type-safe, async
- **Phù hợp**: Python APIs, ML backends
- **Điểm yếu**: Python ecosystem limitations

### Django
- **Mạnh nhất**: Full-stack, ORM, admin panel
- **Phù hợp**: Full web apps, rapid development
- **Điểm yếu**: Heavy, monolithic

### NestJS
- **Mạnh nhất**: Architecture, TypeScript, enterprise
- **Phù hợp**: Large teams, enterprise apps
- **Điểm yếu**: Boilerplate, learning curve

### Hono
- **Mạnh nhất**: Edge runtime, ultra-fast
- **Phù hợp**: Cloudflare Workers, edge computing
- **Điểm yếu**: Ecosystem nhỏ

### Fastify
- **Mạnh nhất**: Performance, plugin system
- **Phù hợp**: High-throughput Node.js APIs
- **Điểm yếu**: Ít phổ biến hơn Express
