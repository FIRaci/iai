## Tổng quan

<Note type="info">
**LLM Runtimes** cho phép chạy mô hình ngôn ngữ lớn cục bộ hoặc trên server. Tùy vào nhu cầu về hiệu suất, dễ sử dụng và khả năng tương thích API mà bạn chọn runtime phù hợp. So sánh 6 công cụ phổ biến nhất.
</Note>

## Bảng so sánh

<Note type="info">
**Ollama** - Dễ dùng nhất - Pull & run models với 1 lệnh - Hỗ trợ Llama, Mistral, Gemma, Qwen - API tương thích OpenAI - Phù hợp: dev cá nhân, prototyping, local development.
</Note>

<Note type="info">
**LM Studio** - GUI đẹp, trực quan - Tải model từ Hugging Face - Chat interface + Local API server - Hỗ trợ GPU acceleration (CUDA/Metal) - Phù hợp: non-technical users, exploration.
</Note>

<Note type="info">
**vLLM** - Hiệu suất inference cao nhất - PagedAttention - Continuous batching - Hỗ trợ đa GPU, tensor parallelism - Phù hợp: production deployment, API server scale lớn.
</Note>

<Note type="info">
**llama.cpp** - Chạy trên CPU hiệu quả - GGUF format - Quantization (Q4, Q5, Q8) - Nhẹ, không cần GPU - Phù hợp: máy cấu hình thấp, edge devices, embedded.
</Note>

<Note type="info">
**LocalAI** - OpenAI API drop-in replacement - Hỗ trợ LLM, Whisper, Image gen - Docker deployment - REST API tương thích - Phù hợp: self-hosted AI platform, muốn thay thế OpenAI.
</Note>

<Note type="info">
**text-generation-webui (oobabooga)** - Giao diện web đầy đủ - Hỗ trợ nhiều loader (Transformers, ExLlama, AutoGPTQ) - LoRA finetuning, chat, instruct modes - Phù hợp: power users, research, fine-tuning.
</Note>

## So sánh hiệu suất

| Runtime | Setup | Performance | GPU | API | Model format |
|---------|-------|-------------|-----|-----|-------------|
| Ollama | Rất dễ | Tốt | CUDA/Metal | OpenAI-like | Modelfile (GGUF) |
| LM Studio | Dễ | Tốt | CUDA/Metal | OpenAI-like | GGUF |
| vLLM | Khó | Xuất sắc | CUDA đa GPU | OpenAI-like | Safetensors |
| llama.cpp | TB | Tốt (CPU) | CPU/CUDA | Custom/OpenAI | GGUF |
| LocalAI | TB | TB | CUDA/Metal | OpenAI drop-in | Nhiều format |
| oobabooga | TB | Tốt | CUDA | Custom/Extensions | Nhiều format |

<Note type="tip">
**Khuyên dùng:** Ollama cho local dev nhanh nhất. vLLM cho production server. llama.cpp cho máy không GPU. LM Studio cho người mới bắt đầu. LocalAI để thay thế OpenAI API tự host.
</Note>
