## Tổng quan

<Note type="info">
**Vector Databases** lưu trữ và tìm kiếm vector embeddings cho ứng dụng RAG, semantic search và AI agents. Mỗi database có điểm mạnh riêng về scalability, filtering, và pricing. So sánh 6 vector database hàng đầu.
</Note>

## Bảng so sánh

<Note type="info">
**Pinecone** - Managed cloud - Serverless index - Giá: $0.25/index/hour - Filter metadata mạnh - Tích hợp sẵn với LangChain, LlamaIndex - Phù hợp: production RAG, không muốn tự host.
</Note>

<Note type="info">
**Qdrant** - Managed + Self-hosted - Rust, siêu nhanh - Filter phức tạp (nested payload) - Giá: $0.40/GB/month - Phù hợp: cần filtering chi tiết, tự do kiểm soát infra.
</Note>

<Note type="info">
**Weaviate** - Managed + Self-hosted - Graph + vector search - Hybrid search (keyword + semantic) - Tích hợp module AI (OpenAI, Cohere) - Phù hợp: hybrid search, knowledge graph apps.
</Note>

<Note type="info">
**Milvus** - Open source - Cloud-native (Kubernetes) - GPU acceleration - Hiệu suất cao nhất với billion-scale - Phù hợp: enterprise scale, billion+ vectors, on-premise.
</Note>

<Note type="info">
**Chroma** - Open source, nhẹ nhất - Python-native - Embedding + DB trong 1 thư viện - Lưu local hoặc in-memory - Phù hợp: prototyping, research, local dev.
</Note>

<Note type="info">
**FAISS (Meta)** - Không phải database - Thư viện similarity search - Hiệu suất cực cao (GPU) - Lưu trên memory/disk - Phù hợp: research, batch processing, không cần persistence.
</Note>

## So sánh tính năng

| DB | Managed | Self-host | Filter | Pricing | Scale |
|----|---------|-----------|--------|---------|-------|
| Pinecone | Có | Không | Mạnh | Pay-per-index | Triệu |
| Qdrant | Có | Có | Rất mạnh | $0.40/GB | Triệu |
| Weaviate | Có | Có | Hybrid | $0.25/GB | Triệu |
| Milvus | Có | Có | Mạnh | Open source | Tỷ |
| Chroma | Không | Có | Cơ bản | Free | Trăm nghìn |
| FAISS | Không | Có | Không | Free | Tỷ |

<Note type="tip">
**Khuyên dùng:** Pinecone cho RAG production nhanh nhất. Qdrant cho custom filter phức tạp. Milvus cho billion-scale enterprise. Chroma cho local prototyping. FAISS cho batch similarity search không cần DB.
</Note>
