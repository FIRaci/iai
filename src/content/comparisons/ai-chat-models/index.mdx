## Khi nào dùng gì

### ChatGPT (GPT-4o)
- **Mạnh nhất**: Đa năng, coding, viết lách
- **Phù hợp**: General purpose, plugin ecosystem
- **Điểm yếu**: Giới hạn free tier

### Claude (3.7 Sonnet)
- **Mạnh nhất**: Phân tích, viết lách, reasoning
- **Phù hợp**: Code review, document analysis, creative writing
- **Điểm yếu**: Không có image generation

### Gemini (2.5 Pro)
- **Mạnh nhất**: Context window 1M, đa phương thức
- **Phù hợp**: Xử lý tài liệu dài, video analysis
- **Điểm yếu**: Coding kém hơn GPT-4/Claude

### DeepSeek (V3)
- **Mạnh nhất**: Giá rẻ, coding tốt
- **Phù hợp**: Budget-conscious, coding tasks
- **Điểm yếu**: Ít tính năng phụ trợ
