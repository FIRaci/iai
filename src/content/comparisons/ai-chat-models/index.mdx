## Tổng quan

<Note type="info">
**AI Chat Models** là trợ lý AI đàm thoại hàng đầu hiện nay. Mỗi model có thế mạnh riêng về lập trình, sáng tạo nội dung, phân tích dữ liệu hay lý luận phức tạp. So sánh dưới đây giúp bạn chọn đúng công cụ cho nhu cầu.
</Note>

## Bảng so sánh

<Note type="info">
**ChatGPT (OpenAI)** - $20/tháng (Plus) - Cửa sổ 128K token - Hỗ trợ đa phương thức (text, image, voice) - Mạnh về code, creative writing, reasoning - Plugin và GPT Store phong phú.
</Note>

<Note type="info">
**Claude (Anthropic)** - $20/tháng (Pro) - Cửa sổ 200K token - Hỗ trợ image, PDF, code - Mạnh về lập trình, phân tích tài liệu dài, safety - Artifacts cho phép xem trước code và design.
</Note>

<Note type="info">
**Gemini (Google)** - Miễn phí (bản basic) - Cửa sổ 1M token (Advanced) - Đa phương thức mạnh nhất (video, image, audio) - Tích hợp Google ecosystem - Mạnh về tìm kiếm và xử lý thông tin.
</Note>

<Note type="info">
**Grok (xAI)** - $16/tháng (SuperGrok) - Cửa sổ 128K token - Hỗ trợ real-time từ X/Twitter - Mạnh về phân tích xu hướng, tin tức - Tính cách hài hước, không filter.
</Note>

<Note type="info">
**DeepSeek** - Miễn phí (Web) - Cửa sổ 128K token - Hỗ trợ text, file upload - Mã nguồn mở - Mạnh về toán học, lập trình, lý luận - Giá API rẻ nhất thị trường.
</Note>

<Note type="info">
**Perplexity** - $20/tháng (Pro) - Cửa sổ không giới hạn - Hỗ trợ đa phương thức + tìm kiếm web - Chuyên về research với citation - Mạnh về tra cứu thông tin chính xác.
</Note>

## So sánh nhanh

| Model | Giá Pro | Cửa sổ | Multimodal | Điểm mạnh |
|-------|---------|--------|------------|-----------|
| ChatGPT | $20 | 128K | Image, Voice | Đa năng, hệ sinh thái lớn |
| Claude | $20 | 200K | Image, PDF | Code, phân tích tài liệu dài |
| Gemini | Free/$20 | 1M | Video, Audio, Image | Đa phương thức, ngữ cảnh siêu dài |
| Grok | $16 | 128K | Image | Real-time news, không filter |
| DeepSeek | Free | 128K | Text, File | Giá rẻ, reasoning mạnh |
| Perplexity | $20 | Unlimited | Web + multimodal | Research, citation chính xác |

<Note type="tip">
**Khuyên dùng:** Claude cho lập trình và phân tích docs dài. Gemini cho research đa phương thức. DeepSeek cho budget thấp. Perplexity cho nghiên cứu học thuật.
</Note>
