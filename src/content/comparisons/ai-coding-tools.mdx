## Tổng quan

<Note type="info">
**AI Coding Tools** giúp lập trình viên tăng tốc độ code gấp 2-3 lần nhờ AI gợi ý, hoàn thiện code và debug tự động. Dưới đây là so sánh chi tiết giữa các công cụ phổ biến nhất hiện nay.
</Note>

## Bảng so sánh chi tiết

<Note type="info">
**Cursor** - $20/tháng - IDE riêng biệt dựa trên VS Code - Hỗ trợ Claude, GPT-4, Gemini - Tính năng Composer, Chat inline, Agent mode - Phù hợp: dev chuyên nghiệp cần AI-native IDE.
</Note>

<Note type="info">
**GitHub Copilot** - $10/tháng - Extension trên VS Code, JetBrains, Neovim - GPT-4o, Claude - Tab autocomplete, Chat - Phù hợp: đa số lập trình viên, tích hợp sẵn trong GitHub.
</Note>

<Note type="info">
**Claude Code** - $20/tháng (Pro) - CLI native - Claude 3.5/4 - Agent tự động đọc/ghi file, chạy lệnh - Phù hợp: terminal-first dev, codebase lớn cần refactor.
</Note>

<Note type="info">
**Windsurf** - $15/tháng - IDE riêng (Codeium) - Cascade engine đa模型 - Tab completion, Supercomplete, Agent - Phù hợp: dev muốn AI hiểu ngữ cảnh sâu.
</Note>

<Note type="info">
**Continue.dev** - Miễn phí (open source) - Extension VS Code, JetBrains - BYOM (bring your own model) - Chat, Edit, Agent - Phù hợp: dev muốn tự do chọn model, local LLM.
</Note>

<Note type="info">
**Amazon Q Developer** - Miễn phí (Basic) - Extension trên VS Code, JetBrains - Amazon Bedrock models - Security scanning, code review - Phù hợp: AWS ecosystem dev, cần bảo mật.
</Note>

## Khi nào nên chọn?

<Note type="tip">
**Khuyên dùng:** Chọn Cursor nếu cần IDE AI-native mạnh mẽ. Copilot cho team dùng GitHub. Claude Code cho tác vụ phức tạp cần agent tự động. Continue nếu muốn dùng local LLM miễn phí.
</Note>
