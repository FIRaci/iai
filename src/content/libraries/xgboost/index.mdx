## Tổng quan

<Note type="info">
**XGBoost** (Extreme Gradient Boosting) là thư viện gradient boosting tối ưu hóa cho speed và performance. Dẫn đầu nhiều Kaggle competitions, hỗ trợ classification, regression, và ranking.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install xgboost`} />

## Sử dụng

<CodeBlock language="python" code={`import xgboost as xgb
from sklearn.datasets import load_boston
from sklearn.model_selection import train_test_split

X, y = load_boston(return_X_y=True)
X_train, X_test, y_train, y_test = train_test_split(X, y)

model = xgb.XGBRegressor(n_estimators=100, learning_rate=0.1)
model.fit(X_train, y_train)
predictions = model.predict(X_test)`} />
