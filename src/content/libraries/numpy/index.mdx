## Tổng quan

<Note type="info">
**NumPy** là foundational package cho numerical computing trong Python. Cung cấp arrays, matrices, và mathematical functions.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install numpy`} />
