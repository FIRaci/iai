## Tổng quan

<Note type="info">
**Kedro** là open-source framework cho data pipeline development của QuantumBlack (McKinsey). Cung cấp project template, data catalog, pipeline abstraction, và modular code organization cho data science projects.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install kedro kedro-viz`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Tao project
kedro new --starter=pandas-iris

# Chay pipeline
kedro run

# Visualize pipeline
kedro viz`} />
