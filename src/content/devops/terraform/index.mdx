## Tổng quan

<Note type="info">
**Terraform** là Infrastructure as Code (IaC) tool của HashiCorp, cho phép định nghĩa và quản lý hạ tầng cloud bằng cấu hình khai báo. Hỗ trợ AWS, Azure, GCP, và 3000+ providers.
</Note>

## Cài đặt

### Cài đặt qua winget

<CodeBlock language="powershell" code={`winget install Hashicorp.Terraform`} />

### Kiểm tra cài đặt

<CodeBlock language="powershell" code={`terraform --version`} />

### Cài đặt autocompletion

<CodeBlock language="powershell" code={`terraform -install-autocomplete`} />

## Sử dụng

### Terraform workflow cơ bản

<CodeBlock language="powershell" code={`# Khởi tạo project
terraform init

# Format & validate
terraform fmt
terraform validate

# Xem kế hoạch thay đổi
terraform plan

# Apply thay đổi
terraform apply

# Xóa tài nguyên
terraform destroy`} />

### File cấu hình mẫu (main.tf)

 <CodeBlock language="hcl" code={`
terraform {
  required_providers {
    aws = {
      source  = "hashicorp/aws"
      version = "~> 5.0"
    }
  }
}

provider "aws" {
  region = "ap-southeast-1"
}

resource "aws_instance" "web" {
  ami           = "ami-0c55b159cbfafe1f0"
  instance_type = "t2.micro"

  tags = {
    Name = "web-server"
  }
}
`} />

### Quản lý state

<CodeBlock language="powershell" code={`# Xem state
terraform state list

# Di chuyển resource
terraform state mv aws_instance.web aws_instance.web-v2

# Xóa khỏi state
terraform state rm aws_instance.web`} />

## Lưu ý

<Note type="warning">
File `terraform.tfstate` chứa secrets (plaintext passwords, keys). Không commit vào git. Dùng remote backend (S3, Azure Storage) để lưu state an toàn.
</Note>

<Note type="tip">
Dùng `terraform workspace` để quản lý nhiều môi trường (dev/staging/prod) với cùng codebase.
</Note>
