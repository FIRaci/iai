## Tổng quan

<Note type="info">
**n8n** là workflow automation tool với visual node-based editor. Kết nối 200+ services và APIs.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install n8n -g`} />

## Sử dụng

<CodeBlock language="powershell" code={`n8n start`} />

Mở `http://localhost:5678` để tạo workflows.
