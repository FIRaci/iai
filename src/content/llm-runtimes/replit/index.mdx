## Tổng quan

<Note type="info">
**Replit Ghostwriter CLI** là AI coding assistant từ Replit, hỗ trợ code completion, explanation, và generation ngay trong terminal.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -g @replit/ghostwriter-cli`} />

## Sử dụng

<CodeBlock language="powershell" code={`ghostwriter explain "function calculateTotal(items)"
ghostwriter generate "Python function to sort a list"
ghostwriter complete "def fibonacci(n):"`} />
