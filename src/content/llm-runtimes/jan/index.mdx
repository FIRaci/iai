## Tổng quan

<Note type="info">
**Jan AI** là open-source desktop application cho local LLM inference. Giao diện đẹp, dễ dùng, hỗ trợ nhiều models.
</Note>

## Cài đặt

1. Tải từ [jan.ai](https://jan.ai)
2. Cài đặt và mở ứng dụng
3. Tải model từ trong app

## Sử dụng

1. Mở Jan
2. Hub → tải model
3. Chat → bắt đầu sử dụng
