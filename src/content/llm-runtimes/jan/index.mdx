## Tổng quan

Jan AI là open-source desktop application cho local LLM inference. Giao diện người dùng thân thiện, hỗ trợ tải và quản lý models từ Hugging Face và các nguồn khác. Tích hợp local API server tương thích OpenAI, hardware acceleration (CUDA, Vulkan), và extension support.

## Cài đặt

<CodeBlock language="powershell" code={`
# Option 1: Tải từ jan.ai
# Download installer cho Windows và chạy

# Option 2: Cài qua winget
winget install Jan.Jan

# Option 3: Từ GitHub Releases
# https://github.com/janhq/jan/releases
`} />

## Sử dụng

1. Mở Jan app sau khi cài đặt
2. Vào Hub (tab bên trái) → Browse models → chọn model (vd: Llama 3.2, Mistral, Phi-4)
3. Nhấn Download và đợi tải xong
4. Vào Chat → chọn model vừa tải → bắt đầu chat
5. Bật API Server: Settings → Local API Server → Start server tại `http://localhost:1337`
6. Dùng API từ bất kỳ tool nào (OpenAI-compatible)

<CodeBlock language="powershell" code={`
# Test API server (sau khi bật trong Settings)
curl -X POST http://localhost:1337/v1/chat/completions ^
  -H "Content-Type: application/json" ^
  -d "{\""model\"": \""llama-3.2-3b-instruct\"", \""messages\"": [{\""role\"": \""user\"", \""content\"": \""Hello\""}]}"
`} />

## Lưu ý

Jan hỗ trợ CUDA và Vulkan cho GPU acceleration. Vào Settings > Advanced để cấu hình. Models được lưu trong `~/jan/models/`. Có thể kéo thả thêm model files thủ công vào folder này. Jan API server tương thích OpenAI format, dùng được với Continue, Cline, và Open Interpreter. Nên dùng model 1-3B param cho máy không có GPU.
