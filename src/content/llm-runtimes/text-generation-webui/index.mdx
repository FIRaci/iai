## Tổng quan

text-generation-webui (Oobabooga) là web UI phổ biến nhất cho local LLM inference. Hỗ trợ nhiều backends: llama.cpp, ExLlamaV2, AutoGPTQ, transformers. Tích hợp chat UI, training LoRA, character management, và OpenAI-compatible API extension.

## Cài đặt

<CodeBlock language="powershell" code={`
git clone https://github.com/oobabooga/text-generation-webui
cd text-generation-webui
start_windows.bat
`} />

Sau khi chạy `start_windows.bat`, script sẽ tự động tạo môi trường conda và cài dependencies.

## Sử dụng

1. Mở `http://localhost:7860`
2. Tab **Model** → chọn backend (llama.cpp cho CPU, ExLlamaV2 cho GPU) → load model
3. Tab **Chat** → bắt đầu chat với character
4. Tab **Training** → fine-tune LoRA trên dữ liệu của bạn

<CodeBlock language="powershell" code={`
# Kích hoạt OpenAI-compatible API
# Vào tab Extensions -> bật openai extension
# API endpoint: http://localhost:5000/v1

# Test API
curl http://localhost:5000/v1/chat/completions -H 'Content-Type: application/json' -d '{"model":"local","messages":[{"role":"user","content":"Hello"}]}'
`} />

## Lưu ý

Mặc định dùng llama.cpp backend cho CPU. Nếu có GPU NVIDIA, cài CUDA toolkit và dùng ExLlamaV2 hoặc AutoGPTQ. Có thể import character cards định dạng TavernAI. Extensions như `openai` tạo API endpoint tương thích với SillyTavern và các frontend khác.
