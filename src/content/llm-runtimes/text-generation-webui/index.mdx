## Tổng quan

<Note type="info">
**text-generation-webui** (Oobabooga) là web UI phổ biến nhất cho local LLM inference. Hỗ trợ nhiều backends: llama.cpp, ExLlamaV2, AutoGPTQ.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`git clone https://github.com/oobabooga/text-generation-webui
cd text-generation-webui
start_windows.bat`} />

## Sử dụng

1. Mở browser tại `http://localhost:7860`
2. Tab **Model** → chọn và load model
3. Tab **Chat** → bắt đầu chat

## Tips

<Note type="tip">
**Extensions:** Cài extensions như `openai` để tạo API endpoint tương thích OpenAI.
</Note>
