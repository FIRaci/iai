## Tổng quan

<Note type="info">
**Triton Inference Server** (NVIDIA) là GPU inference server cho nhiều framework: TensorFlow, PyTorch, ONNX, TensorRT. Tối ưu cho production serving.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker pull nvcr.io/nvidia/tritonserver:24.01-py3`} />

## Sử dụng

<CodeBlock language="powershell" code={`docker run --gpus all --rm -p 8000:8000 -p 8001:8001 -p 8002:8002 -v /path/to/models:/models nvcr.io/nvidia/tritonserver:24.01-py3 tritonserver --model-repository=/models`} />
