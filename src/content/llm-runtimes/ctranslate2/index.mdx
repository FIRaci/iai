## Tổng quan

<Note type="info">
**CTranslate2** là fast inference engine cho transformer models, hỗ trợ translation và text generation. Tối ưu cho CPU và GPU.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install ctranslate2`} />

## Sử dụng

<CodeBlock language="python" code={`import ctranslate2
translator = ctranslate2.Translator("model_dir")
results = translator.translate_batch([["Hello", "world"]])`} />
