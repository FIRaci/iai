## Tổng quan

<Note type="info">
**LM Studio** là desktop application cho local LLM inference. Giao diện GUI dễ dùng, hỗ trợ tải và chạy models trực tiếp.
</Note>

## Cài đặt

1. Tải từ [lmstudio.ai](https://lmstudio.ai)
2. Cài đặt và mở ứng dụng
3. Tìm và tải model từ trong app

## Sử dụng

1. Mở LM Studio
2. Tab **Search** → tìm model (Llama, Mistral, v.v.)
3. Tab **Chat** → bắt đầu chat
4. Tab **Server** → khởi tạo local API server

## Tips

<Note type="tip">
**Local API:** LM Studio tạo OpenAI-compatible API server tại `http://localhost:1234`
</Note>
