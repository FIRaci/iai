## Tổng quan

LM Studio là desktop application cho local LLM inference, giao diện GUI dễ dùng. Hỗ trợ tải và chạy nhiều models (Llama, Mistral, Gemma) với GPU acceleration. Tích hợp OpenAI-compatible local API server và quantized model formats (GGUF).

## Cài đặt

Tải từ lmstudio.ai. Yêu cầu GPU với VRAM >= 4GB (khuyến nghị 8GB+) và RAM >= 16GB.

<CodeBlock language="powershell" code={`
# Tải từ https://lmstudio.ai
# Windows: chạy LM-Studio-win32-x64.exe

# Yêu cầu tối thiểu
# GPU: NVIDIA GTX 1060 6GB hoặc AMD tương đương
# RAM: 16GB (8GB tối thiểu cho 7B models)
`} />

## Sử dụng

1. Tab Search → tìm model (Mistral, Llama 3)
2. Chọn phiên bản GGUF → Download
3. Tab Chat → chọn model → Load
4. Bắt đầu chat hoặc testing

<CodeBlock language="powershell" code={`
# Chọn quantization phù hợp
# Q4_K_M: cân bằng tốc độ/chất lượng
# Q5_K_M: chất lượng cao hơn, tốn VRAM hơn
# Q8_0: chất lượng cao nhất, cần nhiều VRAM
`} />

## Local API Server

LM Studio có sẵn API server OpenAI-compatible.

<CodeBlock language="powershell" code={`
# Tab Server -> Start Server
# Mặc định: http://localhost:1234

# Kiểm tra bằng curl
curl http://localhost:1234/v1/models

# Dùng với Python openai client
python -c "import openai; c=openai.OpenAI(base_url='http://localhost:1234/v1', api_key='lm-studio'); print(c.models.list())"
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
curl http://localhost:1234/v1/chat/completions -H 'Content-Type: application/json' -d '{"model":"local-model","messages":[{"role":"user","content":"Hello"}]}'
`} />

## Lưu ý

LM Studio mặc định dùng GPU acceleration (CUDA/Metal). Trên macOS, khuyến nghị dùng Apple Silicon (M1+). Model 7B cần ~4-6GB VRAM, model 13B cần ~8-10GB VRAM.
