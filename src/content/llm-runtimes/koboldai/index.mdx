## Tổng quan

<Note type="info">
**KoboldAI** là local LLM orchestration tool, tập trung vào creative writing và roleplay. Hỗ trợ nhiều backends và có web UI riêng.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`git clone https://github.com/KoboldAI/KoboldAI-Client
cd KoboldAI-Client
pip install -r requirements.txt
python aiserver.py`} />

## Sử dụng

1. Mở `http://localhost:5000`
2. Chọn model và load
3. Bắt đầu viết/storytelling
