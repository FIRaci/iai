## Tổng quan

Warp là modern terminal với AI built-in, viết bằng Rust. Tích hợp AI command generation, error explanation, smart autocomplete, và workflows. Hỗ trợ persistent SSH, themes, blocks (output grouping), và team sharing.

## Cài đặt

<CodeBlock language="powershell" code={`
# Warp hiện hỗ trợ macOS và Linux
# Tải từ https://warp.dev

# Kích hoạt AI features
# Settings -> AI -> Enable AI
# Đăng nhập Warp account để sử dụng
`} />

## Sử dụng

<CodeBlock language="powershell" code={`
# AI Command Search
# Ctrl+Shift+I -> gõ câu lệnh tự nhiên
# (VD: "find all files larger than 100MB")

# AI Explanation
# Ctrl+Shift+P -> chọn Explain
# Warp giải thích output và lỗi

# Workflows
# Ctrl+Shift+X -> chọn workflow có sẵn

# Blocks (nhóm output)
# Ctrl+Shift+B -> tạo block
# Copy output từng lệnh riêng biệt
`} />

## Themes và Customization

<CodeBlock language="powershell" code={`
# Warp -> Settings -> Appearance -> Theme
# Có sẵn: Dark, Light, Dracula, Nord
# Fonts khuyến nghị: JetBrains Mono, Fira Code
# Custom themes: github.com/warpdotdev/themes
`} />

## Persistent SSH

Warp lưu SSH sessions và tự động kết nối lại khi mở tab.

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
warp --version
# Mở Warp -> Ctrl+Shift+I -> xem AI suggestions
`} />

## Lưu ý

Warp yêu cầu đăng nhập account để dùng AI features. Gói Warp là free, Warp Pro thêm AI features (explain error, suggest fix). Workflows có thể tạo tùy chỉnh bằng file YAML.
