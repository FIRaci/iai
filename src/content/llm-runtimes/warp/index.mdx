## Tổng quan

<Note type="info">
**Warp** là modern terminal với AI built-in. Có AI command search, error explanation, và workflow blocks.
</Note>

## Cài đặt

Tải từ [warp.dev](https://warp.dev)

## Sử dụng

1. Mở Warp terminal
2. Gõ natural language → AI suggest commands
3. Nhấn Enter để execute
4. Click error → AI giải thích và suggest fix
