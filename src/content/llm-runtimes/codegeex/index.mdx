## Tổng quan

<Note type="info">
**CodeGeeX** là AI code generation model, hỗ trợ 20+ ngôn ngữ lập trình. Có thể chạy local qua VS Code extension.
</Note>

## Cài đặt

Cài extension từ VS Code Marketplace: **CodeGeeX**

## Sử dụng

1. Mở file code
2. Nhấn `Ctrl+Enter` để generate code
3. Accept hoặc reject suggestions
