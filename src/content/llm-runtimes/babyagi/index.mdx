## Tổng quan

<Note type="info">
**BabyAGI** là task-driven agent framework, quản lý task list và tự động ưu tiên, thực hiện tasks dựa trên objective.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`git clone https://github.com/yoheinakajima/babyagi
cd babyagi
pip install -r requirements.txt`} />

## Sử dụng

<CodeBlock language="powershell" code={`python babyagi.py`} />
