## Tổng quan

<Note type="info">
**Modal** là serverless platform cho Python applications với GPU/CPU auto-scaling. Modal cho phép deploy functions chạy trên cloud infrastructure, tự động scale xuống 0 khi không dùng. Hỗ trợ GPU (A100, T4, L40S), distributed computing, scheduled jobs, và web endpoints.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install modal
modal token new`} />

## Web Endpoint

<CodeBlock language="python" code={`import modal

app = modal.App("ml-api")

@app.function(gpu="T4", timeout=120)
@modal.web_endpoint(method="POST")
def predict(data: dict) -> dict:
    import torch
    result = {"prediction": "processed", "input": data["text"]}
    return result`} />

## Scheduled Job

<CodeBlock language="python" code={`app = modal.App("daily-task")

@app.function(schedule=modal.Period(days=1))
def daily_report():
    print("Generating daily report...")
    # Code xu ly o day`} />

## Deploy

<CodeBlock language="powershell" code={`modal deploy app.py`} />

## Xác minh

<CodeBlock language="powershell" code={`modal run app.py`} />
