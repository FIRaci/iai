## Tổng quan

<Note type="info">
**Netlify** là cloud platform cho web deployment và serverless functions. Hỗ trợ Git-based CI/CD, automatic HTTPS, Edge Functions, và form handling. Phổ biến cho Jamstack và static sites.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -g netlify-cli
# Login
netlify login`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Deploy tu project
netlify deploy --prod

# Local dev
netlify dev

# Manage sites
netlify sites:list`} />
