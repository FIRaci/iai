## Tổng quan

<Note type="info">
**BentoML** là framework mã nguồn mở cho ML model serving. BentoML đóng gói models, preprocessing code, và dependencies vào standardized Bento package. Hỗ trợ frameworks: PyTorch, TensorFlow, Scikit-learn, XGBoost, Transformers, và ONNX. Deploy lên Docker, Kubernetes, hoặc BentoCloud.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install bentoml`} />

## Build Service

<CodeBlock language="python" code={`import bentoml
from bentoml.io import JSON

model = bentoml.sklearn.get("iris_model:latest")

@bentoml.service(
    resources={"cpu": "2"},
    traffic={"timeout": 10},
)
class IrisClassifier:
    def __init__(self):
        self.model = model.to_runner()
        self.model.init_local()

    @bentoml.api(input=JSON(), output=JSON())
    def predict(self, input_data):
        import numpy as np
        result = self.model.run(np.array([input_data["features"]]))
        return {"prediction": int(result[0])}`} />

## Build Bento và Containerize

<CodeBlock language="powershell" code={`bentoml build
bentoml containerize iris_classifier:latest -t myrepo/iris:v1`} />

## Xác minh

<CodeBlock language="powershell" code={`bentoml serve --production`} />

<Note type="tip">
Dùng `bentoml.models.import_model()` để import models từ training scripts. Bento tự động quản lý versioning và dependency tracking.
</Note>
