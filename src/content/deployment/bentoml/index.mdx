## Tổng quan

<Note type="info">
**BentoML** là model serving framework, đóng gói ML models thành production-ready services. Hỗ trợ nhiều frameworks.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install bentoml`} />

## Sử dụng

<CodeBlock language="python" code={`import bentoml
@bentoml.service
class MyModel:
    def __init__(self):
        self.model = bentoml.models.get("my-model:latest")
    @bentoml.api
    def predict(self, input: str) -> str:
        return self.model.predict(input)`} />

### Serve

<CodeBlock language="powershell" code={`bentoml serve service:MyModel`} />
