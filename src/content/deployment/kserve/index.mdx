## Tổng quan

<Note type="info">
**KServe** (trước đây là KFServing) là serverless model serving platform trên Kubernetes. KServe hỗ trợ auto-scaling, canary rollouts, request batching, và model explainability. KServe tương thích với TensorFlow, PyTorch, Scikit-learn, XGBoost, và Triton Inference Server.
</Note>

## Cài đặt

Cài đặt KServe trên Kubernetes cluster:

<CodeBlock language="powershell" code={`kubectl create namespace kserve
kubectl apply -f https://github.com/kserve/kserve/releases/latest/download/kserve.yaml`} />

## InferenceService CRD

<CodeBlock language="yaml" code={`apiVersion: serving.kserve.io/v1beta1
kind: InferenceService
metadata:
  name: sklearn-iris
spec:
  predictor:
    sklearn:
      storageUri: gs://kfserving-examples/models/sklearn/iris
    minReplicas: 1
    maxReplicas: 5`} />

## Kiểm tra Inference

<CodeBlock language="powershell" code={`kubectl get inferenceservice
curl -v http://sklearn-iris.default.example.com/v1/models/sklearn-iris:predict -d '{"instances":[[5.1, 3.5, 1.4, 0.2],[6.2, 3.4, 5.4, 2.3]]}'`} />

## Lưu ý

<Note type="tip">
KServe hỗ trợ multi-model serving trên cùng một pod. Kết hợp với Knative để auto-scale xuống 0 khi không có traffic. Dùng canary strategy để test model mới trước khi chuyển toàn bộ traffic.
</Note>
