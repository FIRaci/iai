## Tổng quan

<Note type="info">
**Seldon Core** là MLOps platform mã nguồn mở cho ML model deployment trên Kubernetes. Seldon hỗ trợ A/B testing, canary deployments, multi-armed bandit, và model monitoring với Prometheus metrics. Seldon tương thích với TensorFlow, PyTorch, Scikit-learn, XGBoost, và custom models.
</Note>

## Cài đặt

Cài đặt qua Helm:

<CodeBlock language="powershell" code={`helm repo add seldon https://storage.googleapis.com/seldon-charts
helm install seldon-core seldon/seldon-core-operator --namespace seldon-system`} />

## Deploy Model

<CodeBlock language="yaml" code={`apiVersion: machinelearning.seldon.io/v1
kind: SeldonDeployment
metadata:
  name: sklearn-model
spec:
  predictors:
  - name: default
    componentSpecs:
    - spec:
        containers:
        - image: seldonio/sklearn-iris:0.1
          name: classifier
    graph:
      name: classifier
      type: MODEL
      children: []`} />

## Kiểm tra

<CodeBlock language="powershell" code={`kubectl get seldondeployments
curl http://localhost:8000/seldon/default/sklearn-model/api/v1.0/predictions -d '{"data":{"ndarray":[[5.1, 3.5, 1.4, 0.2]]}}'`} />
