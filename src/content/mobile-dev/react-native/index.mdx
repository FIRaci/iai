## Tổng quan

<Note type="info">
**React Native** là framework phát triển ứng dụng di động đa nền tảng bằng React/JavaScript. Cho phép chia sẻ code giữa iOS và Android trong khi vẫn giữ được native performance.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install react-native`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến

## Tips
