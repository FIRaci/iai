## Tổng quan

<Note type="info">
**Kotlin** là ngôn ngữ lập trình hiện đại chạy trên JVM, được Google chọn làm ngôn ngữ chính cho Android development. Kết hợp OOP và functional programming.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install kotlin`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến

## Tips

<Note type="tip">
**Bắt đầu nhanh:** Tham gia cộng đồng chính thức và đọc documentation để nắm vững các concepts cơ bản trước khi bắt đầu dự án thực tế.
</Note>
