## Tổng quan

<Note type="info">
**Meilisearch** là fast, relevant full-text search engine. Open-source và dễ deploy.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -p 7700:7700 getmeili/meilisearch:latest`} />

## Sử dụng

<CodeBlock language="python" code={`import meilisearch
client = meilisearch.Client("http://localhost:7700")
index = client.create_index("movies")
index.add_documents([
    {"id": 1, "title": "The Matrix"},
    {"id": 2, "title": "Inception"}
])
results = index.search("matrix")
print(results)`} />
