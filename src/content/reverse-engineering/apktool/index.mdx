## Tổng quan

<Note type="info">
**APKTool** là công cụ reverse engineering cho Android APK files. Cho phép decode resources, decompile smali code và rebuild APK.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install apktool`} />
