## Tổng quan

<Note type="info">
**IDA Pro** là interactive disassembler và debugger chuyên nghiệp nhất cho reverse engineering. Hỗ trợ đa nền tảng, decompiler và plugin ecosystem phong phú.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install ida-pro`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
