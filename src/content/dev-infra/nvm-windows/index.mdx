## Tổng quan

<Note type="info">
**nvm-windows** là Node.js version manager cho Windows. Cài và switch giữa nhiều Node.js versions dễ dàng.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install CoreyButler.NVMforWindows`} />

## Sử dụng

<CodeBlock language="powershell" code={`nvm install 20
nvm use 20
nvm list`} />
