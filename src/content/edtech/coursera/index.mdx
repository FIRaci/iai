## Tổng quan

<Note type="info">
**Coursera** là nền tảng học trực tuyến hợp tác với các đại học hàng đầu thế giới. Cung cấp courses và degrees trực tuyến.
</Note>

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
