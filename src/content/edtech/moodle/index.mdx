## Tổng quan

<Note type="info">
**Moodle** là hệ thống quản lý học tập mã nguồn mở phổ biến nhất thế giới. Hỗ trợ tạo khóa học online và quản lý bài tập.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install moodle`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến

## Tips

</Note>
