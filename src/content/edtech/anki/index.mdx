## Tổng quan

<Note type="info">
**Anki** là phần mềm flashcard thông minh sử dụng spaced repetition algorithm. Tối ưu hóa việc ghi nhớ dài hạn.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install anki`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
