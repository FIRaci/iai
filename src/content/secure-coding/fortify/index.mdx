## Tổng quan

<Note type="info">
**Fortify** là bộ giải pháp Application Security từ Micro Focus, cung cấp static và dynamic code analysis. Phát hiện sớm vulnerabilities trong SDLC.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install fortify`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
