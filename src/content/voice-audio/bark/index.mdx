## Tổng quan

<Note type="info">
**Bark** (Suno AI) là text-to-speech model generative, tạo giọng nói tự nhiên với emotions và sound effects.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install git+https://github.com/suno-ai/bark.git`} />

## Sử dụng

<CodeBlock language="python" code={`from bark import generate_audio
from scipy.io.wavfile import write as write_wav

audio_array = generate_audio("Hello, how are you today?")
write_wav("output.wav", 48000, audio_array)`} />
