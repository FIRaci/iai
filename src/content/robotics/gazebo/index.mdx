## Tổng quan

<Note type="info">
**Gazebo** là 3D robot simulator mạnh mẽ, tích hợp chặt chẽ với ROS/ROS2. Cung cấp physics engine và sensor simulation realistic.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install gazebo`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
