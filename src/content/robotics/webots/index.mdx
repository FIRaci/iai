## Tổng quan

<Note type="info">
**Webots** là open-source robot simulator với GUI thân thiện, hỗ trợ nhiều robot models và sensors. Có thể lập trình bằng C, C++, Python, Java và MATLAB.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install webots`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
