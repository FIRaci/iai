## Tổng quan

<Note type="info">
**ROS 2** là phiên bản kế thừa của ROS, thiết kế cho production-grade robotics. Hỗ trợ real-time, đa nền tảng và DDS-based communication.
</Note>

<CodeBlock language="powershell" code={`npm install ros2`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
