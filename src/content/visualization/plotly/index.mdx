## Tổng quan

<Note type="info">
**Plotly** là interactive visualization library, tạo charts đẹp và dashboards. Hỗ trợ Python, R, và JavaScript.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install plotly`} />

## Sử dụng

<CodeBlock language="python" code={`import plotly.express as px
df = px.data.iris()
fig = px.scatter(df, x="sepal_width", y="sepal_length", color="species")
fig.show()`} />
