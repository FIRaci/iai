## Tổng quan

<Note type="info">
**BLIP** (Salesforce) là vision-language model cho image captioning, VQA, và feature extraction. Tối ưu cho multimodal understanding.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install transformers`} />

## Sử dụng

<CodeBlock language="python" code={`from transformers import BlipProcessor, BlipForConditionalGeneration
from PIL import Image

processor = BlipProcessor.from_pretrained("Salesforce/blip-image-captioning-base")
model = BlipForConditionalGeneration.from_pretrained("Salesforce/blip-image-captioning-base")
image = Image.open("image.jpg")
inputs = processor(image, return_tensors="pt")
out = model.generate(**inputs)
caption = processor.decode(out[0], skip_special_tokens=True)
print(caption)`} />
