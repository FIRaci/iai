## Tổng quan

<Note type="info">
**SWC** là super-fast JavaScript/TypeScript compiler. Viết bằng Rust, nhanh hơn Babel 20x.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev @swc/core @swc/cli`} />

## Sử dụng

<CodeBlock language="powershell" code={`// .swcrc
{
  "jsc": {
    "parser": { "syntax": "typescript" },
    "target": "es2020"
  }
}

# Compile
npx swc src -d dist

# Watch mode
npx swc src -d dist -w`} />

## Tips

<Note type="tip">
**Next.js:** Next.js dùng SWC mặc định cho fast refresh và minification.
