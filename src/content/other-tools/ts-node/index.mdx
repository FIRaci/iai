## Tổng quan

<Note type="info">
**ts-node** là TypeScript execution engine cho Node.js. Run TypeScript files trực tiếp không cần compile trước.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev ts-node typescript`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Run TypeScript file
npx ts-node script.ts

# REPL
npx ts-node

# Với ESM
npx ts-node --esm script.ts

# Trong package.json
"scripts": {
  "dev": "ts-node src/index.ts"
}`} />

## Tips

<Note type="tip">
**tsx:** Consider dùng `tsx` thay thế - nhanh hơn và hỗ trợ ESM tốt hơn.
</Note>
