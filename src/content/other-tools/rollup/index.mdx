## Tổng quan

<Note type="info">
**Rollup** là module bundler cho JavaScript. Tối ưu cho libraries và code splitting với tree-shaking.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev rollup`} />

## Sử dụng

<CodeBlock language="powershell" code={`// rollup.config.js
export default {
  input: 'src/index.js',
  output: {
    file: 'dist/bundle.js',
    format: 'esm'
  },
  plugins: []
}

# Build
npx rollup -c`} />

## Tips

<Note type="tip">
**Tree-shaking:** Rollup tự động loại bỏ unused code - bundle size nhỏ hơn.
</Note>
