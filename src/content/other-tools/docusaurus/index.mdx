## Tổng quan

<Note type="info">
**Docusaurus** là static site generator của Meta cho documentation websites. Hỗ trợ Markdown, versioning, và i18n.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npx create-docusaurus@latest my-docs classic`} />

## Sử dụng

<CodeBlock language="powershell" code={`cd my-docs
# Start dev server
npm start

# Build production
npm run build

# Serve production build
npm run serve`} />

## Tips

<Note type="tip">
**Versioning:** Dùng `npm run docusaurus docs:version 1.0.0` để tạo docs versions.
</Note>
