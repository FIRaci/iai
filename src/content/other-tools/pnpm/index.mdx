## Tổng quan

<Note type="info">
**pnpm** là fast, disk-efficient package manager. Symlink-based node_modules tiết kiệm disk space và install time.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -g pnpm`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Install dependencies
pnpm install

# Add package
pnpm add express

# Dev dependency
pnpm add -D typescript

# Run scripts
pnpm run build`} />

## Tips

<Note type="tip">
**Workspace:** pnpm workspace hỗ trợ monorepo management tốt hơn npm/yarn.
</Note>
