## Tổng quan

<Note type="info">
**Lua** là lightweight scripting language. Nhúng được vào applications, dùng trong game dev và config files.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Lua.Lua`} />

## Sử dụng

<CodeBlock language="powershell" code={`-- script.lua
print("Hello, World!")

-- Variables
local name = "Lua"
local version = 5.4

-- Functions
function greet(name)
    return "Hello, " .. name .. "!"
end

-- Run
lua script.lua`} />

## Tips

<Note type="tip">
**Embedding:** Lua dễ dàng nhúng vào C/C++ applications - dùng cho scripting và config.
</Note>
