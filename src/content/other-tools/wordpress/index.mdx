## Tổng quan

<Note type="info">
**WordPress** là open-source CMS platform. Power 40%+ websites với themes, plugins, và flexible content management.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 8080:80 wordpress`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:8080
# 1. Setup WordPress (language, admin account)
# 2. Chọn theme
# 3. Cài plugins
# 4. Tạo pages và posts
# 5. Customize với Customizer`} />

## Tips
