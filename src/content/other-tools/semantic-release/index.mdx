## Tổng quan

<Note type="info">
**semantic-release** là automated versioning và package publishing. Conventional commits trigger releases tự động.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev semantic-release @semantic-release/github`} />

## Sử dụng

<CodeBlock language="powershell" code={`# .releaserc.json
{
  "branches": ["main"],
  "plugins": [
    "@semantic-release/commit-analyzer",
    "@semantic-release/release-notes-generator",
    "@semantic-release/github"
  ]
}

# Run (thường qua CI)
npx semantic-release`} />

## Tips

<Note type="tip">
**CI Integration:** Chạy semantic-release trong GitHub Actions sau mỗi merge to main.
</Note>
