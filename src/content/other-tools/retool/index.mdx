## Tổng quan

<Note type="info">
**Retool** là low-code platform để build internal tools nhanh chóng. Drag-and-drop UI với database và API integrations.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 3000:3000 tryretool/onprem`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:3000
# 1. Tạo app mới
# 2. Kéo thả components (Table, Form, Button)
# 3. Connect resources (PostgreSQL, REST API)
# 4. Bind data và events
# 5. Publish app`} />

## Tips

<Note type="tip">
**Queries:** Retool queries hỗ trợ JavaScript để transform data trước khi hiển thị.
</Note>
