## Tổng quan

<Note type="info">
**WezTerm** là GPU-accelerated terminal emulator viết bằng Rust. Cross-platform, ligatures, và Lua config.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install wez.wezterm`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Config trong ~/.wezterm.lua
local wezterm = require 'wezterm'
return {
  font = wezterm.font('Cascadia Code'),
  font_size = 12.0,
  color_scheme = 'Catppuccin Mocha',
}

# Mở WezTerm
wezterm`} />

## Tips

<Note type="tip">
**Lua config:** WezTerm dùng Lua cho configuration - powerful và flexible.
</Note>
