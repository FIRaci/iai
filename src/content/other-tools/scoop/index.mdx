## Tổng quan

<Note type="info">
**Scoop** là command-line installer cho Windows. Giống Homebrew nhưng cho Windows - cài đặt dev tools dễ dàng.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`Set-ExecutionPolicy RemoteSigned -Scope CurrentUser; irm get.scoop.sh | iex`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Tìm package
scoop search nodejs

# Cài đặt
scoop install nodejs

# Update
scoop update

# List installed
scoop list`} />

## Tips

<Note type="tip">
**Buckets:** Thêm buckets với `scoop bucket add extras` để access nhiều packages hơn.
</Note>
