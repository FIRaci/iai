## Tổng quan

<Note type="info">
**ConEmu** là customizable terminal emulator cho Windows. Hỗ trợ tabs, splits, và nhiều shells trong một window.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Maximus5.ConEmu`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Mở ConEmu
# Tạo tabs mới cho các shells khác nhau
# PowerShell, CMD, Git Bash, WSL

# Split panes
# Win+A - split horizontal
# Win+Shift+A - split vertical`} />

## Tips

<Note type="tip">
**Tasks:** Cấu hình Tasks để mở nhiều shells cùng lúc khi khởi động.
</Note>
