## Tổng quan

<Note type="info">
**Sanity** là headless CMS với real-time collaboration. Structured content platform với customizable editor.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -g @sanity/cli`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Tạo project
sanity init

# Start dev server
sanity dev

# Deploy
sanity deploy

# Query với GROQ
*[_type == "post"]{title, slug}`} />

## Tips

<Note type="tip">
**GROQ:** Sanity dùng GROQ query language - mạnh mẽ và type-safe.
</Note>
