## Tổng quan

<Note type="info">
**Postman** là API platform để develop, test, và document APIs. GUI-based với collection management và automation.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Postman.Postman`} />

## Sử dụng

<CodeBlock language="powershell" code={`# 1. Mở Postman
# 2. Tạo new request
# 3. Chọn method, nhập URL
# 4. Thêm headers, body, params
# 5. Send và xem response
# 6. Save vào collection`} />

## Tips

<Note type="tip">
**Collections:** Tổ chức requests vào collections để share với team.
</Note>
