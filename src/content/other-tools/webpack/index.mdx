## Tổng quan

<Note type="info">
**Webpack** là static module bundler cho JavaScript. Bundle, optimize, và serve assets cho web applications.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev webpack webpack-cli`} />

## Sử dụng

<CodeBlock language="powershell" code={`// webpack.config.js
module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist')
  },
  module: {
    rules: [{ test: /\.css$/, use: ['style-loader', 'css-loader'] }]
  }
}

# Build
npx webpack`} />

## Tips

<Note type="tip">
**Dev Server:** Dùng `webpack-dev-server` cho hot reload during development.
</Note>
