## Tổng quan

<Note type="info">
**Alacritty** là GPU-accelerated terminal emulator nhanh nhất. Written in Rust, cross-platform, và highly configurable.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Alacritty.Alacritty`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Cấu hình trong %APPDATA%\alacritty\alacritty.toml
[font]
normal = { family = "Cascadia Code", style = "Regular" }
size = 12

[colors.primary]
background = "#1e1e2e"
foreground = "#cdd6f4" `} />

## Tips

<Note type="tip">
**Performance:** Alacritty không có tabs built-in - dùng với tmux hoặc Windows Terminal.
</Note>
