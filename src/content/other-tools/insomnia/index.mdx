## Tổng quan

<Note type="info">
**Insomnia** là API client để test và debug REST, GraphQL, và gRPC APIs. Alternative cho Postman với UI clean hơn.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Insomnia.Insomnia`} />

## Sử dụng

<CodeBlock language="powershell" code={`# 1. Tạo new request
# 2. Chọn method (GET, POST, v.v.)
# 3. Nhập URL và headers
# 4. Send và xem response
# 5. Save vào collection để tái sử dụng`} />

## Tips

<Note type="tip">
**Environments:** Sử dụng environments để switch giữa dev/staging/production configs.
</Note>
