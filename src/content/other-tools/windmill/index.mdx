## Tổng quan

<Note type="info">
**Windmill** là developer platform để build workflows và internal apps. Code-first với UI builder.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 8000:80 ghcr.io/windmill-labs/windmill`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:8000
# 1. Tạo workspace
# 2. Write scripts (Python, TypeScript, Go)
# 3. Create flows từ scripts
# 4. Build UIs với drag-and-drop
# 5. Schedule và trigger`} />

## Tips

<Note type="tip">
**Multi-language:** Windmill hỗ trợ Python, TypeScript, Go, Bash, và nhiều languages.
</Note>
