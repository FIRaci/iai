## Tổng quan

<Note type="info">
**tmux** là terminal multiplexer. Split terminals, persistent sessions, và remote pair programming.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install tmux`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Start new session
tmux new -s mysession

# Split panes
# Ctrl+b % - vertical
# Ctrl+b " - horizontal

# Detach
# Ctrl+b d

# Reattach
tmux attach -t mysession`} />

## Tips

<Note type="tip">
**Prefix key:** Mặc định Ctrl+b - đổi thành Ctrl+a trong `.tmux.conf` cho dễ bấm.
</Note>
