## Tổng quan

<Note type="info">
**wget** là non-interactive network downloader. Download files từ web với resume support và recursive download.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# Đã có sẵn trên WSL. Windows: winget install GNU.Wget`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Download file
wget https://example.com/file.zip

# Resume download
wget -c https://example.com/file.zip

# Download entire site
wget --recursive --no-clobber https://example.com

# With output filename
wget -o output.zip https://example.com/file.zip`} />

## Tips

<Note type="tip">
**Recursive:** wget có thể download entire websites với `--recursive` flag.
</Note>
