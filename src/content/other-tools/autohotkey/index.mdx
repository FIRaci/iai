## Tổng quan

<Note type="info">
**AutoHotkey** là scripting language cho Windows automation. Tạo hotkeys, macros, và tự động hóa GUI tasks.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install AutoHotkey.AutoHotkey`} />

## Sử dụng

<CodeBlock language="powershell" code={`; script.ahk
; Remap keys
CapsLock::Ctrl

; Hotkey để mở ứng dụng
#n::Run notepad.exe

; Auto-replace text
::btw::by the way

; Chạy script
autohotkey script.ahk`} />

## Tips

<Note type="tip">
**Window Spy:** Dùng Window Spy (có sẵn) để inspect window controls và tạo automation chính xác.
</Note>
