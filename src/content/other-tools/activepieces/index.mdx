## Tổng quan

<Note type="info">
**ActivePieces** là open-source no-code automation platform. Tạo workflows tự động kết nối nhiều services mà không cần code.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 3000:80 activepieces/activepieces`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:3000
# 1. Tạo flow mới
# 2. Chọn trigger (ví dụ: new email)
# 3. Thêm actions (ví dụ: send Slack message)
# 4. Activate flow`} />

## Tips

<Note type="tip">
**Templates:** Sử dụng pre-built templates để tạo flows nhanh hơn.
</Note>
