## Tổng quan

<Note type="info">
**cmder** là terminal emulator cho Windows dựa trên ConEmu. Tích hợp Git, MSysGit, và Clink cho trải nghiệm terminal tốt hơn.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Cmder.Cmder`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Mở Cmder
cmder

# Sử dụng như terminal bình thường
# Hỗ trợ bash-like commands trên Windows
# Tích hợp Git bash commands`} />

## Tips

<Note type="tip">
**Tabs:** Cmder hỗ trợ multi-tab - dùng Ctrl+T để mở tab mới.
</Note>
