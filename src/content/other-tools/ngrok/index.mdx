## Tổng quan

<Note type="info">
**ngrok** là tunneling service để expose localhost ra internet. Tạo secure public URLs cho development.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -g ngrok`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Authenticate
ngrok config add-authtoken YOUR_TOKEN

# Expose port 3000
ngrok http 3000

# Với custom domain
ngrok http --domain=myapp.ngrok-free.app 3000`} />

## Tips

<Note type="tip">
**Web UI:** Ngrok cung cấp web UI tại http://localhost:4040 để inspect requests.
</Note>
