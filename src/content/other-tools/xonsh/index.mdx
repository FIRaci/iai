## Tổng quan

<Note type="info">
**xonsh** là Python-powered shell. Combines bash-like syntax với Python capabilities trong một shell.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install xonsh`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Xonsh shell
# Python syntax trong shell
for i in range(5):
    echo $i

# Subprocess commands
ls -la | grep .py

# Run script
xonsh script.xsh`} />

## Tips

<Note type="tip">
**Python first:** Mọi thứ trong xonsh là Python - dễ dàng viết complex scripts.
</Note>
