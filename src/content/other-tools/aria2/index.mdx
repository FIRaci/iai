## Tổng quan

<Note type="info">
**aria2** là lightweight multi-protocol download utility. Hỗ trợ HTTP/HTTPS, FTP, SFTP, BitTorrent, Metalink.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install aria2`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Download file đơn
aria2c https://example.com/file.zip

# Download với nhiều connections
aria2c -x 16 -s 16 https://example.com/file.zip

# Download torrent
aria2c file.torrent`} />

## Tips

<Note type="tip">
**Multi-connection:** Dùng `-x 16 -s 16` để tăng tốc download đáng kể.
</Note>
