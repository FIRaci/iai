## Tổng quan

<Note type="info">
**Ruff** là extremely fast Python linter và formatter. Viết bằng Rust, nhanh hơn flake8 + black 10-100x.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install ruff`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Lint
ruff check src/

# Fix auto-fixable issues
ruff check --fix src/

# Format
ruff format src/

# Config trong pyproject.toml
[tool.ruff]
line-length = 88`} />

## Tips

<Note type="tip">
**Drop-in replacement:** Ruff thay thế flake8, isort, black với single tool.
</Note>
