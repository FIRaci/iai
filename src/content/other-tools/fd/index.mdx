## Tổng quan

<Note type="info">
**fd** là fast alternative cho find. Viết bằng Rust, hỗ trợ regex, ignore .gitignore, và colors.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install sharkdp.fd`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Tìm files
fd pattern

# Tìm với extension
fd -e ts

# Tìm với regex
fd --type f "^test.*\.ts$"

# Execute command trên kết quả
fd -e js -x wc -l`} />

## Tips

<Note type="tip">
**Speed:** fd nhanh hơn find đáng kể trên large directories nhờ parallel processing.
</Note>
