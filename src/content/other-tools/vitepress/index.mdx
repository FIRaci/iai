## Tổng quan

<Note type="info">
**VitePress** là static site generator powered by Vite. Build documentation sites với Vue components và Markdown.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -D vitepress`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Khởi tạo
npx vitepress init

# Dev server
npx vitepress dev

# Build
npx vitepress build

# Preview production
npx vitepress preview`} />

## Tips

<Note type="tip">
**Vue SFC:** VitePress hỗ trợ Vue Single File Components trong Markdown.
</Note>
