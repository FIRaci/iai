## Tổng quan

<Note type="info">
**Mosh** là mobile shell thay thế SSH. Roaming support, intelligent local echo, và resilient to network changes.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install mosh`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Connect to server (thay thế ssh)
mosh user@server.com

# Với specific port
mosh --server="mosh-server --port=60001" user@server.com

# Mosh tự động reconnect khi network thay đổi`} />

## Tips

<Note type="tip">
**UDP:** Mosh dùng UDP thay vì TCP - cần mở ports 60000-61000 trên firewall.
</Note>
