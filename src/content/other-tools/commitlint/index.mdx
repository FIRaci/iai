## Tổng quan

<Note type="info">
**Commitlint** là tool để lint commit messages theo conventional commits format. Đảm bảo consistency trong commit history.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev @commitlint/cli @commitlint/config-conventional`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Tạo config file
echo "module.exports = {extends: ['@commitlint/config-conventional']}" > commitlint.config.js

# Test commit message
echo "feat: add user authentication" | npx commitlint

# Tích hợp với Husky
npx husky add .husky/commit-msg 'npx --no-install commitlint --edit "$1"'`} />

## Tips

<Note type="tip">
**Conventional Commits:** Format: `type(scope): description` - types: feat, fix, docs, style, refactor, test, chore.
</Note>
