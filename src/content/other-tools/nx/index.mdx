## Tổng quan

<Note type="info">
**Nx** là build system và monorepo framework. Smart caching, affected commands, và task orchestration.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npx create-nx-workspace@latest`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Tạo workspace
npx create-nx-workspace my-workspace

# Generate app
nx generate @nx/react:app my-app

# Build affected projects
nx affected --target=build

# Run tests
nx test my-app`} />

## Tips

<Note type="tip">
**Nx Cloud:** Sử dụng Nx Cloud để remote caching và distributed task execution.
</Note>
