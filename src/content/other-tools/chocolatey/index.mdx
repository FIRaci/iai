## Tổng quan

<Note type="info">
**Chocolatey** là package manager cho Windows. Cài đặt software từ command line giống apt trên Linux.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Tìm package
choco search nodejs

# Cài đặt
choco install nodejs -y

# Upgrade tất cả
choco upgrade all -y

# List installed
choco list --local-only`} />

## Tips

<Note type="tip">
**Admin Rights:** Chạy PowerShell as Administrator để cài Chocolatey và packages.
</Note>
