## Tổng quan

<Note type="info">
**Docker** là containerization platform. Package applications và dependencies vào containers để deploy nhất quán across environments.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Docker.DockerDesktop`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Chạy container
docker run -d -p 80:80 nginx

# Build image từ Dockerfile
docker build -t my-app .

# Docker Compose
docker-compose up -d

# List containers
docker ps`} />

## Tips

<Note type="tip">
**Dockerfile:** Dùng multi-stage builds để giảm image size đáng kể.
</Note>
