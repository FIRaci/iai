## Tổng quan

<Note type="info">
**BookStack** là wiki/documentation platform mã nguồn mở. Tổ chức docs theo Books > Chapters > Pages.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 8080:80 linuxserver/bookstack`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:8080
# Default login: admin@admin.com / password
# 1. Tạo Book mới
# 2. Thêm Chapters và Pages
# 3. Sử dụng WYSIWYG hoặc Markdown editor`} />

## Tips

<Note type="tip">
**Permissions:** Cấu hình role-based access control cho teams.
</Note>
