## Tổng quan

<Note type="info">
**Turbo** là incremental bundler và task runner cho monorepos. Smart caching và parallel execution.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev turbo`} />

## Sử dụng

<CodeBlock language="powershell" code={`# turbo.json
{
  "$schema": "https://turbo.build/schema.json",
  "tasks": {
    "build": { "dependsOn": ["^build"], "outputs": ["dist/**"] },
    "test": { "dependsOn": ["build"] },
    "dev": { "cache": false }
  }
}

# Run
npx turbo run build`} />

## Tips

<Note type="tip">
**Remote Cache:** Turbo remote cache chia sẻ cache giữa team members và CI.
</Note>
