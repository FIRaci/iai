## Tổng quan

<Note type="info">
**rclone** là command-line program để sync files với cloud storage. Hỗ trợ 70+ providers (S3, GDrive, Dropbox).
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Rclone.Rclone`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Configure remote
rclone configure

# Sync local to remote
rclone sync /local/path remote:bucket/path

# List files
rclone ls remote:bucket

# Mount cloud storage
rclone mount remote:bucket X:`} />

## Tips

<Note type="tip">
**Dry run:** Dùng `--dry-run` để xem what would happen trước khi sync thật.
</Note>
