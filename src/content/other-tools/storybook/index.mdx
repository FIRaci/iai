## Tổng quan

<Note type="info">
**Storybook** là frontend workshop cho UI components. Develop, test, và document components trong isolation.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npx storybook@latest init`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Start dev server
npm run storybook

# Tạo story
// Button.stories.ts
export default { title: 'Button' }
export const Primary = () => '<button class="primary">Click me</button>'

# Build static
npm run build-storybook`} />

## Tips

<Note type="tip">
**Addons:** Storybook addons thêm capabilities: controls, actions, a11y testing, v.v.
</Note>
