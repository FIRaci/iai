## Tổng quan

<Note type="info">
**bat** là cat clone với syntax highlighting và Git integration. Hiển thị file content đẹp hơn cat truyền thống.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install sharkdp.bat`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Xem file với syntax highlighting
bat file.py

# Xem nhiều files
bat src/*.ts

# Với line numbers và Git changes
bat --style=numbers,changes file.py`} />

## Tips

<Note type="tip">
**Alias:** Thêm `alias cat=bat` vào profile để thay thế cat mặc định.
</Note>
