## Tổng quan

<Note type="info">
**uv** là extremely fast Python package installer. Viết bằng Rust, nhanh hơn pip 10-100x.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install uv`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Install packages
uv pip install requests

# Create venv
uv venv

# Run script với dependencies
uv run --with requests python script.py

# Sync from requirements
uv pip sync requirements.txt`} />

## Tips

<Note type="tip">
**Drop-in replacement:** `uv pip` thay thế `pip` với cùng interface nhưng nhanh hơn nhiều.
</Note>
