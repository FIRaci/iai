## Tổng quan

<Note type="info">
**Drizzle ORM** là TypeScript ORM nhẹ và nhanh. Type-safe queries với developer experience tốt hơn Prisma.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install drizzle-orm better-sqlite3`} />

## Sử dụng

<CodeBlock language="powershell" code={`import { drizzle } from 'drizzle-orm/better-sqlite3'
import { sqliteTable, text, integer } from 'drizzle-orm/sqlite-core'

const users = sqliteTable('users', {
  id: integer('id').primaryKey(),
  name: text('name').notNull()
})

const db = drizzle(database)
const result = await db.select().from(users)`} />

## Tips

<Note type="tip">
**Drizzle Kit:** Dùng `drizzle-kit generate` để auto-create migrations từ schema.
</Note>
