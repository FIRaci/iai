## Tổng quan

<Note type="info">
**Joplin** là open-source note-taking app. Hỗ trợ Markdown, sync với nhiều services, và end-to-end encryption.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Joplin.Joplin`} />

## Sử dụng

<CodeBlock language="powershell" code={`# 1. Mở Joplin
# 2. Tạo notebooks và notes
# 3. Viết notes với Markdown
# 4. Sync với Dropbox/OneDrive/S3
# 5. Search và organize với tags`} />

## Tips
