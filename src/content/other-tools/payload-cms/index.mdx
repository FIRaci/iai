## Tổng quan

<Note type="info">
**Payload CMS** là headless CMS viết bằng TypeScript/Node.js. Code-first approach với React admin panel.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npx create-payload-app@latest`} />

## Sử dụng

<CodeBlock language="powershell" code={`// payload.config.ts
import { buildConfig } from 'payload/config'
export default buildConfig({
  collections: [
    {
      slug: 'posts',
      fields: [
        { name: 'title', type: 'text', required: true },
        { name: 'content', type: 'richText' }
      ]
    }
  ]
})

# Start dev server
npm run dev`} />

## Tips

<Note type="tip">
**Code-first:** Định nghĩa collections và fields trong code thay vì UI.
</Note>
