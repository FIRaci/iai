## Tổng quan

<Note type="info">
**Mintlify** là documentation platform hiện đại. Build docs sites với MDX, search, và AI-powered features.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm i -g mintlify`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Khởi tạo docs
mint init

# Start dev server
mint dev

# Build production
mint build

# Deploy
mintlify deploy`} />

## Tips

<Note type="tip">
**MDX Components:** Sử dụng custom MDX components cho interactive documentation.
</Note>
