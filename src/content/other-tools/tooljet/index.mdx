## Tổng quan

<Note type="info">
**Tooljet** là open-source low-code platform. Build internal tools, dashboards, và admin panels nhanh chóng.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 8080:80 tooljet/tooljet`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:8080
# 1. Tạo application mới
# 2. Drag-and-drop widgets
# 3. Connect datasources (DB, API)
# 4. Bind data và events
# 5. Deploy app`} />

## Tips

<Note type="tip">
**Datasources:** ToolJet hỗ trợ PostgreSQL, MongoDB, REST API, GraphQL, và nhiều hơn.
</Note>
