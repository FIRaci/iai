## Tổng quan

<Note type="info">
**Strapi** là open-source headless CMS. Build APIs nhanh chóng với customizable content types và admin panel.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npx create-strapi-app@latest my-project`} />

## Sử dụng

<CodeBlock language="powershell" code={`cd my-project
# Start dev server
npm run develop

# Truy cập admin: http://localhost:1337/admin
# 1. Tạo content types
# 2. Define fields và relations
# 3. Configure permissions
# 4. Access via REST/GraphQL API`} />

## Tips

<Note type="tip">
**Plugins:** Strapi có plugin ecosystem - thêm features như i18n, SEO, upload.
</Note>
