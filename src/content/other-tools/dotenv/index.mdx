## Tổng quan

<Note type="info">
**dotenv** là module để load environment variables từ .env file vào process.env. Quản lý config secrets an toàn.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install dotenv`} />

## Sử dụng

<CodeBlock language="powershell" code={`// index.js
require('dotenv').config()
console.log(process.env.DB_HOST)
console.log(process.env.API_KEY)

# .env file
DB_HOST=localhost
API_KEY=your-secret-key`} />

## Tips

<Note type="tip">
**Never commit .env:** Thêm `.env` vào `.gitignore` để tránh leak secrets.
</Note>
