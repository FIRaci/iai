## Tổng quan

<Note type="info">
**Hyper** là terminal emulator build với web technologies (Electron). Extensible với plugins và themes.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Hyper.Hyper`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Cấu hình trong ~/.hyper.js
module.exports = {
  config: {
    fontSize: 14,
    fontFamily: 'Cascadia Code',
    theme: 'hyper-material',
  }
}

# Cài plugins
hyper i hyperterm-tabs`} />

## Tips

<Note type="tip">
**Plugins:** Hyper có ecosystem plugins phong phú - tìm tại hyper.is/plugins.
</Note>
