## Tổng quan

<Note type="info">
**Tcl** là Tool Command Language. Scripting language đơn giản, thường dùng cho testing và automation.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install ActiveState.ActiveTcl`} />

## Sử dụng

<CodeBlock language="powershell" code={`# script.tcl
puts "Hello, World!"

set name "TCL"
puts "Hello, $name!"

# Run
tclsh script.tcl`} />

## Tips

<Note type="tip">
**Tk:** Tcl đi kèm Tk toolkit cho GUI development.
</Note>
