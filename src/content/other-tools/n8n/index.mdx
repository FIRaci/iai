## Tổng quan

<Note type="info">
**n8n** là workflow automation platform. Tự động hóa tasks giữa nhiều services với visual workflow editor.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 5678:5678 n8nio/n8n`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:5678
# 1. Tạo workflow mới
# 2. Thêm nodes (trigger, action)
# 3. Connect nodes với edges
# 4. Test và activate workflow`} />
