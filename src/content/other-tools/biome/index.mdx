## Tổng quan

<Note type="info">
**Biome** là fast formatter và linter cho JavaScript/TypeScript. Nhanh hơn ESLint + Prettier gấp 20-30x.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev @biomejs/biome`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Khởi tạo config
npx @biomejs/biome init

# Format code
npx biome format --write src/

# Lint code
npx biome lint src/

# Check (format + lint)
npx biome check --write src/`} />

## Tips

<Note type="tip">
**Performance:** Biome viết bằng Rust nên nhanh hơn đáng kể so với ESLint + Prettier.
</Note>
