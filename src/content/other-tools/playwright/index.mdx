## Tổng quan

<Note type="info">
**Playwright** là end-to-end testing framework của Microsoft. Hỗ trợ Chromium, Firefox, WebKit với single API.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm init playwright@latest`} />

## Sử dụng

<CodeBlock language="powershell" code={`// tests/example.spec.ts
import { test, expect } from '@playwright/test'
test('has title', async ({ page }) => {
  await page.goto('https://example.com')
  await expect(page).toHaveTitle(/Example/)
})

# Run tests
npx playwright test

# Run with UI
npx playwright test --ui`} />

## Tips

<Note type="tip">
**Codegen:** Dùng `npx playwright codegen` để record tests tự động.
</Note>
