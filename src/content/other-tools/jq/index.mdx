## Tổng quan

<Note type="info">
**jq** là command-line JSON processor. Parse, filter, và transform JSON data từ terminal.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install jqlang.jq`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Pretty print JSON
cat data.json | jq .

# Extract field
cat data.json | jq '.name'

# Filter array
cat data.json | jq '.[] | select(.age > 25)'

# Transform
cat data.json | jq '{name: .name, email: .email}'`} />

## Tips

<Note type="tip">
**Chaining:** Kết hợp multiple filters với `|` operator giống bash pipes.
</Note>
