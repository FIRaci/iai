## Tổng quan

<Note type="info">
**Prettier** là opinionated code formatter. Hỗ trợ JS, TS, CSS, HTML, JSON, YAML, và nhiều languages khác.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev prettier`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Format files
npx prettier --write src/**/*.js

# Check without writing
npx prettier --check src/**/*.js

# Config trong .prettierrc
{
  "semi": true,
  "singleQuote": true,
  "tabWidth": 2
}`} />

## Tips

<Note type="tip">
**Editor integration:** Cài Prettier extension trong VS Code để format on save.
</Note>
