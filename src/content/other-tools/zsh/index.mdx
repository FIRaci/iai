## Tổng quan

<Note type="info">
**Zsh** là powerful shell với features như auto-completion, spell correction, và plugin support.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# Có sẵn trên macOS. Windows: cài qua WSL`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Zsh shell
# Auto-completion: Tab
# Spell correction: yes/no prompt

# Plugins với Oh My Zsh
plugins=(git zsh-autosuggestions)

# Config trong ~/.zshrc
alias ll='ls -la'`} />

## Tips

<Note type="tip">
**Oh My Zsh:** Cài Oh My Zsh để có themes, plugins, và better defaults out of the box.
</Note>
