## Tổng quan

<Note type="info">
**Homebrew** là package manager cho macOS và Linux. Cài đặt software từ command line dễ dàng.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# Trên Windows: sử dụng qua WSL
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Tìm package
brew search node

# Cài đặt
brew install node

# Upgrade
brew upgrade

# List installed
brew list`} />

## Tips

<Note type="tip">
**Casks:** Dùng `brew install --cask` để cài GUI applications.
</Note>
