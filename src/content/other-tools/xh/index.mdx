## Tổng quan

<Note type="info">
**xh** là friendly and fast HTTP client. Rust-based alternative cho curl với intuitive syntax.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install ducaale.xh`} />

## Sử dụng

<CodeBlock language="powershell" code={`# GET request
xh GET https://api.example.com/users

# POST với JSON
xh POST https://api.example.com/users name=John email=john@test.com

# Download
xh --download https://example.com/file.zip

# Với headers
xh GET https://api.example.com Authorization:"Bearer token"`} />

## Tips

<Note type="tip">
**Speed:** xh nhanh hơn curl và httpie nhờ Rust implementation.
</Note>
