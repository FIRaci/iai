## Tổng quan

<Note type="info">
**ssh** là secure shell protocol. Remote login và command execution với encryption.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# Đã có sẵn trên Windows 10+`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Connect to server
ssh user@hostname

# Với specific port
ssh -p 2222 user@hostname

# SSH key authentication
ssh -i ~/.ssh/id_rsa user@hostname

# Port forwarding
ssh -L 8080:localhost:80 user@hostname`} />

## Tips

<Note type="tip">
**Config file:** Dùng `~/.ssh/config` để định nghĩa host aliases và options.
</Note>
