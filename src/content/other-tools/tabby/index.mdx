## Tổng quan

<Note type="info">
**Tabby** là highly configurable terminal emulator. Tabs, splits, themes, và plugins cho modern terminal experience.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Eugeny.Tabby`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Mở Tabby
# 1. Tạo profiles cho các connections

# 4. Cài plugins từ community`} />

## Tips

<Note type="tip">
**Profiles:** Tabby profiles hỗ trợ SSH, serial, và local terminals.
</Note>
