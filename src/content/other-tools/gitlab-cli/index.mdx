## Tổng quan

<Note type="info">
**GitLab CLI** là command-line interface cho GitLab. Quản lý projects, issues, và CI/CD pipelines từ terminal.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install GitLab.CLI`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Authenticate
glab auth login

# List MRs
glab mr list

# Tạo MR
glab mr create --title "Fix bug" --description "Description"

# View pipeline
glab ci view`} />

## Tips

<Note type="tip">
**GLAB_EDITOR:** Set editor preference với `glab config set editor vim`.
</Note>
