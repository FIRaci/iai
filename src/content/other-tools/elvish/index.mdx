## Tổng quan

<Note type="info">
**Elvish** là expressive programming language và versatile interactive shell. Combines scripting và functional programming.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Elvish.Elvish`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Elvish shell
# Pipes với structured data
ps | each [p]{ echo $p[name] }

# Functions
fn greet [name] { echo "Hello, $name!" }
greet World

# Run script
elvish script.elv`} />

## Tips

<Note type="tip">
**Structured Pipes:** Elvish pipes truyền objects thay vì text - mạnh hơn bash pipes.
</Note>
