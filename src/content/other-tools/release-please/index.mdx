## Tổng quan

<Note type="info">
**Release Please** là automated changelog và version bump tool từ Google. Conventional commits driven releases.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev release-please`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Tạo release PR
npx release-please release-pr --token=$GITHUB_TOKEN --repo-url=https://github.com/owner/repo

# Hoặc dùng GitHub Action
# .github/workflows/release.yml
# uses: googleapis/release-please-action@v4`} />

## Tips

<Note type="tip">
**Conventional Commits:** Release-please dựa vào commit messages để determine version bumps.
</Note>
