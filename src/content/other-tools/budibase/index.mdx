## Tổng quan

<Note type="info">
**Budibase** là low-code platform để build internal apps nhanh chóng. Tạo CRUD apps, admin panels, và dashboards.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d --name budibase -p 80:80 budibase/budibase`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost
# 1. Create app from template hoặc blank
# 2. Connect data source (PostgreSQL, REST API, CSV)
# 3. Build UI với drag-and-drop components
# 4. Publish app`} />

## Tips

<Note type="tip">
**Built-in DB:** Budibase có internal database - không cần external DB cho simple apps.
</Note>
