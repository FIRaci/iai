## Tổng quan

<Note type="info">
**zoxide** là smarter cd command. Học từ navigation patterns để jump đến directories thường dùng nhanh hơn.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install ajeetdsouza.zoxide`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Thêm vào shell config
# PowerShell:
Invoke-Expression (& { (zoxide init powershell | Out-String) })

# Usage
z project  # cd đến directory chứa "project"
z -i       # interactive select
z -l       # list directories`} />

## Tips

<Note type="tip">
**Fuzzy matching:** zoxide dùng fuzzy matching - không cần nhớ exact path.
</Note>
