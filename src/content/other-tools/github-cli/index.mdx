## Tổng quan

<Note type="info">
**GitHub CLI** là command-line interface cho GitHub. Tạo PRs, issues, quản lý repositories từ terminal.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install GitHub.cli`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Authenticate
gh auth login

# Tạo PR
gh pr create --title "Fix bug" --body "Description"

# List PRs
gh pr list

# Clone repo
gh repo clone owner/repo`} />

## Tips

<Note type="tip">
**Aliases:** Tạo custom aliases với `gh alias set` để shortcut commands.
</Note>
