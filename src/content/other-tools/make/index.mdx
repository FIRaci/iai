## Tổng quan

<Note type="info">
**Make** là build automation tool. Định nghĩa tasks và dependencies trong Makefile để automate workflows.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install GnuWin32.Make`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Makefile
.PHONY: build test clean

build:
	npm run build

test: build
	npm test

clean:
	rm -rf dist

# Run target
make build
make test`} />

## Tips

<Note type="tip">
**PHONY:** Luôn khai báo `.PHONY` targets để tránh conflict với file names.
</Note>
