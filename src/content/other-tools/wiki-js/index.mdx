## Tổng quan

<Note type="info">
**Wiki.js** là modern wiki software. Markdown-based với powerful editor và beautiful UI.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 3000:3000 requarks/wiki`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:3000
# 1. Setup admin account
# 2. Tạo spaces và pages
# 3. Viết với Markdown editor
# 4. Organize với tags và categories
# 5. Manage permissions`} />

## Tips

<Note type="tip">
**Search:** Wiki.js có full-text search mạnh mẽ với multiple engines.
</Note>
