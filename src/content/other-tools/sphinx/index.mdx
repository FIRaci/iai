## Tổng quan

<Note type="info">
**Sphinx** là documentation generator cho Python projects. Tạo docs từ reStructuredText với nhiều output formats.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install sphinx`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Khởi tạo project
sphinx-quickstart docs

# Build HTML docs
sphinx-build -b html docs/ _build/html

# Build PDF
sphinx-build -b latex docs/ _build/latex

# Serve locally
python -m http.server --directory _build/html`} />

## Tips

<Note type="tip">
**Autodoc:** Sphinx autodoc tự động generate docs từ Python docstrings.
</Note>
