## Tổng quan

<Note type="info">
**Cargo** là package manager và build tool cho Rust. Quản lý dependencies, build, test, và publish crates.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Rustlang.Rustup`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Tạo project mới
cargo new my-project
cd my-project

# Build
cargo build

# Run
cargo run

# Test
cargo test`} />

## Tips

<Note type="tip">
**Crates.io:** Tìm packages tại crates.io - Rust's package registry.
</Note>
