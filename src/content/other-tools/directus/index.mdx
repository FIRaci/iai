## Tổng quan

<Note type="info">
**Directus** là headless CMS và data platform. Wrap SQL database với REST/GraphQL API và admin app.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 8055:8055 directus/directus`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:8055
# 1. Tạo project mới
# 2. Define collections (tables)
# 3. Add fields và configure permissions
# 4. Access data via REST API hoặc GraphQL`} />

## Tips

<Note type="tip">
**Auto-Admin:** Directus tự động tạo admin UI từ database schema.
</Note>
