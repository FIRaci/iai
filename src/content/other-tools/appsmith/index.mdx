## Tổng quan

<Note type="info">
**Appsmith** là open-source low-code platform để build internal tools. Kéo thả UI components và kết nối với databases/APIs.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d --name appsmith -p 80:80 -p 443:443 appsmith/appsmith-ce`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost
# 1. Tạo new application
# 2. Kéo thả widgets (Table, Form, Button)
# 3. Connect datasource (PostgreSQL, REST API)
# 4. Bind data to widgets`} />

## Tips

<Note type="tip">
**JS Queries:** Sử dụng JavaScript để transform data trước khi hiển thị.
</Note>
