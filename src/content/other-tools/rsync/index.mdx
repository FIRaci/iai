## Tổng quan

<Note type="info">
**rsync** là fast file sync tool. Incremental transfers chỉ copy changed parts - tiết kiệm bandwidth.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# Có sẵn trên WSL. Windows: cài qua Cygwin hoặc WSL`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Sync local directories
rsync -avz /source/ /destination/

# Sync to remote
rsync -avz /source/ user@remote:/destination/

# Dry run (preview)
rsync -avzn /source/ /destination/`} />

## Tips

<Note type="tip">
**--delete:** Dùng `--delete` để sync exact (xóa files không có ở source).
</Note>
