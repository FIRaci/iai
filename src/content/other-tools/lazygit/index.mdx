## Tổng quan

<Note type="info">
**Lazygit** là simple terminal UI cho Git commands. Quản lý commits, branches, stash với keyboard shortcuts.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install JesseDuffield.lazygit`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Mở lazygit trong repo
lazygit

# Keyboard shortcuts
# c - commit
# p - push
# n - new branch
# s - stage file
# d - diff`} />

## Tips

<Note type="tip">
**Custom Commands:** Định nghĩa custom commands trong config.yml cho workflow riêng.
</Note>
