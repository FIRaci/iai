## Tổng quan

<Note type="info">
**Changesets** là tool để quản lý changelogs và versioning cho monorepos. Tạo changeset files và tự động generate changelogs.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev @changesets/cli`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Khởi tạo
npx changeset init

# Tạo changeset mới
npx changeset

# Version bump
npx changeset version

# Publish packages
npx changeset publish`} />

## Tips

<Note type="tip">
**CI Integration:** Dùng GitHub Actions để auto-version và publish trên mỗi merge.
</Note>
