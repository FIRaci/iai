## Tổng quan

<Note type="info">
**Nushell** là modern shell viết bằng Rust. Structured data pipelines thay vì text-based pipes.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Nushell.Nushell`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Nushell shell
# Structured data pipelines
ls | where size > 1kb | sort-by name

# Filter table data
ps | where cpu > 0

# JSON processing
open data.json | get users | each { |u| echo $u.name }`} />

## Tips

<Note type="tip">
**Data-first:** Nu treats everything as structured data - dễ dàng filter và transform.
</Note>
