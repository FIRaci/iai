## Tổng quan

<Note type="info">
**GitLab CI** là CI/CD platform của GitLab. Tích hợp sẵn với GitLab repositories cho automated pipelines.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# Tích hợp sẵn với GitLab`} />

## Sử dụng

<CodeBlock language="powershell" code={`# .gitlab-ci.yml
stages:
  - test
  - build
  - deploy

test:
  stage: test
  script:
    - npm ci
    - npm test

build:
  stage: build
  script:
    - npm run build`} />

## Tips

<Note type="tip">
**Runners:** Sử dụng shared runners hoặc self-hosted runners cho pipeline execution.
</Note>
