## Tổng quan

<Note type="info">
**GitHub Actions** là CI/CD platform của GitHub. Automate workflows từ GitHub events như push, pull request, release.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# Tích hợp sẵn với GitHub repositories`} />

## Sử dụng

<CodeBlock language="powershell" code={`# .github/workflows/ci.yml
name: CI
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
      - run: npm ci
      - run: npm test`} />

## Tips

<Note type="tip">
**Secrets:** Lưu API keys và tokens trong repository Settings > Secrets.
</Note>
