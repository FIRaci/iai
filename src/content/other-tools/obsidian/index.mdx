## Tổng quan

<Note type="info">
**Obsidian** là knowledge base app dùng Markdown files. Bidirectional links, graph view, và plugin ecosystem.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Obsidian.Obsidian`} />

## Sử dụng

<CodeBlock language="powershell" code={`# 1. Tạo vault mới (folder)
# 2. Viết notes với Markdown
# 3. Link notes với [[note-name]]
# 4. Xem graph view để thấy connections
# 5. Cài plugins để extend`} />

## Tips

<Note type="tip">
**Local-first:** Obsidian lưu files locally - bạn sở hữu data của mình.
</Note>
