## Tổng quan

<Note type="info">
**ESLint** là JavaScript/TypeScript linter. Find và fix problems trong code, enforce coding standards.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev eslint`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Khởi tạo config
npx eslint --init

# Lint files
npx eslint src/**/*.js

# Fix auto-fixable issues
npx eslint src/**/*.js --fix

# Với TypeScript
npx eslint src/**/*.ts --ext .ts`} />

## Tips

<Note type="tip">
**Flat config:** ESLint v9 dùng `eslint.config.js` thay cho `.eslintrc`.
</Note>
