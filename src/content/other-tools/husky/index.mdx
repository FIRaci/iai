## Tổng quan

<Note type="info">
**Husky** là Git hooks manager. Chạy scripts trước commit, push, v.v. để enforce code quality.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev husky`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Khởi tạo
npx husky init

# Thêm pre-commit hook
echo "npm test" >> .husky/pre-commit

# Thêm commit-msg hook
echo "npx commitlint --edit $1" >> .husky/commit-msg`} />

## Tips

<Note type="tip">
**Git Hooks:** Husky thay thế manual git hooks setup với simple npm commands.
</Note>
