## Tổng quan

<Note type="info">
**tRPC** là end-to-end typesafe API framework. Shared types giữa client và server không cần code generation.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install @trpc/server @trpc/client zod`} />

## Sử dụng

<CodeBlock language="powershell" code={`// server/router.ts
export const appRouter = router({
  greeting: publicProcedure
    .input(z.object({ name: z.string() }))
    .query((opts) => 'Hello ' + opts.input.name + '!')
})

// client
const greeting = await trpc.greeting.query({ name: 'World' })`} />

## Tips

<Note type="tip">
**Zod:** tRPC dùng Zod cho input validation - typesafe từ đầu đến cuối.
</Note>
