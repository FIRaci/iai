## Tổng quan

<Note type="info">
**Ghostty** là fast, feature-rich terminal emulator viết bằng Zig. GPU-accelerated với cross-platform support.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Ghostty.Ghostty`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Cấu hình trong %APPDATA%\ghostty\config
font-family = Cascadia Code
font-size = 12
theme = catppuccin-mocha

# Mở Ghostty
ghostty`} />

## Tips

<Note type="tip">
**Performance:** Ghostty nhanh hơn nhiều terminal emulators nhờ Zig và GPU acceleration.
</Note>
