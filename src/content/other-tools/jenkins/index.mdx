## Tổng quan

<Note type="info">
**Jenkins** là open-source automation server cho CI/CD. Build, test, và deploy applications với pipelines.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 8080:8080 -p 50000:50000 jenkins/jenkins:lts`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:8080
# 1. Unlock với admin password
# 2. Cài suggested plugins
# 3. Tạo new item (pipeline)
# 4. Define Jenkinsfile
# 5. Build và monitor`} />

## Tips

<Note type="tip">
**Jenkinsfile:** Định nghĩa pipelines as code với declarative hoặc scripted syntax.
</Note>
