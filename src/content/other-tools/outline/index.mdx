## Tổng quan

<Note type="info">
**Outline** là knowledge base và wiki platform cho teams. Real-time collaboration với Markdown editor.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -d -p 3000:3000 outline/outline`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Truy cập http://localhost:3000
# 1. Tạo workspace
# 2. Tạo collections và documents
# 3. Invite team members
# 4. Write với Markdown và slash commands`} />

## Tips

<Note type="tip">
**Integrations:** Outline tích hợp với Slack, GitHub, và SSO providers.
</Note>
