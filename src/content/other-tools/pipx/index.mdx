## Tổng quan

<Note type="info">
**pipx** là tool để install và run Python applications trong isolated environments. Tránh dependency conflicts.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install pipx`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Install package
pipx install black

# Run without install
pipx run black --check .

# List installed
pipx list

# Upgrade all
pipx upgrade-all`} />

## Tips

<Note type="tip">
**Isolated envs:** Mỗi package được install trong virtual environment riêng - không conflict.
</Note>
