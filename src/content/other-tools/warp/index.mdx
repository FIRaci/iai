## Tổng quan

<Note type="info">
**Warp** là Rust-based terminal emulator với AI features. Block-based input, command palette, và team workflows.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Warp.Warp`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Mở Warp
# 1. Gõ commands với block-based input
# 2. Sử dụng AI để suggest commands
# 3. Search command history
# 4. Share outputs với team`} />

## Tips

<Note type="tip">
**AI Commands:** Warp AI giúp generate và explain commands từ natural language.
</Note>
