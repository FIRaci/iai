## Tổng quan

<Note type="info">
**iTerm2** là terminal emulator cho macOS. Features: split panes, search, autocomplete, và nhiều hơn nữa.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# macOS only: brew install --cask iterm2`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Split panes
# Cmd+D - vertical split
# Cmd+Shift+D - horizontal split

# Search
# Cmd+F - find trong terminal output

# Instant Replay
# Cmd+Opt+B - xem lại output gần đây`} />

## Tips

<Note type="tip">
**Profiles:** Tạo profiles khác nhau cho các projects với custom colors và startup commands.
</Note>
