## Tổng quan

<Note type="info">
**tsx** là fast TypeScript executor cho Node.js. Run TS/TSX files với native ESM support và caching.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev tsx`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Run TypeScript file
npx tsx script.ts

# Watch mode
npx tsx --watch script.ts

# Trong package.json
"scripts": {
  "dev": "tsx watch src/index.ts"
}`} />

## Tips

<Note type="tip">
**Performance:** tsx nhanh hơn ts-node nhờ esbuild dưới hood.
</Note>
