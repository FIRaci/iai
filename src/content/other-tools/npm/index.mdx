## Tổng quan

<Note type="info">
**npm** là package manager cho Node.js. Cài đặt, quản lý dependencies, và publish JavaScript packages.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# Đi kèm với Node.js installation`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Cài đặt package
npm install express

# Dev dependency
npm install --save-dev typescript

# Run scripts
npm run build

# Initialize project
npm init -y`} />

## Tips

<Note type="tip">
**npx:** Dùng `npx` để run packages mà không cần install global.
</Note>
