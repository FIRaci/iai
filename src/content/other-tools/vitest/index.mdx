## Tổng quan

<Note type="info">
**Vitest** là fast unit testing framework powered by Vite. Compatible với Jest API, native TypeScript support.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev vitest`} />

## Sử dụng

<CodeBlock language="powershell" code={`// tests/example.test.ts
import { describe, it, expect } from 'vitest'
describe('Math', () => {
  it('should add numbers', () => {
    expect(1 + 2).toBe(3)
  })
})

# Run tests
npx vitest

# Watch mode
npx vitest watch`} />

## Tips

<Note type="tip">
**Jest compat:** Vitest tương thích với hầu hết Jest APIs - migrate dễ dàng.
</Note>
