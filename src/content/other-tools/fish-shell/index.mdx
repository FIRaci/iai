## Tổng quan

<Note type="info">
**Fish Shell** là smart and user-friendly shell. Auto-suggestions, syntax highlighting, và không cần config phức tạp.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install fish-shell.fish-shell`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Fish shell
# Auto-suggestions hiện ra khi gõ (Tab để accept)

# Functions
function greet
    echo "Hello, $argv!"
end

# Universal variables (persist across sessions)
set -U fish_greeting "Welcome to Fish!"`} />

## Tips

<Note type="tip">
**Web config:** Chạy `fish_config` để mở web UI cấu hình theme và prompts.
</Note>
