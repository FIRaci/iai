## Tổng quan

<Note type="info">
**esbuild** là cực nhanh JavaScript bundler viết bằng Go. Build times nhanh hơn Webpack 10-100x.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install --save-dev esbuild`} />

## Sử dụng

<CodeBlock language="powershell" code={`// build.js
require('esbuild').build({
  entryPoints: ['src/index.js'],
  bundle: true,
  outfile: 'dist/bundle.js',
  minify: true,
}).catch(() => process.exit(1))

# Build
node build.js`} />

## Tips

<Note type="tip">
**Watch mode:** Dùng `--watch` flag để auto-rebuild khi files thay đổi.
</Note>
