## Tổng quan

ComfyUI là node-based graphical user interface cho Stable Diffusion. Cho phép tạo image generation workflows phức tạp thông qua visual node editor, hỗ trợ custom nodes, model management, và workflow sharing.

## Cài đặt

<CodeBlock language="powershell" code={`
# Clone repository
git clone https://github.com/comfyanonymous/ComfyUI
cd ComfyUI

# Tạo virtual environment (khuyến nghị)
python -m venv .venv
.venv\Scripts\activate

# Cài dependencies
pip install -r requirements.txt

# Download model cơ bản (SDXL)
# Tải https://huggingface.co/stabilityai/stable-diffusion-xl-base-1.0
# Đặt file .safetensors vào models/checkpoints/
`} />

## Sử dụng

1. Kích hoạt venv và chạy `python main.py`
2. Mở trình duyệt tại `http://localhost:8188`
3. Tạo workflow text-to-image: Load Checkpoint → CLIP Text Encode → KSampler → Decode → Save Image
4. Kéo thả nodes để kết nối, điều chỉnh parameters (steps, CFG, sampler)
5. Cài custom nodes: clone repo vào `custom_nodes/` folder
6. Export workflow dạng JSON để chia sẻ (Workflow → Save/Export)

<CodeBlock language="powershell" code={`
# Chạy ComfyUI
python main.py

# Cài custom nodes (ví dụ)
cd custom_nodes
git clone https://github.com/ltdrdata/ComfyUI-Manager
# Restart ComfyUI, vào Manager để browse và cài nodes
`} />

## Lưu ý

Luôn dùng virtual environment để tránh xung đột package. ComfyUI Manager là custom node bắt buộc - giúp cài nodes và models từ UI. Workflow từ community thường có sẵn JSON để drag-drop vào ComfyUI. Trên Windows, cần CUDA toolkit nếu dùng GPU NVIDIA. Có thể dùng `--force-fp16` để tiết kiệm VRAM.
