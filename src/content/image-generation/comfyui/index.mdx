## Tổng quan

<Note type="info">
**ComfyUI** là node-based GUI cho Stable Diffusion. Tạo workflows phức tạp với visual node editor.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`git clone https://github.com/comfyanonymous/ComfyUI
cd ComfyUI
pip install -r requirements.txt`} />

## Sử dụng

1. Chạy `python main.py`
2. Mở `http://localhost:8188`
3. Tạo workflow với nodes
4. Generate images
