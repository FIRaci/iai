## Tổng quan

<Note type="info">
**Cocos2d** là game engine mã nguồn mở đa nền tảng, hỗ trợ C++, JavaScript và Lua. Nổi tiếng với hiệu suất cao trên mobile và khả năng xuất bản lên nhiều nền tảng.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install cocos2d`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
