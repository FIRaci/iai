## Tổng quan

<Note type="info">
**Phaser** là HTML5 game framework mã nguồn mở, sử dụng JavaScript/TypeScript để tạo game 2D chạy trên trình duyệt. Hỗ trợ đầy đủ physics, animation, audio và input handling.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install phaser`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
