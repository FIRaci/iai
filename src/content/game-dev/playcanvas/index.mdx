## Tổng quan

<Note type="info">
**PlayCanvas** là cloud-based game engine và WebGL framework cho phép phát triển game 3D và 2D trực tiếp trên trình duyệt. Hỗ trợ real-time collaboration và xuất bản ngay lập tức lên web.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install playcanvas`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
