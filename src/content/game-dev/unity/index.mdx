## Tổng quan

<Note type="info">
**Unity** là công cụ phát triển game đa nền tảng mạnh mẽ, hỗ trợ 2D, 3D, AR/VR. Được sử dụng rộng rãi trong ngành công nghiệp game và các ứng dụng tương tác thời gian thực. Với hệ thống Asset Store phong phú và cộng đồng lớn mạnh, Unity là lựa chọn hàng đầu cho indie developer và studio chuyên nghiệp.
</Note>

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
