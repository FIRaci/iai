## Tổng quan

<Note type="info">
**Godot Engine** là game engine mã nguồn mở miễn phí, hỗ trợ xuất bản đa nền tảng. Với kiến trúc node-scene độc đáo và ngôn ngữ GDScript riêng, Godot mang đến trải nghiệm phát triển game nhẹ nhàng nhưng đầy sức mạnh.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install godot`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
