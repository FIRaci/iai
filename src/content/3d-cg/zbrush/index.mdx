## Tổng quan

<Note type="info">
**ZBrush** là phần mềm digital sculpting và painting từ Pixologic. Cho phép tạo 3D models với độ chi tiết cực cao.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install zbrush`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
