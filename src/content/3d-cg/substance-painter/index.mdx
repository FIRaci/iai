## Tổng quan

<Note type="info">
**Substance Painter** là 3D texture painting software từ Adobe, tiêu chuẩn công nghiệp cho PBR texturing. Hỗ trợ real-time painting và smart materials.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install substance-painter`} />
