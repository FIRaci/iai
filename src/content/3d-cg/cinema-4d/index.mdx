## Tổng quan

<Note type="info">
**Cinema 4D** là phần mềm 3D modeling, animation và rendering từ Maxon. Nổi tiếng với motion graphics tools và workflow trực quan.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install cinema-4d`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
