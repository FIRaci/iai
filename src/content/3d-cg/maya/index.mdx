## Tổng quan

<Note type="info">
**Maya** là phần mềm 3D computer graphics, modeling, animation và rendering chuyên nghiệp từ Autodesk. Tiêu chuẩn công nghiệp trong phim ảnh và game.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install maya`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
