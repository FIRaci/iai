## Tổng quan

TypeScript là superset của JavaScript thêm static typing, interfaces, và advanced tooling. Được phát triển bởi Microsoft, TypeScript giúp phát hiện lỗi tại compile-time và cải thiện developer experience với autocomplete và refactoring.

## Cài đặt

Cài đặt TypeScript globally qua npm:

<CodeBlock language="powershell" code={`# Cài đặt TypeScript globally
npm install -g typescript

# Kiểm tra cài đặt
tsc --version`} />

Hoặc cài đặt local cho project:

<CodeBlock language="powershell" code={`npm install --save-dev typescript
npx tsc --version`} />

## Sử dụng

### Compile TypeScript

<CodeBlock language="powershell" code={`# Compile file .ts sang .js
tsc hello.ts

# Watch mode — auto-compile khi file thay đổi
tsc --watch hello.ts

# Compile với tsconfig.json
tsc

# Chỉ check lỗi, không output file
tsc --noEmit`} />

### Khởi tạo cấu hình

<CodeBlock language="powershell" code={`# Tạo tsconfig.json
tsc --init

# Các cấu hình thường dùng trong tsconfig.json:
# - "target": "ES2022"
# - "module": "ESNext"
# - "strict": true
# - "outDir": "./dist"
# - "rootDir": "./src"`} />

### TypeScript cơ bản

<CodeBlock language="powershell" code={`
// hello.ts
function greet(name: string): string {
  return "Hello, " + name + "!";
}

const message: string = greet("World");
console.log(message);
`} />

## Lưu ý

Luôn bật `strict: true` trong tsconfig.json để tận dụng tối đa type checking. Dùng `tsc --noEmit` trong CI/CD để kiểm tra lỗi type mà không sinh file.
