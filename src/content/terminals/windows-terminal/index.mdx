## Tổng quan

Windows Terminal là ứng dụng terminal hiện đại từ Microsoft, hỗ trợ GPU-accelerated rendering, multiple tabs và panes, profile quản lý cho PowerShell, CMD, WSL, SSH. Có thể custom theme, font, background image, và hỗ trợ Unicode/emoji đầy đủ. Mã nguồn mở trên GitHub và có sẵn trên Microsoft Store.

## Cài đặt

<CodeBlock language="powershell" code={`
# Option 1: winget (khuyến nghị)
winget install Microsoft.WindowsTerminal

`} />

## Sử dụng

1. **Profiles:** Settings -> Profiles. Tạo riêng cho PowerShell 7, Command Prompt, WSL Ubuntu, và các distro khác. Cấu hình command line, startup directory, icon riêng.
2. **Color schemes:** Settings -> Color schemes. Chọn hoặc tạo scheme mới (One Half Dark, Campbell, Solarized Light, Tango Dark). Áp dụng cho từng profile.
3. **Font configuration:** Cài Cascadia Code Nerd Font hoặc Meslo Nerd Font cho Powerline icons. Settings -> Profile -> Appearance -> Font face.
4. **Startup settings:** Cấu hình default profile (PowerShell 7 khuyến nghị), launch size (columns/rows), và launch mode (maximized, full screen).
5. **Command Palette:** `Ctrl+Shift+P` mở command palette với tất cả actions: split pane, đổi theme, chọn profile, focus mode.
6. **Quake mode:** `Win+Backtick` mở terminal dạng dropdown từ đỉnh màn hình, rất tiện cho quick commands.

## Lưu ý

Nên cài PowerShell 7 (`winget install Microsoft.PowerShell`) làm default profile thay vì Windows PowerShell 5.1. Settings file JSON nằm ở `%LOCALAPPDATA%\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json`. Experimental features có thể bật trong Settings -> Experiments (rounded corners, acrylic background, animated GIF). Quake mode (`Win+Backtick`) là tính năng được sử dụng nhiều nhất của developer.
