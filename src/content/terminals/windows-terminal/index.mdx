## Tổng quan

<Note type="info">
**Windows Terminal** là modern terminal application từ Microsoft. Hỗ trợ multiple tabs, panes, themes, và GPU acceleration.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Microsoft.WindowsTerminal`} />

## Sử dụng

1. Mở Windows Terminal
2. `Ctrl+Shift+T` để mở tab mới
3. `Alt+Shift+D` để split pane
4. Customize settings qua JSON
