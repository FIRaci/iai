## Tổng quan

<Note type="info">
**Krita** là phần mềm digital painting mã nguồn mở chuyên nghiệp. Brush engine mạnh mẽ với hỗ trợ animation tools.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install krita`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
