## Tổng quan

<Note type="info">
**Canva** là nền tảng thiết kế đồ họa trực tuyến dễ sử dụng với AI-powered features. Cung cấp hàng ngàn templates.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install canva`} />

## Sử dụng cơ bản

<CodeBlock language="powershell" code={`# Khởi tạo project mới với Canva
# Cấu hình môi trường phát triển
# Bắt đầu với ví dụ hello-world

# Xem tài liệu chính thức để biết thêm chi tiết`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến

## Tips

<Note type="tip">
**Bắt đầu nhanh:** Tham gia cộng đồng chính thức và đọc documentation để nắm vững các concepts cơ bản trước khi bắt đầu dự án thực tế.
</Note>
