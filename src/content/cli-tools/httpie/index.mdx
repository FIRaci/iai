## Tổng quan

<Note type="info">
**HTTPie** là user-friendly HTTP client cho terminal, thay thế cho curl với syntax đơn giản hơn.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install httpie`} />

## Sử dụng

<CodeBlock language="powershell" code={`http GET https://api.example.com/users
http POST https://api.example.com/users name="John" age:=30
http PUT https://api.example.com/users/1 name="Jane"`} />
