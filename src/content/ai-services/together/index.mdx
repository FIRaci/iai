## Tổng quan

<Note type="info">
**Together AI** là cloud inference platform cho open-source LLMs. Together cung cấp API truy cập đến hàng trăm models: Llama, Mistral, Mixtral, DeepSeek, Qwen, và nhiều models khác. Ngoài inference, Together còn hỗ trợ fine-tuning với custom datasets và model deployment.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install together`} />

## Inference cơ bản

<CodeBlock language="python" code={`from together import Together

client = Together(api_key="your-api-key")

response = client.chat.completions.create(
    model="meta-llama/Llama-3.3-70B-Instruct-Turbo",
    messages=[{"role": "user", "content": "Viet mot doan van ngan"}],
    temperature=0.7,
    max_tokens=512,
)

print(response.choices[0].message.content)`} />

## Image Generation

<CodeBlock language="python" code={`response = client.images.generate(
    prompt="A cat in space, digital art",
    model="black-forest-labs/FLUX.1-schnell",
    steps=4,
)

print(response.data[0].url)`} />
