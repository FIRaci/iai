## Tổng quan

<Note type="info">
**Replicate** là cloud platform cho phép chạy ML models qua API mà không cần quản lý infrastructure. Replicate có hàng ngàn models: Stable Diffusion, Llama, Whisper, Flux, và nhiều models khác từ cộng đồng. Mỗi model có thể chạy với các input parameters khác nhau.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install replicate`} />

## Image Generation

<CodeBlock language="python" code={`import replicate

output = replicate.run(
    "black-forest-labs/flux-schnell",
    input={
        "prompt": "a cat wearing a spacesuit, realistic photo",
        "num_outputs": 1,
        "aspect_ratio": "1:1",
    }
)

print(output[0])  # URL of generated image`} />

## LLM Inference

<CodeBlock language="python" code={`for event in replicate.stream(
    "meta/meta-llama-3.1-405b-instruct",
    input={
        "prompt": "Viet ve AI trong 50 words",
        "temperature": 0.7,
    },
):
    print(event, end="")`} />

## Lưu ý

<Note type="tip">
Replicate tính phí theo giây chạy model. Dùng `replicate.run(..., wait=False)` để chạy async và kiểm tra status sau. Webhook cho phép nhận kết quả khi model hoàn thành.
</Note>
