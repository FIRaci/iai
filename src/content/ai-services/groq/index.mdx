## Tổng quan

<Note type="info">
**Groq** cung cấp ultra-fast LLM inference nhờ LPU (Language Processing Unit) chips độc quyền. Groq có thể đạt tốc độ hơn 1,000 tokens/giây trên Llama 3 8B, nhanh hơn 10-20 lần so với GPU truyền thống. Groq hỗ trợ Llama, Mixtral, Gemma, và các open-source models phổ biến.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install groq`} />

## Streaming Chat

<CodeBlock language="python" code={`from groq import Groq

client = Groq(api_key="gsk-your-key")

stream = client.chat.completions.create(
    model="llama3-70b-8192",
    messages=[{"role": "user", "content": "Giai thich LPU la gi?"}],
    stream=True,
)

for chunk in stream:
    print(chunk.choices[0].delta.content or "", end="")`} />

## Tool Calling

<CodeBlock language="python" code={`response = client.chat.completions.create(
    model="llama3-70b-8192",
    messages=[{"role": "user", "content": "Thoi tiet hom nay?"}],
    tools=[{
        "type": "function",
        "function": {
            "name": "get_weather",
            "parameters": {"type": "object", "properties": {"city": {"type": "string"}}}
        }
    }],
)`} />
