## Tổng quan

<Note type="info">
**GitHub Copilot** là AI pair programmer được tích hợp trong VS Code. Hỗ trợ inline suggestions, chat, và Copilot Edits — giúp bạn code nhanh hơn với AI gợi ý thông minh.
</Note>

## Yêu cầu

- **VS Code** (xem [hướng dẫn cài đặt](/dev-tools/vscode))
- **Tài khoản GitHub** (đăng ký miễn phí)
- Kết nối Internet

## Cài đặt

<StepBlock number={1} title="Mở VS Code">
<CodeBlock language="powershell">code</CodeBlock>
</StepBlock>

<StepBlock number={2} title="Cài Extension">
1. Mở Extensions (`Ctrl+Shift+X`)
2. Tìm "GitHub Copilot"
3. Click Install
4. (Khuyến khích) Cài thêm "GitHub Copilot Chat"
</StepBlock>

<StepBlock number={3} title="Đăng nhập GitHub">
1. Click icon Copilot ở góc dưới bên phải status bar
2. Sign in với GitHub account
3. Authorize extension
4. Confirm subscription (free tier có 2000 completions/tháng hoặc 300 chats)
</StepBlock>

## Sử dụng

### Inline Suggestions

Gõ code → Copilot tự động suggest → `Tab` để chấp nhận:

<CodeBlock language="typescript" code={`
// Gõ: function calculateTotal(items)
// Copilot suggest:
function calculateTotal(items: Item[]) {
  return items.reduce((sum, item) => sum + item.price * item.quantity, 0)
}

// Gõ: async function fetchUserData(userId)
// Copilot suggest:
async function fetchUserData(userId: string) {
  const response = await fetch(\`/api/users/\${userId}\`)
  if (!response.ok) throw new Error('Failed to fetch user')
  return response.json()
}
`} />

### Chat (`Ctrl+I`)

Mở chat inline → hỏi về code hiện tại:
<CodeBlock language="text">"Giải thích function này"
"Thêm error handling"
"Viết unit test cho class này"
"Tối ưu performance của đoạn code này"</CodeBlock>

### Copilot Edits (`Ctrl+Shift+I`)

Chỉnh sửa nhiều file cùng lúc với AI. Phù hợp cho:
- Thêm tính năng mới
- Refactor code
- Đổi tên biến/function

<Note type="tip">
**Shortcuts quan trọng:**
- `Tab`: Accept suggestion
- `Ctrl+→`: Accept word
- `Alt+]`: Next suggestion
- `Ctrl+Enter`: Open suggestions panel (xem 10 suggestions cùng lúc)
- `Ctrl+I`: Mở Copilot Chat inline
</Note>

## Khắc phục sự cố

### Copilot không suggest

Kiểm tra:
1. Icon Copilot ở status bar — phải có dấu tích xanh
2. File type được hỗ trợ (`.ts`, `.js`, `.py`, `.go`, etc.)
3. Kết nối Internet

Nếu vẫn không được, reload VS Code:
<CodeBlock language="powershell">Ctrl+Shift+P → "Developer: Reload Window"</CodeBlock>

### "Copilot không available ở quốc gia của bạn"

Kiểm tra GitHub account settings:
1. Vào https://github.com/settings/copilot
2. Đảm bảo subscription active
3. Nếu vẫn lỗi, dùng VPN

### Completion chậm

Giảm số lượng suggestions hiển thị:
- Settings → Search "Copilot" → "Editor: Suggest Line" → giảm xuống
