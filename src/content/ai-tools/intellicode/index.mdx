## Tổng quan

<Note type="info">
**IntelliCode** là AI code completion extension của Microsoft cho Visual Studio và VS Code. Dùng ML models để đưa ra context-aware suggestions (IntelliSense nâng cao), code style consistent, và team completions.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# VS Code: Install tu Extensions tab
# Tim "IntelliCode" va Install
# Hoac dung command:
code --install-extension visualstudioexptteam.vscodeintellicode`} />

## Sử dụng

<CodeBlock language="text" code={`Khi code trong editor, IntelliCode tu dong:
- Rank completion items (icon ⭐)
- De xuat parameter suggestions
- Hoc code style cua team
- Cung cap AI-powered IntelliSense`} />
