## Tổng quan

GPT4All là ứng dụng desktop chạy LLM local với giao diện đơn giản, không cần GPU mạnh. Tối ưu cho CPU và laptop thông thường, cho phép chat với AI hoàn toàn offline. Hỗ trợ tải model từ thư viện tích hợp sẵn và local document RAG.

## Cài đặt

Tải installer từ gpt4all.io và chạy.

<CodeBlock language="powershell" code={`
# Visit https://www.nomic.ai/gpt4all và tải bản Windows
# Chạy GPT4All-Installer-x.x.x.exe

# Hoặc cài qua scoop:
scoop bucket add extras
scoop install gpt4all
`} />

## Sử dụng

Mở GPT4All, chọn model, và bắt đầu chat.

<CodeBlock language="powershell" code={`
# 1. Mở GPT4All
# 2. Tab "Models" → Download model bạn muốn
#    (Ví dụ: Llama 3.2 3B, Mistral 7B OpenOrca)
# 3. Chọn model từ dropdown
# 4. Bắt đầu chat

# Local Document RAG:
# 1. Click "Local Documents" icon
# 2. Add folder/file (PDF, TXT, DOCX...)
# 3. Chat với AI — nó sẽ dùng context từ documents

# Settings quan trọng:
# - Model path: C:\Users\<User>\AppData\Local\nomic.ai\GPT4All\models
# - Context length: 2048 (mặc định)
# - Threads: Số CPU cores (cao hơn = nhanh hơn)
`} />

## Lưu ý

GPT4All chạy hoàn toàn offline — không cần internet sau khi tải model. Model chạy trên CPU, phù hợp với laptop không có GPU rời. Trên Windows 11, nên dùng model 3B-7B cho tốc độ khả dụng. Model GGUF format được lưu trong thư mục AppData local.
