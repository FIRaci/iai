## Tổng quan

LM Studio là ứng dụng desktop cho Windows để tải, quản lý, và chạy LLM local với giao diện đồ hoạ trực quan. Hỗ trợ tìm kiếm model từ Hugging Face, chat trực tiếp trong app, và expose REST API tương thích OpenAI.

## Cài đặt

Tải file .exe từ lmstudio.ai và cài đặt.

<CodeBlock language="powershell" code={`
winget install LMStudio.LMStudio
`} />

Hoặc tải từ https://lmstudio.ai/ và chạy LM-Studio-Setup.exe.

## Sử dụng

Mở LM Studio, tìm model từ tab Search, và bắt đầu chat.

<CodeBlock language="powershell" code={`
# 1. Tab Search → tìm model (llama, deepseek, mistral...)
# 2. Download model GGUF
# 3. Tab Chat → chọn model → Start Server

# REST API (port 1234, tương thích OpenAI)
curl http://localhost:1234/v1/chat/completions -H "Content-Type: application/json" -d "{""model"": ""local-model"", ""messages"": [{""role"": ""user"", ""content"": ""Hello""}]}"
`} />

## Xác minh cài đặt

Mở LM Studio → Help → About để kiểm tra version.

## Lưu ý

LM Studio chỉ hỗ trợ model GGUF format. GPU acceleration yêu cầu NVIDIA CUDA hoặc Vulkan. Server mặc định port 1234. Trên Windows 11, model 7B-14B là lựa chọn tốt nếu không có GPU mạnh.
