## Tổng quan

<Note type="info">
**Prompt Engineering Guide** (DAIR.AI) là comprehensive guide về prompt engineering techniques. Gồm zero-shot, few-shot, chain-of-thought, tree-of-thought, ReAct, và các advanced prompting strategies.
</Note>

## Kỹ thuật chính

<Note type="info">
- **Zero-shot**: Prompt trực tiếp không cần examples
- **Few-shot**: Cung cấp examples trong prompt
- **Chain-of-Thought**: Hướng dẫn model reasoning step-by-step
- **Tree-of-Thought**: Khám phá nhiều reasoning paths
- **ReAct**: Kết hợp reasoning + action (tool use)
</Note>
