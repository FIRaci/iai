## Tổng quan

Cline là VS Code extension biến AI thành autonomous agent có thể đọc/viết file, chạy terminal commands, browse web, và tương tác với browser. Hỗ trợ Claude, GPT-4, DeepSeek, và Ollama — cho phép AI thực thi task phức tạp mà không cần can thiệp thủ công.

## Cài đặt

Cài từ VS Code Marketplace.

<CodeBlock language="powershell" code={`
# Cài từ VS Code Extensions marketplace
code --install-extension saoudrizwan.claude-dev
`} />

## Sử dụng

Mở Cline từ sidebar và cấu hình API provider.

<CodeBlock language="powershell" code={`
# 1. Click icon Cline ở sidebar VS Code
# 2. Chọn API provider (Anthropic, OpenAI, OpenRouter, Ollama...)
# 3. Nhập API key
# 4. Bắt đầu gõ task

# Ví dụ task:
# "Create a React component that fetches data from an API
#  and displays it in a table with pagination"

# Cline có thể:
# - Đọc và edit files trong project
# - Chạy npm/pip/terminal commands
# - Mở browser và tương tác với web
# - Search web để tìm giải pháp
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
# Kiểm tra Cline icon ở VS Code sidebar
# Mở View → Extensions → tìm "Cline"
`} />

## Lưu ý

Cline có quyền truy cập đầy đủ vào terminal và file system — luôn review changes trước khi approve. Trên Windows 11, Cline hoạt động tốt với PowerShell 7 và Windows Terminal. Có thể dùng Ollama để chạy local model miễn phí.
