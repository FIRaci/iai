## Tổng quan

<Note type="info">
**Skills** (Vercel) là open-source CLI và ecosystem cho AI agent skills — `npx skills`. Cho phép cài đặt, quản lý, và chia sẻ skill files (SKILL.md) giữa 55+ AI coding agents.
</Note>

## Cài đặt

<CodeBlock language="bash" code={`# Cài đặt skills từ marketplace
npx skills add vercel-labs/agent-skills

# Cài đặt cho tất cả agents
npx skills add vercel-labs/agent-skills --all

# List tất cả skills có sẵn
npx skills list

# Tìm skill theo keyword
npx skills find react

# Xoá skill
npx skills remove my-skill`} />

## Sử dụng

<CodeBlock language="bash" code={`# Cài đặt specific skills
npx skills add vercel-labs/agent-skills -s react,nextjs --agent claude-code

# Cài đặt global (dùng cho mọi project)
npx skills add vercel-labs/agent-skills -g

# Tạo skill mới
npx skills init my-skill`} />

## Supported Agents

<Note type="info">
Hỗ trợ Claude Code, OpenCode, Cursor, Codex, Gemini CLI, GitHub Copilot, Cline, Windsurf, Aider, Antigravity, và 45+ agents khác.
</Note>
