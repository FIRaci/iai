## Tổng quan

Windsurf (trước đây là Codeium) là AI-native IDE được xây dựng trên VS Code, cung cấp autocomplete miễn phí, chat với context-aware, và agent mode tự động hoá workflow. Đặc biệt mạnh với tính năng Cascade — agent hiểu toàn bộ codebase và thực thi lệnh tự động.

## Cài đặt

Tải installer từ windsurf.com và cài đặt như ứng dụng Windows thông thường.

<CodeBlock language="powershell" code={`
# Option 1: Download from windsurf.com
# Visit https://codeium.com/windsurf/download and run the installer

# Option 2: Install via winget
winget install Codeium.Windsurf
`} />

## Sử dụng

Mở Windsurf và bắt đầu với các phím tắt quen thuộc.

<CodeBlock language="powershell" code={`
# Cascade (Agent mode):
# Ctrl+Shift+I - Mở Cascade để thực thi task tự động
# Cascade có thể: đọc file, edit code, chạy terminal, search web

# Chat:
# Ctrl+L - Chat với context từ file hiện tại

# Autocomplete: Tự động chạy khi gõ code
# Tab để accept suggestion

# Terminal: Windsurf CLI
windsurf --version
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
windsurf --version
# Ví dụ output: 1.2.34
`} />

## Lưu ý

Windsurf có bản miễn phí không giới hạn autocomplete. Tính năng Cascade yêu cầu tài khoản Pro. Trên Windows 11, Windsurf tương thích tốt với WSL và Dev Drive.
