## Tổng quan

<Note type="info">
**Windsurf** là AI IDE mới từ Codeium. Nổi bật với Cascade feature và AI Flow.
</Note>

## Cài đặt

<StepBlock number={1} title="Tải Windsurf">
https://codeium.com/windsurf → Download cho Windows
</StepBlock>

<StepBlock number={2} title="Cài đặt">
Chạy installer → Sign in với Google/GitHub
</StepBlock>

## Cascade

Cascade là AI assistant hiểu context toàn bộ codebase:

<CodeBlock language="text">Cascade: "add authentication to this API"
Cascade: "explain the database schema"
Cascade: "find where the user login is handled"</CodeBlock>

## AI Flow

Tự động multi-file edit:
1. Prompt → AI phân tích
2. AI suggest changes trên nhiều file
3. Review từng change → Accept/Reject

<Note type="tip">
Windsurf có free tier với limited credits. Premium $15/tháng.
</Note>
