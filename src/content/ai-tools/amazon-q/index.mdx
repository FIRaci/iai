## Tổng quan

<Note type="info">
**Amazon Q Developer** là AI assistant của AWS dành cho developers. Giúp viết code, debug, và optimize applications trong AWS ecosystem.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -g @aws-amplify/cli`} />

## Sử dụng

<CodeBlock language="powershell" code={`amplify configure
# Sau đó:
amplify add api
amplify push`} />

## Tips

<Note type="tip">
**AWS Integration:** Kết nối trực tiếp với AWS services như Lambda, DynamoDB, S3.
</Note>
