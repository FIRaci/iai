## Tổng quan

<Note type="info">
**Amp** là AI coding agent đa mô hình mới cho 2026. Chạy trong terminal, hỗ trợ nhiều models (Claude, GPT, Gemini) cho agentic coding workflows.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -g @amp-dev/cli
amp init`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Start coding session
amp

# Assign task
"Refactor the authentication module to use JWT"

# Amp sẽ đọc codebase, propose changes, và chờ review`} />

## Tips

<Note type="tip">
**Multi-model:** Amp hỗ trợ switch giữa Claude, GPT, và Gemini - chọn model phù hợp cho từng task.
</Note>
