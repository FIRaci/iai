## Tổng quan

<Note type="info">
**Gemini CLI** là AI agent cho terminal từ Google. Powered by Gemini 3.5 Flash, hỗ trợ code understanding, file manipulation, và command execution.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -g @anthropic-ai/gemini-cli
gemini login`} />

## Sử dụng

<CodeBlock language="powershell" code={`# Start Gemini CLI
gemini

# Ask questions hoặc yêu cầu
"Explain this codebase"
"Refactor the auth module"
"Run tests and fix failures" `} />

## Tips

<Note type="tip">
**Free tier:** Gemini CLI có free tier với usage limits cao.
</Note>
