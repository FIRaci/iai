## Tổng quan

<Note type="info">
**AWS Kiro** là AI-powered development tool từ AWS. Giúp developers viết code, debug, và optimize applications nhanh hơn.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -g @aws/kiro`} />

## Sử dụng

<CodeBlock language="powershell" code={`kiro init
kiro generate "Create a REST API for user management"
kiro deploy`} />

## Tips

<Note type="tip">
**AWS Integration:** Kết nối với AWS services để deploy và test applications.
</Note>
