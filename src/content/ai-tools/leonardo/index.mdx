## Tổng quan

<Note type="info">
Leonardo AI là nền tảng tạo ảnh AI, đặc biệt mạnh cho game assets, concept art và illustration.
</Note>

## Sử dụng

### Truy cập

Đăng ký tại [leonardo.ai](https://leonardo.ai) — free tier: 150 tokens/ngày.

### Tính năng

| Tính năng | Mô tả |
|-----------|-------|
| Image Generation | Tạo ảnh từ text |
| Canvas Editor | Chỉnh sửa ảnh AI |
| Texture Generation | Tạo texture cho 3D |
| Realtime Canvas | Vẽ realtime với AI |
| Motion | Tạo animation từ ảnh |

## Tips sử dụng

<Note type="tip">
**Game Assets:** Dùng model "Leonardo Diffusion XL" cho game assets chất lượng cao.
</Note>
