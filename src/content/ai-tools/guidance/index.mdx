## Tổng quan

<Note type="info">
**Guidance** là framework của Microsoft cho structured text generation với LLMs. Cho phép kiểm soát output format (JSON, regex, grammar) và interleave generation với logic điều khiển (if/else, loops).
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install guidance`} />

## Sử dụng

<CodeBlock language="powershell" code={`import guidance
program = guidance("""
{{#system}}You are a helpful assistant.{{/system}}
{{#user}}Tell me about {{topic}}{{/user}}
{{#assistant}}{{gen 'response' max_tokens=100}}{{/assistant}}
""")
result = program(topic="AI")
print(result["response"])`} />
