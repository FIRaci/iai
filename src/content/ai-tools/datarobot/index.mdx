## Tổng quan

<Note type="info">
**DataRobot** là enterprise AI/ML platform hỗ trợ automated machine learning (AutoML), model deployment, MLOps, và generative AI. Cho phép xây dựng và deploy models với minimal code.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install datarobot`} />

## Sử dụng

<CodeBlock language="powershell" code={`import datarobot as dr
dr.Client(token="your-token", endpoint="https://app.datarobot.com/api/v2")
# Train model tu dong
project = dr.Project.start("my-project")
project.analyze_and_model("target_column")
best_model = project.get_models()[0]
# Deploy
prediction_server = dr.PredictionServer.get()
deployment = dr.Deployment.create_from_learning_model(best_model.id)`} />
