## Tổng quan

<Note type="info">
**OpenClaw** là AI coding tool mã nguồn mở, hỗ trợ vibe coding trong terminal.
</Note>

## Cài đặt

<StepBlock number={1} title="Cài đặt">
<CodeBlock language="powershell">npm install -g openclaw</CodeBlock>
</StepBlock>

<StepBlock number={2} title="Cấu hình API key">
<CodeBlock language="powershell">openclaw config set api-key "your-key-here"</CodeBlock>
</StepBlock>

<StepBlock number={3} title="Bắt đầu">
<CodeBlock language="powershell">openclaw</CodeBlock>
</StepBlock>

## Sử dụng

<CodeBlock language="text">openclaw "tạo React component cho user profile"
openclaw "giải thích code trong file này"
openclaw "fix bug trong function calculateTotal"</CodeBlock>
