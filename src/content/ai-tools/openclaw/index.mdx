## Tổng quan

<Note type="info">
**OpenClaw** là AI coding tool mã nguồn mở, hoàn toàn miễn phí, hỗ trợ vibe coding trong terminal. Là lựa chọn tuyệt vời cho người dùng muốn trải nghiệm AI coding không tốn phí.
</Note>

## Yêu cầu

- **Node.js >= 18** (xem [hướng dẫn cài Node.js](/dev-tools/nodejs))
- API key từ Claude (Anthropic) hoặc OpenAI
- Terminal (PowerShell/CMD)

## Cài đặt

<StepBlock number={1} title="Cài đặt OpenClaw">
<CodeBlock language="powershell">npm install -g openclaw</CodeBlock>
</StepBlock>

<StepBlock number={2} title="Cấu hình API key">
<CodeBlock language="powershell">openclaw config set api-key "your-claude-api-key-here"</CodeBlock>
Hoặc dùng OpenAI:
<CodeBlock language="powershell">openclaw config set provider openai
openclaw config set api-key "sk-your-openai-key"</CodeBlock>
</StepBlock>

<StepBlock number={3} title="Kiểm tra">
<CodeBlock language="powershell">openclaw --version</CodeBlock>
</StepBlock>

<StepBlock number={4} title="Bắt đầu">
<CodeBlock language="powershell">openclaw</CodeBlock>
Mở interactive mode, hoặc dùng command trực tiếp:
<CodeBlock language="powershell">openclaw "tạo React component cho user profile"</CodeBlock>
</StepBlock>

## OpenClaw vs Claude Code CLI

| Tính năng | OpenClaw | Claude Code CLI |
|-----------|----------|-----------------|
| **Giá** | Free + API key | Free + API key |
| **API** | Claude + OpenAI | Claude only |
| **Mã nguồn** | Có MIT | Không Đóng |
| **File context** | Có Cơ bản | Có Nâng cao |
| **Terminal** | Có | Có Native |
| **Vibe coding** | Có | Có |

## Khắc phục sự cố

### "command not found: openclaw"

Kiểm tra npm global path:
<CodeBlock language="powershell">npm list -g openclaw
Get-Command openclaw -ErrorAction SilentlyContinue</CodeBlock>
Nếu không thấy, cài lại:
<CodeBlock language="powershell">npm install -g openclaw</CodeBlock>

### API key lỗi

Kiểm tra key hợp lệ:
<CodeBlock language="powershell">openclaw config get api-key
openclaw config set api-key "sk-ant-your-correct-key"</CodeBlock>

### Rate limit

Nếu bạn gặp rate limit, giảm tần suất request hoặc nâng cấp API tier.
