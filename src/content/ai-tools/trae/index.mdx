## Tổng quan

<Note type="info">
**Trae** là AI-native IDE từ ByteDance, đối thủ của Cursor và Windsurf. Có built-in Claude và GPT models, agent context management thông minh, hoàn toàn miễn phí tính đến giữa 2026.
</Note>

## Cài đặt

<CodeBlock language="bash" code={`# Download từ website
# https://www.trae.ai

# macOS
brew install --cask trae

# Windows: download installer từ trang chủ`} />

## Sử dụng

<CodeBlock language="bash" code={`# Mở project
trae .

# Composer mode (multi-file edits)
# Ctrl+K -> describe what you want

# Agent mode
# Ctrl+Shift+P -> "Agent: Start"

# Inline editing
# Select code -> Ctrl+I`} />

## Tính năng nổi bật

<Note type="tip">
**Miễn phí:** Trae hiện tại free hoàn toàn — không giới hạn số lượng requests như Cursor.
</Note>

<Note type="tip">
**Agent context:** Context management khác biệt so với Cursor/Windsurf, tối ưu cho iterative development.
</Note>
