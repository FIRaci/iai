## Tổng quan

<Note type="info">
**ClaudeKit** là framework orchestration cho Claude AI agents. Giúp quản lý skills, hooks, agents, và workflows — cho phép xây dựng hệ thống AI phức tạp từ các module nhỏ.
</Note>

## Yêu cầu

- **Node.js >= 18** (xem [hướng dẫn cài Node.js](/dev-tools/nodejs))
- **Git** (xem [hướng dẫn cài Git](/dev-tools/git))
- **pnpm** (khuyến khích dùng thay npm)
- Kiến thức cơ bản về CLI và JavaScript/TypeScript

## Cài đặt

<StepBlock number={1} title="Clone repository">
<CodeBlock language="powershell">git clone https://github.com/anomalyco/claudekit.git
cd claudekit</CodeBlock>
</StepBlock>

<StepBlock number={2} title="Cài đặt dependencies">
<CodeBlock language="powershell">pnpm install</CodeBlock>
</StepBlock>

<StepBlock number={3} title="Cấu hình">
Copy file env mẫu:
<CodeBlock language="powershell">cp .env.example .env</CodeBlock>
Thêm API keys vào `.env`:
<CodeBlock language="text">ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...</CodeBlock>
</StepBlock>

<StepBlock number={4} title="Kiểm tra">
<CodeBlock language="powershell">pnpm dev</CodeBlock>
</StepBlock>

## Cấu trúc

```
claudekit/
├── skills/          # Skill definitions
├── hooks/           # Hook system
├── agents/          # Agent configurations
├── rules/           # Development rules
└── scripts/         # Utility scripts
```

## Viết skill đầu tiên

Tạo file `skills/my-skill.ts`:
<CodeBlock language="typescript" code={`import { Skill } from "@claudekit/core"

export const mySkill: Skill = {
  name: "hello-world",
  description: "Skill chào hỏi đơn giản",
  execute: async (input: string) => { return "Xin chào: " + input },
}
`} />

## Sử dụng cơ bản

Tạo `workflows/hello.yml`:
<CodeBlock language="yaml" code={`name: hello-workflow
steps:
  - skill: hello-world
    input: "Hello ClaudeKit!"`} />

Chạy:
<CodeBlock language="powershell">pnpm run workflow hello</CodeBlock>

## Khắc phục sự cố

### Lỗi "Cannot find module"

Dependencies chưa được cài đúng:
<CodeBlock language="powershell">rm -rf node_modules && pnpm install</CodeBlock>

### API key không hợp lệ

Kiểm tra file `.env` có đúng key không:
<CodeBlock language="powershell">Get-Content .env</CodeBlock>
Đảm bảo key bắt đầu bằng `sk-ant-` cho Anthropic hoặc `sk-` cho OpenAI.

### Port already in use

Mặc định ClaudeKit dùng port 3000. Đổi port:
<CodeBlock language="powershell">$env:PORT=3001; pnpm dev</CodeBlock>
