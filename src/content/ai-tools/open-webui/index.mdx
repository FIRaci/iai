## Tổng quan

Open WebUI là giao diện web chat giống ChatGPT nhưng self-hosted, kết nối với Ollama hoặc OpenAI-compatible API backend. Hỗ trợ đa người dùng, RAG (Retrieval-Augmented Generation), voice input, image generation, và quản lý model tập trung.

## Cài đặt

Chạy qua Docker hoặc cài trực tiếp với Python.

 <CodeBlock language="powershell" code={`
# Yeu cau: Docker Desktop cho Windows
docker run -d -p 3000:8080 -v open-webui-data:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:main

# Mo http://localhost:3000
`} />

## Sử dụng

Đăng ký tài khoản admin và bắt đầu chat.

<CodeBlock language="powershell" code={`
# 1. Mở http://localhost:3000
# 2. Đăng ký tài khoản admin
# 3. Settings → Connections → Ollama
#    URL: http://host.docker.internal:11434

# Hoặc kết nối OpenAI:
# Settings → Connections → OpenAI
# API Key: sk-...
# API URL: https://api.openai.com/v1

# Tính năng:
# - Chat với nhiều model cùng lúc
# - Upload PDF/DOCX/Code để RAG
# - Tạo custom GPTs (Workspaces)
# - Voice input
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
docker ps --filter name=open-webui
# Container phải có status "Up"
`} />

## Lưu ý

Open WebUI kết nối với Ollama qua `host.docker.internal` khi chạy Docker trên Windows. Nếu Ollama chạy trên máy host, đảm bảo Ollama bind address là `0.0.0.0`. Có thể dùng với LM Studio, text-generation-webui, hoặc bất kỳ OpenAI-compatible API nào.
