## Tổng quan

Continue là open-source AI coding assistant cho VS Code và JetBrains, cho phép kết nối với bất kỳ LLM nào (Ollama, OpenAI, Anthropic, etc.). Hỗ trợ chat context-aware, tab autocomplete, và inline editing — tất cả đều tự host và customize được hoàn toàn.

## Cài đặt

Cài extension từ VS Code Marketplace hoặc JetBrains Marketplace.

<CodeBlock language="powershell" code={`
# VS Code: Cài từ marketplace
# Mở VS Code → Extensions → Search "Continue"

# Hoặc cài từ CLI:
code --install-extension Continue.continue
`} />

## Sử dụng

Cấu hình model provider và bắt đầu sử dụng chat và autocomplete.

<CodeBlock language="powershell" code={`
# Mở file config.json (Ctrl+Shift+P → Continue: Open config)
# ~/.continue/config.json:
# {
#   "models": [{
#     "title": "Claude 3.5 Sonnet",
#     "provider": "anthropic",
#     "model": "claude-sonnet-4-20250514"
#   }],
#   "tabAutocompleteModel": {
#     "title": "Codestral",
#     "provider": "mistral",
#     "model": "codestral-latest"
#   }
# }

# Phím tắt:
# Ctrl+L - Mở chat tab
# Ctrl+I - Inline edit
# Ctrl+Shift+L - Quick chat
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
# Mở VS Code → View → Extensions → Continue
# Kiểm tra icon Continue xuất hiện ở sidebar
`} />

## Lưu ý

Continue hoàn toàn miễn phí và open-source. Bạn tự cung cấp API key cho model bạn muốn dùng. Có thể dùng Ollama local để không tốn phí API. Trên Windows 11, nếu dùng Ollama, đảm bảo Ollama chạy background trước khi dùng Continue.
