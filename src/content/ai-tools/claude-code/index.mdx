## Tổng quan

<Note type="info">
**Claude Code** là công cụ AI coding mạnh mẽ từ Anthropic, chạy trực tiếp trong terminal. Nó cho phép bạn lập trình bằng tiếng Việt thông qua các câu lệnh tự nhiên (vibe coding) — từ sửa bug, thêm tính năng, đến refactor code.
</Note>

## Yêu cầu

- **Node.js >= 18** (xem [hướng dẫn cài Node.js](/dev-tools/nodejs))
- **API key Anthropic** (đăng ký tại console.anthropic.com)
- Terminal (PowerShell khuyên dùng)

## Cài đặt

<StepBlock number={1} title="Cài đặt Node.js">
<CodeBlock language="powershell">winget install OpenJS.NodeJS.LTS
node --version</CodeBlock>
</StepBlock>

<StepBlock number={2} title="Cài đặt Claude Code">
<CodeBlock language="powershell">npm install -g @anthropic-ai/claude-code
claude --version</CodeBlock>
</StepBlock>

<StepBlock number={3} title="Lấy API Key">
1. Truy cập https://console.anthropic.com
2. Đăng ký tài khoản
3. Vào API Keys → Create Key
4. Copy key (bắt đầu bằng `sk-ant-...`)
</StepBlock>

<StepBlock number={4} title="Cấu hình và chạy">
Mở terminal tại project của bạn:
<CodeBlock language="powershell">cd du-an-cua-ban
claude</CodeBlock>
Lần đầu chạy, Claude Code sẽ yêu cầu nhập API key. Dán key và Enter.
</StepBlock>

## Sử dụng cơ bản

<Note type="tip">
Claude Code hiểu tiếng Việt! Bạn có thể prompt bằng tiếng Việt và Claude sẽ trả lời bằng tiếng Việt.
</Note>

### Các lệnh cơ bản

| Lệnh | Mô tả |
|------|-------|
| `claude` | Mở Claude Code trong terminal |
| `claude "viết function API login bằng TypeScript"` | Chạy task trực tiếp |
| `claude --model claude-sonnet-4-20250514` | Chọn model cụ thể |
| `/help` | Xem trợ giúp trong Claude Code |

## Tips

<Note type="tip">
- Dùng `Ctrl+C` để interrupt khi Claude đang chạy
- Claude Code tự động tạo file `.claude/settings.json` để cấu hình rules
- Dùng `claude --dangerously-skip-permissions` để bỏ qua confirm (cẩn thận với quyền này!)
- Claude Code tự động đọc `.gitignore` để biết files nào không cần xử lý
</Note>

## Claude Code Rules

Tạo file `.claude/settings.json` trong project root:
<CodeBlock language="json" code={`{
  "allowedTools": ["Read", "Write", "Edit", "Bash", "Search"],
  "rules": [
    "Luôn dùng TypeScript với strict mode",
    "Viết unit test cho mọi function mới",
    "Follow functional components pattern",
    "Dùng tiếng Việt cho comments"
  ]
}`} />

## Khắc phục sự cố

### "claude not recognized"

Lỗi này xảy ra khi npm global path không trong PATH:
<CodeBlock language="powershell">$env:Path += ";$env:APPDATA\npm"
claude --version</CodeBlock>
Để fix vĩnh viễn, thêm vào PATH:
<CodeBlock language="powershell">[Environment]::SetEnvironmentVariable("Path", "$env:Path;$env:APPDATA\npm", "User")</CodeBlock>

### API key không hợp lệ

Set lại key:
<CodeBlock language="powershell">$env:ANTHROPIC_API_KEY="sk-ant-your-key-here"
claude</CodeBlock>
Hoặc tạo file `.env` trong project:
<CodeBlock language="text">ANTHROPIC_API_KEY=sk-ant-your-key-here</CodeBlock>

### Claude Code quá chậm

- Dùng model nhỏ hơn: `claude --model claude-sonnet-4-20250514`
- Giới hạn context bằng `--max-thinking-tokens 8000`
- Chia task lớn thành nhiều task nhỏ
