## Tổng quan

<Note type="info">
**Claude Code** là công cụ AI coding mạnh mẽ từ Anthropic, chạy trực tiếp trong terminal. Nó cho phép bạn lập trình bằng tiếng Việt thông qua các câu lệnh tự nhiên (vibe coding).
</Note>

## Cài đặt

<StepBlock number={1} title="Cài đặt Node.js">
Mở **PowerShell (Admin)** và chạy:

<CodeBlock language="powershell">winget install OpenJS.NodeJS.LTS</CodeBlock>

Kiểm tra:
<CodeBlock language="powershell">node --version && npm --version</CodeBlock>
</StepBlock>

<StepBlock number={2} title="Cài đặt Claude Code">
<CodeBlock language="powershell">npm install -g @anthropic-ai/claude-code</CodeBlock>
</StepBlock>

<StepBlock number={3} title="Lấy API Key">
1. Truy cập https://console.anthropic.com
2. Đăng ký tài khoản
3. Tạo API key mới
4. Copy key để sử dụng
</StepBlock>

<StepBlock number={4} title="Cấu hình lần đầu">
Mở terminal tại project của bạn:
<CodeBlock language="powershell">cd du-an-cua-ban
claude</CodeBlock>
Lần đầu chạy, Claude Code sẽ yêu cầu nhập API key.
</StepBlock>

## Sử dụng cơ bản

<Note type="tip">
Claude Code hiểu tiếng Việt! Bạn có thể prompt bằng tiếng Việt.
</Note>

### Các lệnh cơ bản

| Lệnh | Mô tả |
|------|-------|
| `claude` | Mở Claude Code trong terminal |
| `claude "viết function API login bằng TypeScript"` | Chạy task trực tiếp |
| `claude --model claude-sonnet-4-20250514` | Chọn model cụ thể |
| `/help` | Xem trợ giúp trong Claude Code |

## Tips

<Note type="tip">
- Dùng `Cmd+I` để interrupt khi Claude đang chạy
- Dùng `/init` để tạo project structure mới
- Dùng `claude --dangerously-skip-permissions` để bỏ qua confirm (cẩn thận!)
</Note>

## Troubleshooting

<Note type="warning">
**Lỗi: "claude not recognized"**
→ Chạy lại terminal hoặc thêm npm global path vào PATH:
<CodeBlock language="powershell">$env:Path += ";$env:APPDATA\npm"</CodeBlock>
</Note>
