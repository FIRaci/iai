## Tổng quan

<Note type="info">
**Axolotl** là tool để fine-tune LLMs (Llama, Mistral, v.v.) dễ dàng. Hỗ trợ LoRA, QLoRA, full fine-tuning.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install axolotl`} />

## Sử dụng

<CodeBlock language="powershell" code={`accelerate launch -m axolotl.cli.train config.yaml
# Config file định nghĩa model, dataset, training params`} />

## Tips

<Note type="tip">
**QLoRA:** Sử dụng QLoRA để fine-tune models lớn trên GPU nhỏ.
</Note>
