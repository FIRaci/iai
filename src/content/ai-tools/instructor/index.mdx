## Tổng quan

<Note type="info">
**Instructor** là Python library cho structured LLM outputs. Sử dụng Pydantic để định nghĩa response schema, tự động retry với validation, hỗ trợ OpenAI, Anthropic, Gemini, và Mistral APIs.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install instructor`} />

## Sử dụng

<CodeBlock language="powershell" code={`import instructor
from pydantic import BaseModel
from openai import OpenAI

client = instructor.from_openai(OpenAI())

class User(BaseModel):
    name: str
    age: int

user = client.chat.completions.create(
    model="gpt-4o",
    response_model=User,
    messages=[{"role": "user", "content": "Create John, 25"}]
)`} />
