## Tổng quan

<Note type="info">
**LMSYS Chatbot Arena** (arena.ai) là platform crowdsourced để so sánh và benchmark AI models. Người dùng chat blind với 2 models anonymous, vote model nào tốt hơn — tạo ra leaderboard khách quan nhất.
</Note>

## Truy cập

Truy cập [arena.ai](https://arena.ai) hoặc [chat.lmsys.org](https://chat.lmsys.org)

## Cách hoạt động

### Blind Test

1. Nhập prompt
2. Nhận response từ 2 models ẩn danh (Model A và Model B)
3. Vote: A tốt hơn, B tốt hơn, hòa, hoặc cả hai đều tệ
4. Sau vote, danh tính models được reveal

### Leaderboard

Kết quả voting tạo ra **Elo rating** cho mỗi model, xếp hạng trên leaderboard.

## Categories

| Category | Description |
|----------|-------------|
| **Overall** | General chat quality |
| **Coding** | Code generation & debugging |
| **Hard Prompts** | Complex, multi-step tasks |
| **Long Context** | Long document understanding |
| **Math** | Mathematical reasoning |
| **Language** | Multi-language support |

## Sử dụng

### Vote models

1. Vào [chat.lmsys.org](https://chat.lmsys.org)
2. Chọn "Arena (battle)" mode
3. Gõ prompt bất kỳ
4. So sánh 2 responses và vote

### Xem Leaderboard

1. Vào tab **Leaderboard**
2. Filter theo category (coding, math, v.v.)
3. Xem Elo ratings và rankings

## Tips

<Note type="tip">
**Coding leaderboard:** Filter theo "Coding" category để xem models nào tốt nhất cho development.
</Note>

<Note type="tip">
**Free access:** Hoàn toàn miễn phí, không cần account.
</Note>

<Note type="tip">
**Real-world benchmark:** Khác với synthetic benchmarks — đây là đánh giá từ actual users với real tasks.
</Note>

## Tại sao quan trọng?

- **Khách quan:** Blind voting eliminates brand bias
- **Cập nhật:** Leaderboard update liên tục khi có models mới
- **Đa dạng:** 100+ models được so sánh
- **Community-driven:** Hàng trăm ngàn votes từ real users
