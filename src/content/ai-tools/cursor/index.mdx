## Tổng quan

Cursor là AI-native code editor forked từ VS Code, tích hợp sâu các tính năng AI như autocomplete, chat context-aware, và agent mode. Hỗ trợ Claude, GPT-4, và nhiều model khác trực tiếp trong editor, giúp Windows developers tăng tốc độ code đáng kể.

## Cài đặt

Tải installer từ trang chủ và chạy file .exe.

<CodeBlock language="powershell" code={`
# Option 1: Download from cursor.com
# Visit https://cursor.com/downloads and run the installer

# Option 2: Install via winget
winget install Cursor.Cursor
`} />

## Sử dụng

Mở Cursor và bắt đầu dùng Composer (Ctrl+I) hoặc Chat (Ctrl+L) để tương tác với AI.

<CodeBlock language="powershell" code={`
# Các phím tắt chính trong Cursor:
# Ctrl+I  - Mở Composer (chỉnh sửa code theo hướng dẫn)
# Ctrl+L  - Mở Chat với context từ file hiện tại
# Ctrl+K  - Generate code từ comment
# Ctrl+Enter - Agent mode (tự động tạo/sửa nhiều file)

# Terminal: Dùng Cursor CLI
cursor --help
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`

`} />

## Lưu ý

Cursor yêu cầu đăng nhập tài khoản để dùng bản dùng thử. Bản Pro ($20/tháng) bao gồm 500 fast requests/tháng. Trên Windows 11, đảm bảo đã cài VC++ Redistributable nếu gặp lỗi khi chạy.
