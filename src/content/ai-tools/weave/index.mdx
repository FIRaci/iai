## Tổng quan

<Note type="info">
**Weave** là LLM ops platform của Weights & Biases. Cung cấp LLM monitoring, prompt versioning, dataset management, evaluation, và tracing cho production LLM applications.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install weave`} />
