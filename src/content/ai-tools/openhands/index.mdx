## Tổng quan

OpenHands (trước đây là OpenDevin) là nền tảng AI agent mã nguồn mở chạy trong Docker, cho phép AI thực thi các task lập trình phức tạp một cách tự động. Agent có thể viết code, chạy lệnh, tạo file, và quản lý toàn bộ dự án phần mềm từ terminal đến deployment.

## Cài đặt

Chạy qua Docker trên Windows 11.

<CodeBlock language="powershell" code={`
# Yeu cau: Docker Desktop cho Windows
docker pull ghcr.io/all-hands-ai/openhands:latest
docker run -it --rm --pull=always -e LLM_API_KEY="sk-ant-..." -e LLM_MODEL="claude-sonnet-4-20250514" -v "$pwd:/workspace" -p 3000:3000 ghcr.io/all-hands-ai/openhands:latest
`} />

## Sử dụng

Mở http://localhost:3000 và giao tiếp với agent qua web UI.

<CodeBlock language="powershell" code={`
# Trong web UI, go task nhu: "Build a REST API with Express.js, TypeScript, and SQLite"

# Hoac dung CLI (CodeAct):
docker run -it --rm --pull=always -e LLM_API_KEY="sk-ant-..." -e LLM_MODEL="claude-sonnet-4-20250514" -v "$pwd:/workspace" ghcr.io/all-hands-ai/openhands:latest python -m openhands.core.main -t "Create a Python script"

# File output se o trong thu muc hien tai
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
docker run --rm ghcr.io/all-hands-ai/openhands:latest --help
`} />

## Lưu ý

OpenHands yêu cầu Docker Desktop chạy trên Windows 11. Đảm bảo WSL2 backend được cấu hình trong Docker Desktop. Agent có quyền chạy terminal commands — luôn sandbox trong container. Tốn nhiều tokens API vì agent chạy nhiều bước.
