## Tổng quan

Aider là AI pair programming tool chạy trong terminal, cho phép chat với AI để chỉnh sửa code files và tự động commit changes vào Git. Hỗ trợ nhiều LLM (Claude, GPT-4, DeepSeek, Ollama) và có khả năng hiểu toàn bộ codebase để đưa ra suggestion chính xác.

## Cài đặt

Cài qua pip trong Python environment.

<CodeBlock language="powershell" code={`
# Cài đặt Aider
pip install aider-chat

# Hoặc cài với hỗ trợ browser (cho voice mode)
pip install "aider-chat[browser]"

# Yêu cầu Python 3.10+
python --version
`} />

## Sử dụng

Chạy Aider trong terminal với files cần chỉnh sửa.

<CodeBlock language="powershell" code={`
# Chạy với API key
set ANTHROPIC_API_KEY=sk-ant-...
aider --model claude-sonnet-4-20250514 main.py utils.js

# Hoặc dùng Ollama local
aider --model ollama/llama3.3-70b --ollama-api-base http://localhost:11434

# Các lệnh trong chat:
# /add file.py       - Thêm file vào context
# /drop file.py      - Bỏ file khỏi context
# /commit            - Commit changes
# /undo              - Undo last change
# /help              - Xem danh sách commands
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
aider --version

`} />

## Lưu ý

Aider tự động tạo Git commits cho mỗi thay đổi — luôn chạy trong repo Git. Trên Windows 11, Aider hoạt động tốt trong PowerShell 7, Windows Terminal, và WSL. Nếu gặp lỗi encoding, set `$env:PYTHONIOENCODING='utf-8'`.
