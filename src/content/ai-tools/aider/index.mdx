## Tổng quan

<Note type="info">
**Aider** là AI pair programming tool chạy trong terminal. Chat với AI để edit code files, commit changes tự động vào Git. Hỗ trợ nhiều ngôn ngữ lập trình.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install aider-chat`} />

## Sử dụng

<CodeBlock language="powershell" code={`aider file1.py file2.py
# Trong terminal:
# /add file3.py
# Implement a function to sort users by name`} />

## Tips

<Note type="tip">
**Git integration:** Aider tự động commit changes sau mỗi edit.
</Note>
