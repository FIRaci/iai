## Tổng quan

<Note type="info">
**AutoTrain** là Hugging Face tool để auto-train ML models mà không cần code. Hỗ trợ text classification, NER, image classification.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install autotrain-advanced`} />

## Sử dụng

<CodeBlock language="powershell" code={`autotrain --config config.yaml
# Hoặc sử dụng AutoTrain UI trên Hugging Face Spaces`} />

## Tips

<Note type="tip">
**Hugging Face Integration:** Push trained models trực tiếp lên Hugging Face Hub.
</Note>
