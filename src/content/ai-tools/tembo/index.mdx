## Tổng quan

<Note type="info">
**Tembo** là AI agent orchestration platform. Chạy Claude Code, Cursor, Codex như background agents across multiple repos. Trigger từ Slack, Linear, GitHub.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`# Truy cập web app tại tembo.io`} />

## Sử dụng

<CodeBlock language="powershell" code={`# 1. Connect GitHub/GitLab
# 2. Tag @tembo trong Slack hoặc Linear
# 3. AI agent picks up task và chạy
# 4. Agent mở PR khi hoàn thành`} />

## Tips

<Note type="tip">
**Multi-agent:** Tembo orchestrate nhiều coding agents cùng lúc - Claude Code, Cursor, Codex, Gemini.
</Note>
