## Tổng quan

Claude Desktop là ứng dụng desktop của Anthropic cho phép sử dụng Claude AI trực tiếp trên máy tính Windows. Hỗ trợ đọc file, phân tích code, viết văn bản, và tương tác với file hệ thống. Tích hợp sâu với Claude Code cho terminal-based AI development.

## Cài đặt

Tải file .exe từ claude.ai và cài đặt.

<CodeBlock language="powershell" code={`
# Download from https://claude.ai/download
# Run the installer Claude Setup x.x.x.exe

# Đăng nhập bằng tài khoản Anthropic
# Yêu cầu Claude Pro subscription ($20/tháng)
`} />

## Sử dụng

Mở Claude Desktop và bắt đầu chat hoặc upload file để phân tích.

<CodeBlock language="powershell" code={`
# Claude Desktop hỗ trợ:
# - Chat trực tiếp với Claude
# - Upload file (code, PDF, hình ảnh) để phân tích
# - Copy-paste code để review/debug
# - Tích hợp Claude Code CLI: 
npx claude-code
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
# Mở Start Menu và tìm "Claude"
# Hoặc kiểm tra version trong Settings > About
`} />

## Lưu ý

Claude Desktop yêu cầu đăng nhập tài khoản Anthropic và subscription Pro. Trên Windows 11, đảm bảo .NET 8 runtime đã được cài đặt. Claude Desktop có thể đọc file local nhưng không có quyền edit file trực tiếp (dùng Claude Code cho việc đó).
