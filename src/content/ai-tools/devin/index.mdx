## Tổng quan

<Note type="info">
**Devin** là AI software engineer đầu tiên, hoạt động trên nền web. Có thể lập kế hoạch, code, test, và deploy.
</Note>

## Cách sử dụng

<StepBlock number={1} title="Truy cập Devin">
https://app.cognition.ai
</StepBlock>

<StepBlock number={2} title="Tạo task mới">
Mô tả dự án bạn muốn làm bằng tiếng Việt
</StepBlock>

<StepBlock number={3} title="Quan sát Devin làm việc">
Devin sẽ:
1. Lập kế hoạch
2. Viết code
3. Chạy terminal
4. Tạo Pull Request
</StepBlock>

## Tính năng

- **Browser**: Devin có thể dùng browser để test
- **Terminal**: Chạy commands trực tiếp
- **Code Editor**: Xem và edit code real-time
- **Slack**: Tích hợp Slack để nhận report

<Note type="warning">
Devin là paid service. Có free trial nhưng giới hạn.
</Note>
