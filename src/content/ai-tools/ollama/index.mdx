## Tổng quan

Ollama là tool chạy LLM local trên Windows, hỗ trợ Llama 3, DeepSeek, Mistral, Qwen, Phi-4 và nhiều model khác. Cung cấp REST API tương thích OpenAI, tích hợp với Continue, Cline, Open WebUI mà không cần internet.

## Cài đặt

Tải installer từ ollama.com và chạy.

<CodeBlock language="powershell" code={`
winget install Ollama.Ollama
`} />

Hoặc tải từ https://ollama.com/download/windows và chạy OllamaSetup.exe.

## Sử dụng

Sau khi cài, Ollama chạy background service. Dùng CLI để pull và chạy model.

<CodeBlock language="powershell" code={`
# Pull model
ollama pull llama3.3:70b
ollama pull deepseek-r1:14b

# Chat trực tiếp
ollama run llama3.3:70b

# REST API (port 11434)
curl http://localhost:11434/api/generate -d "{""model"": ""llama3.3:70b"", ""prompt"": ""Hello""}"

# List models
ollama list
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
ollama --version
`} />

## Lưu ý

Ollama chạy model trên GPU (NVIDIA CUDA) hoặc CPU. Trên Windows 11, GPU acceleration yêu cầu NVIDIA Driver và CUDA. Model càng lớn càng tốn RAM — Llama 3.3 70B cần ~40GB RAM, Mistral 7B cần ~8GB.
