## Tổng quan

<Note type="info">
**Kaggle** là platform cho data science competitions, datasets, và notebooks. Nguồn datasets và learning resources phổ biến.
</Note>

## Cài đặt CLI

<CodeBlock language="powershell" code={`pip install kaggle`} />

## Sử dụng

### Tải Dataset

<CodeBlock language="powershell" code={`kaggle datasets download -d username/dataset-name`} />

### Upload Dataset

<CodeBlock language="powershell" code={`kaggle datasets init -p /path/to/dataset
kaggle datasets create -p /path/to/dataset`} />
