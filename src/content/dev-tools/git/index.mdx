## Tổng quan

<Note type="info">
**Git** là version control system phổ biến nhất. **GitHub** là nền tảng lưu trữ và chia sẻ code.
</Note>

## Cài đặt

<StepBlock number={1} title="Cài Git cho Windows">
<CodeBlock language="powershell">winget install Git.Git</CodeBlock>
</StepBlock>

<StepBlock number={2} title="Cấu hình lần đầu">
<CodeBlock language="powershell">git config --global user.name "Your Name"
git config --global user.email "your@email.com"</CodeBlock>
</StepBlock>

<StepBlock number={3} title="Tạo SSH Key (cho GitHub)">
<CodeBlock language="powershell">ssh-keygen -t ed25519 -C "your@email.com"
Get-Content ~/.ssh/id_ed25519.pub | clip</CodeBlock>
Thêm key lên GitHub: Settings → SSH and GPG keys → New SSH key
</StepBlock>
