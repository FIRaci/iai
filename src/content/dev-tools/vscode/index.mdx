## Tổng quan

<Note type="info">
**VS Code** (Visual Studio Code) là code editor phổ biến nhất hiện nay, hỗ trợ mọi ngôn ngữ lập trình.
</Note>

## Cài đặt

<StepBlock number={1} title="Tải VS Code">
<CodeBlock language="powershell">winget install Microsoft.VisualStudioCode</CodeBlock>
Hoặc tải từ: https://code.visualstudio.com
</StepBlock>

<StepBlock number={2} title="Cài Code CLI">
Mở VS Code → `Ctrl+Shift+P` → "Shell Command: Install 'code' command in PATH"
</StepBlock>

## Essential Extensions cho AI Development

| Extension | Mục đích |
|-----------|----------|
| GitHub Copilot | AI pair programming |
| Cursor/Claude Code | AI chat trong editor |
| Prettier | Code formatter |
| ESLint | Linting |
| Tailwind CSS IntelliSense | Tailwind CSS hỗ trợ |
| GitLens | Git visualization |

## Shortcuts

| Shortcut | Hành động |
|----------|-----------|
| `Ctrl+P` | Quick open file |
| `Ctrl+Shift+P` | Command palette |
| `Ctrl+\`` | Toggle terminal |
| `Ctrl+B` | Toggle sidebar |
| `Alt+Up/Down` | Move line up/down |
