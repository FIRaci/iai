## Tổng quan

GitHub Desktop là Git client GUI chính thức của GitHub, giúp quản lý repositories Git trực quan mà không cần dùng command line. Hỗ trợ clone, commit, push, pull, branch, merge, và review changes.

## Cài đặt

### Phương pháp 1: Winget (khuyến nghị)

<CodeBlock language="powershell" code={`winget install GitHub.GitHubDesktop`} />

### Phương pháp 2: Tải trực tiếp

Tải từ [desktop.github.com](https://desktop.github.com) và chạy installer.

## Sử dụng

### Clone repository

1. File -> Clone Repository
2. Chọn repository từ GitHub (hoặc nhập URL)
3. Chọn local path -> Clone

### Commit và Push

1. Sửa code trong editor
2. GitHub Desktop tự động detect changes
3. Nhập Summary (bắt buộc) và Description
4. Commit to main
5. Push origin

### Branching

| Thao tác | Cách thực hiện |
|----------|----------------|
| Tạo branch | `Ctrl+Shift+N` -> nhập tên |
| Chuyển branch | Click branch name -> chọn |
| Merge | Branch -> Merge into current |
| Pull Request | Branch -> Create PR |

## Xác minh cài đặt

Mở GitHub Desktop từ Start Menu. Kiểm tra Help -> About GitHub Desktop.

## Lưu ý

<Note type="tip">
Dùng `Ctrl+Enter` để commit nhanh. `Ctrl+P` để push ngay sau commit.
</Note>

<Note type="warning">
GitHub Desktop không hỗ trợ tất cả Git features (rebase interactive, cherry-pick phức tạp). Với thao tác Git nâng cao, dùng command line.
</Note>
