## Tổng quan

<Note type="info">
GitHub Desktop là Git client GUI chính thức của GitHub, giúp quản lý repositories dễ dàng mà không cần command line.
</Note>

## Cài đặt trên Windows 11

### Bước 1: Tải GitHub Desktop

Truy cập [desktop.github.com](https://desktop.github.com) và tải bản Windows.

### Bước 2: Cài đặt

Chạy installer, đăng nhập bằng tài khoản GitHub.

## Sử dụng cơ bản

### Clone repository

1. File → Clone repository
2. Chọn repository từ GitHub
3. Chọn local path
4. Nhấp "Clone"

### Commit và Push

1. Thay đổi files trong repository
2. GitHub Desktop tự động detect changes
3. Viết commit message
4. Nhấp "Commit to main"
5. Nhấp "Push origin"

### Các tính năng chính

| Tính năng | Mô tả |
|-----------|-------|
| Clone | Tải repository từ GitHub |
| Commit | Lưu thay đổi |
| Push/Pull | Đồng bộ với remote |
| Branch | Quản lý branches |
| History | Xem lịch sử commits |
| Diff | Xem thay đổi chi tiết |

## Tips sử dụng

<Note type="tip">
**Keyboard shortcuts:** `Ctrl+1/2/3` để switch giữa Changes, History, Branches tabs.
</Note>

<Note type="warning">
GitHub Desktop không hỗ trợ tất cả Git features. Dùng command line cho advanced operations.
</Note>
