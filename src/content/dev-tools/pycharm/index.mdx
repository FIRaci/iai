## Tổng quan

<Note type="info">
**PyCharm** là IDE chuyên dụng cho Python development từ JetBrains. Hỗ trợ Django, Flask, FastAPI, data science, và machine learning. Tích hợp debugger, test runner, database tools, và version control.
</Note>

## Cài đặt trên Windows 11

### Bước 1: Tải PyCharm

Truy cập [jetbrains.com/pycharm](https://jetbrains.com/pycharm). Có 2 phiên bản:
- **Community:** Miễn phí, đủ cho Python cơ bản
- **Professional:** Trả phí, hỗ trợ web frameworks và database tools

### Bước 2: Cài đặt

1. Chạy installer
2. Chọn components (khuyến nghị chọn tất cả)
3. Hoàn tất cài đặt

### Bước 3: Cấu hình Python interpreter

1. Mở PyCharm
2. File → Settings → Project → Python Interpreter
3. Chọn Python installation hoặc tạo virtual environment

## Sử dụng cơ bản

### Tính năng chính

| Tính năng | Mô tả |
|-----------|-------|
| Smart Completion | Autocomplete thông minh cho Python |
| Refactoring | Rename, extract method, inline variable |
| Debugging | Debug Python code với breakpoints |
| Testing | Chạy pytest, unittest, doctest tích hợp |
| Database Tools | Quản lý database trong IDE (Professional) |
| Jupyter Notebooks | Hỗ trợ Jupyter notebooks tích hợp |

### Keyboard shortcuts

| Phím tắt | Chức năng |
|----------|-----------|
| `Ctrl+Shift+A` | Find action |
| `Ctrl+N` | Goto class |
| `Ctrl+Shift+N` | Goto file |
| `Shift+F10` | Run |
| `Shift+F9` | Debug |

## Tips sử dụng

<Note type="tip">
**Virtual Environment:** Luôn dùng virtual environment cho mỗi project. PyCharm tự động phát hiện và config venv khi mở project.
</Note>

<Note type="tip">
**Database Tools (Professional):** PyCharm Professional có database tools tích hợp — kết nối PostgreSQL, MySQL, SQLite trực tiếp trong IDE.
</Note>
