## Tổng quan

PyCharm là Python IDE chuyên nghiệp của JetBrains. Hỗ trợ Django, FastAPI, Flask, data science (NumPy, Pandas, Matplotlib), và machine learning. Có hai bản: Community (free) va Professional (paid).

## Cài đặt

<CodeBlock language="powershell" code={`winget install JetBrains.PyCharm.Community
# Hoac Professional: winget install JetBrains.PyCharm.Professional
# Tai tu: https://www.jetbrains.com/pycharm/download/`} />

## Sử dụng

### Code completion va navigation

<CodeBlock language="powershell" code={`
# Ctrl+Space: Basic completion
# Ctrl+Shift+Space: Type-aware completion
# Ctrl+P: Xem parameter info
# Ctrl+Q: Quick documentation

# Navigation:
# Ctrl+N: Tim class/file Python
# Ctrl+B: Go to declaration
# Ctrl+Alt+B: Go to implementation
# Alt+F7: Find usages
`} />

### Debugging va testing

<CodeBlock language="powershell" code={`
# Debugging:
# Click gutter de set breakpoint
# Ctrl+D: Debug
# Alt+F8: Evaluate expression
# F9: Resume program
# Ctrl+F8: Toggle breakpoint

# Testing:
# Ctrl+Shift+F10: Run test tai cursor
# Pytest integration: Phat hien test tu dong
# Coverage: Xem % coverage sau khi chay test
`} />

### Python-specific tools

<CodeBlock language="powershell" code={`
# Virtual env: Tao .venv tu dong
# Package manager: pip, conda, poetry GUI
# Database: SQL editor, schema browser
# Profiler: Phat hien bottleneck
# Refactoring: Rename (Shift+F6), Extract method
# Type hints: Phat hien loi kieu du lieu
`} />
