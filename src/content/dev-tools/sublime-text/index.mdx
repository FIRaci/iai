## Tổng quan

Sublime Text là code editor nhe, nhanh, va tuy bien cao. Noi tieng voi toc do mo file cuc nhanh, multiple selection, Command Palette, va he thong package phong phu qua Package Control.

## Cài đặt

<CodeBlock language="powershell" code={`winget install SublimeHQ.SublimeText.4
# Hoac tai tu https://www.sublimetext.com/download`} />

## Sử dụng

### Multiple Selection

<CodeBlock language="powershell" code={`
# Ctrl+D: Chon tu tiep theo giong tu hien tai
# Alt+F3: Chon TAT CA tu giong nhau
# Ctrl+Shift+L: Chia selection thanh nhieu dong
# Middle-click + drag: Column selection

# Ung dung: Doi ten bien, format hang loat
`} />

### Command Palette

<CodeBlock language="powershell" code={`
# Ctrl+Shift+P: Mo Command Palette
# Goi lenh ma khong can nho phim tat:
# - "ss html" -> Set Syntax: HTML
# - "reindent" -> Reindent file
# - "install" -> Package Control: Install Package
# - "sort lines" -> Sort lines alphabetically
`} />

### GoTo Anything

<CodeBlock language="powershell" code={`
# Ctrl+P: Mo file bang ten
# Ctrl+R: Tim function/class trong file
# Ctrl+G: Den dong so X
# Ctrl+;: Den tu khoa nao do

# @ ky hieu: Tim symbol (VD: @functionName)
# # ky hieu: Tim trong file hien tai
`} />

### Package Control

<CodeBlock language="powershell" code={`# Cai Package Control:
# View > Show Console -> dan code tu https://packagecontrol.io/installation

# Popular packages:
# - A File Icon: File icons
# - BracketHighlighter: Highlight cap ngoac
# - SideBarEnhancements: Right-click context menu
# - Emmet: HTML/CSS snippets
# - SublimeLinter: Code linting`} />

## Luu y

<Note type="tip">
Multiple Selection la feature manh nhat cua Sublime. Luyen tap Ctrl+D de edit nhieu dong cung luc. Cau hinh trong Preferences > Settings (JSON) bang custom key bindings.
</Note>
