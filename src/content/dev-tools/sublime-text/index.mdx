## Tổng quan

<Note type="info">
**Sublime Text** là text editor nhẹ, nhanh, và mạnh mẽ. Nổi tiếng với tốc độ khởi động nhanh, multiple selection, và hệ thống package phong phú. Phù hợp cho quick edits và lightweight development.
</Note>

## Cài đặt trên Windows 11

### Bước 1: Tải Sublime Text

Truy cập [sublimetext.com](https://sublimetext.com) và tải bản Windows.

### Bước 2: Cài đặt

1. Chạy installer
2. Chọn "Add to explorer context menu" (khuyến nghị)
3. Hoàn tất cài đặt

### Bước 3: Cài Package Control

1. Mở Sublime Text
2. `Ctrl+Shift+P` → `Install Package Control`
3. Restart Sublime Text

## Sử dụng cơ bản

### Tính năng chính

| Tính năng | Mô tả |
|-----------|-------|
| Multiple Selection | Chọn và edit nhiều vị trí cùng lúc |
| Command Palette | Truy cập mọi tính năng qua `Ctrl+Shift+P` |
| Package Control | Hệ thống package/plugin phong phú |
| Goto Anything | Tìm file, symbol, line nhanh chóng |
| Distraction Free Mode | Chế độ viết không xao nhãng |

### Keyboard shortcuts

| Phím tắt | Chức năng |
|----------|-----------|
| `Ctrl+P` | Goto file |
| `Ctrl+R` | Goto symbol |
| `Ctrl+G` | Goto line |
| `Ctrl+D` | Select word (nhấn tiếp để select next) |
| `Ctrl+Shift+L` | Split selection into lines |

## Tips sử dụng

<Note type="tip">
**Packages thiết yếu:** Cài `Emmet` (HTML/CSS), `GitGutter` (xem Git changes), `SideBarEnhancements` (cải thiện sidebar).
</Note>

<Note type="tip">
**Settings:** Tạo file `Preferences → Settings` để tùy chỉnh font, theme, và behavior. Dùng JSON format.
</Note>
