## Tổng quan

<Note type="info">
**WebStorm** là IDE chuyên dụng cho JavaScript và web development từ JetBrains. Tích hợp sẵn mọi công cụ cần thiết: linter, debugger, test runner, version control, và database tools. Phù hợp cho developer chuyên nghiệp.
</Note>

## Cài đặt trên Windows 11

### Bước 1: Tải WebStorm

Truy cập [jetbrains.com/webstorm](https://jetbrains.com/webstorm) và tải bản Windows. Có bản miễn phí cho sinh viên và open-source contributors.

### Bước 2: Cài đặt

1. Chạy installer
2. Chọn components (khuyến nghị chọn tất cả)
3. Hoàn tất cài đặt

### Bước 3: Kích hoạt license

- Dùng license JetBrains (có bản free cho sinh viên)
- Hoặc dùng trial 30 ngày

## Sử dụng cơ bản

### Tính năng chính

| Tính năng | Mô tả |
|-----------|-------|
| Smart Completion | Autocomplete thông minh theo ngữ cảnh |
| Refactoring | Refactor code an toàn với preview |
| Debugging | Debug JavaScript/TypeScript tích hợp |
| Testing | Chạy Jest, Mocha, Karma trực tiếp |
| Database Tools | Quản lý database trong IDE |
| Version Control | Git, SVN, Mercurial tích hợp |

### Keyboard shortcuts

| Phím tắt | Chức năng |
|----------|-----------|
| `Ctrl+Shift+A` | Find action |
| `Ctrl+N` | Goto class |
| `Ctrl+Shift+N` | Goto file |
| `Shift+F6` | Rename refactoring |
| `Alt+Enter` | Quick fix / intention |

## Tips sử dụng

<Note type="tip">
**Live Templates:** Dùng `Ctrl+J` để insert code snippets. Tự tạo template cho patterns bạn dùng thường xuyên.
</Note>

<Note type="tip">
**Database Tools:** WebStorm có database tools tích hợp — kết nối PostgreSQL, MongoDB, MySQL trực tiếp trong IDE, không cần tool riêng.
</Note>
