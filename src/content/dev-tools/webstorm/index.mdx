## Tổng quan

WebStorm là IDE chuyên cho JavaScript/TypeScript development của JetBrains. Hỗ trợ React, Angular, Vue, Node.js, và nhiều framework khác. Tích hợp sẵn debugger, testing tools, version control, và database tools.

## Cài đặt

<CodeBlock language="powershell" code={`winget install JetBrains.WebStorm
# Hoac tai tu https://www.jetbrains.com/webstorm/download/`} />

## Sử dụng

### Features chinh

<CodeBlock language="powershell" code={`
# Smart completion: Autocomplete voi context hien tai
# Ctrl+Space: Basic completion
# Ctrl+Shift+Space: Smart completion (goi y kieu du lieu)

# Navigation:
# Ctrl+N: Tim class/file
# Ctrl+Shift+N: Tim file bat ky
# Ctrl+E: Recent files
# Double Shift: Search everywhere
`} />

### Debugging

<CodeBlock language="powershell" code={`
# Click ben trai so dong de set breakpoint
# Ctrl+D: Debug current file/configuration
# F8: Step over
# F7: Step into
# Shift+F8: Step out
# Alt+F9: Run to cursor

# Evaluate expression (Alt+F8): Xem gia tri bien
# Watches: Theo doi gia tri bien lien tuc
`} />

### Tools tich hop

<CodeBlock language="powershell" code={`
# Terminal (Alt+F12): Tich hop terminal
# Git (Alt+Backtick): Commit, push, pull, branch
# npm (Ctrl+Shift+P): Chay npm scripts
# Test: Jest, Mocha, Vitest integration
# Database: Connect va query truc tiep tu IDE
`} />

## Luu y

<Note type="tip">
WebStorm co he thong keyboard shortcuts rat manh. Hoc navigation shortcuts (Ctrl+N, Ctrl+E) de tang toc do code dang ke. Dung Live Templates (Ctrl+J) de tao code snippets nhanh.
</Note>
