## Tổng quan

WSL (Windows Subsystem for Linux) là lớp tương thích của Microsoft cho phép chạy Linux binary executables trực tiếp trên Windows 11 mà không cần virtual machine. WSL 2 sử dụng full Linux kernel, cho hiệu năng gần như native.

## Cài đặt

### Một lệnh duy nhất (WSL 2 mặc định)

<CodeBlock language="powershell" code={`wsl --install`} />

Lệnh này sẽ bật WSL feature, cài Ubuntu (mặc định), và cài WSL 2 kernel.

### Cài đặt distro khác

<CodeBlock language="powershell" code={`wsl --list --online
wsl --install -d Debian
wsl --install -d Fedora
wsl --install -d Ubuntu-24.04`} />

## Sử dụng

### Lệnh cơ bản

<CodeBlock language="powershell" code={`wsl
wsl ls -la
wsl sudo apt update`} />

### Quản lý WSL

<CodeBlock language="powershell" code={`wsl -l -v
wsl --set-default-version 2
wsl --set-default Ubuntu
wsl --terminate Ubuntu
wsl --shutdown`} />

### Cài dev tools trong WSL

<CodeBlock language="bash" code={`sudo apt update && sudo apt upgrade -y
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs
sudo apt install docker.io`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`wsl --version`} />

## Lưu ý

<Note type="tip">
Truy cập Windows files từ WSL qua `/mnt/c/`. Nên để project files trong WSL filesystem (`\\wsl.localhost\Ubuntu\...`) để có performance tốt nhất.
</Note>

<Note type="warning">
WSL 2 nhanh hơn cho Linux operations nhưng chậm hơn khi access Windows files qua `/mnt/c/`. Dùng `wsl -l -v` để kiểm tra version.
</Note>
