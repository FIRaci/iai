## Tổng quan

<Note type="info">
**Android Studio** là IDE chính thức cho Android development, dựa trên IntelliJ IDEA. Cung cấp đầy đủ công cụ để build, test, và debug ứng dụng Android — từ emulator đến layout editor.
</Note>

## Cài đặt trên Windows 11

### Bước 1: Tải Android Studio

Truy cập [developer.android.com/studio](https://developer.android.com/studio) và tải bản Windows.

### Bước 2: Cài đặt

1. Chạy installer
2. Chọn components (Android SDK, Android Virtual Device)
3. Hoàn tất cài đặt

### Bước 3: Cấu hình SDK

1. Mở Android Studio
2. SDK Manager → Download SDK Platform và Build Tools
3. AVD Manager → Tạo Android Virtual Device để test

## Sử dụng cơ bản

### Tính năng chính

| Tính năng | Mô tả |
|-----------|-------|
| Layout Editor | Thiết kế UI Android trực quan |
| Emulator | Chạy và test app trên virtual device |
| Profiler | Phân tích performance (CPU, memory, network) |
| Database Inspector | Xem và edit database realtime |
| Compose Preview | Preview Jetpack Compose UI ngay trong IDE |

### Keyboard shortcuts

| Phím tắt | Chức năng |
|----------|-----------|
| `Ctrl+Shift+A` | Find action |
| `Ctrl+N` | Goto class |
| `Shift+F10` | Run app |
| `Shift+F9` | Debug app |
| `Ctrl+Alt+M` | Extract method |

## Tips sử dụng

<Note type="tip">
**Emulator Performance:** Bật hardware acceleration (Intel HAXM hoặc Windows Hypervisor Platform) để emulator chạy nhanh hơn.
</Note>

<Note type="tip">
**Instant Run / Apply Changes:** Dùng "Apply Changes" để push code changes lên emulator mà không cần rebuild toàn bộ app — tiết kiệm thời gian đáng kể.
</Note>
