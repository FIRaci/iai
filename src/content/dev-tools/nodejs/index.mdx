## Tổng quan

Node.js là runtime JavaScript phía server dựa trên V8 engine, cho phép xây dựng ứng dụng mạng và web với khả năng non-blocking I/O. Được sử dụng rộng rãi trong phát triển backend, CLI tools, và API servers.

## Cài đặt

### Phương pháp 1: Winget (khuyến nghị)

<CodeBlock language="powershell" code={`winget install OpenJS.NodeJS.LTS`} />

### Phương pháp 2: NVM-Windows (quản lý nhiều phiên bản)

<CodeBlock language="powershell" code={`# Cài NVM-Windows trước
winget install CoreyButler.NVMforWindows

# Sau đó cài Node.js qua NVM
nvm install lts
nvm use lts`} />

### Phương pháp 3: Tải trực tiếp từ trang chủ

Tải bản MSI từ [nodejs.org](https://nodejs.org) và chạy installer.

## Sử dụng

### Chạy JavaScript file

<CodeBlock language="powershell" code={`node app.js`} />

### Mở REPL

<CodeBlock language="powershell" code={`node`} />

### Kiểm tra phiên bản

<CodeBlock language="powershell" code={`node --version
npm --version`} />

### Chạy server cơ bản

<CodeBlock language="powershell" code={`node -e "const http = require('http');
http.createServer((req, res) => {
  res.writeHead(200);
  res.end('Hello from Node.js!');
}).listen(3000);"`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`node -v
# Kết quả mẫu: v22.x.x`} />

## Lưu ý

<Note type="tip">
Luôn dùng bản LTS (Long Term Support) cho production để đảm bảo ổn định. Bản hiện tại (Current) phù hợp để thử nghiệm tính năng mới.
</Note>

<Note type="warning">
Nếu gặp lỗi "node is not recognized", hãy thêm Node.js vào PATH hoặc khởi động lại terminal sau khi cài đặt.
</Note>
