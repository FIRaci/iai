## Tổng quan

PowerShell 7 là phiên bản cross-platform, open-source của PowerShell, chạy trên Windows, Linux, và macOS. Sử dụng .NET Core, nhanh hơn Windows PowerShell 5.1, hỗ trợ modules mới và tương thích ngược.

## Cài đặt

### Phương pháp 1: Winget (khuyến nghị)

<CodeBlock language="powershell" code={`winget install Microsoft.PowerShell`} />

### Phương pháp 2: Microsoft Store

Tìm "PowerShell" trong Microsoft Store và cài đặt.

### Phương pháp 3: GitHub Releases

Tải MSI từ [github.com/PowerShell/PowerShell/releases](https://github.com/PowerShell/PowerShell/releases).

## Sử dụng

### Kiểm tra phiên bản

<CodeBlock language="powershell" code={`$PSVersionTable.PSVersion`} />

### Lệnh cơ bản

| Lệnh | Mô tả | Alias |
|------|-------|-------|
| `Get-ChildItem` | Liệt kê files | `ls`, `dir` |
| `Get-Content` | Đọc file | `cat`, `type` |
| `Set-Content` | Ghi file | |
| `Select-String` | Tìm kiếm trong file | `sls` |
| `Where-Object` | Lọc dữ liệu | `where` |

### Pipeline

<CodeBlock language="powershell" code={`Get-Process | Where-Object CPU -gt 100 | Sort-Object CPU -Descending
Get-Service | Export-Csv services.csv -NoTypeInformation`} />

### Modules

<CodeBlock language="powershell" code={`Get-Module -ListAvailable
Install-Module -Name Az -Repository PSGallery`} />

</Note>
