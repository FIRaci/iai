## Tổng quan

<Note type="info">
**Antigravity** là AI-native IDE mới, tích hợp AI sâu vào workflow coding. Thiết kế để giúp developer code nhanh hơn với sự hỗ trợ của AI trong mọi thao tác — từ autocomplete đến refactor.
</Note>

## Cài đặt trên Windows 11

### Bước 1: Tải Antigravity

Truy cập [antigravity.dev](https://antigravity.dev) và tải bản Windows.

### Bước 2: Cài đặt

1. Chạy installer
2. Chọn thư mục cài đặt
3. Hoàn tất cài đặt

### Bước 3: Cấu hình AI

1. Mở Antigravity
2. Vào Settings → AI
3. Chọn model AI (Claude, GPT, hoặc local model)
4. Nhập API key nếu cần

## Sử dụng cơ bản

### Tính năng chính

| Tính năng | Mô tả |
|-----------|-------|
| AI Autocomplete | Dự đoán code thông minh theo ngữ cảnh |
| AI Chat | Chat với AI trong IDE |
| Auto Refactor | AI tự động refactor code |
| Multi-file Edit | Chỉnh sửa nhiều file cùng lúc |
| Terminal tích hợp | Terminal built-in trong IDE |

### Keyboard shortcuts

| Phím tắt | Chức năng |
|----------|-----------|
| `Ctrl+Space` | AI autocomplete |
| `Ctrl+I` | Mở AI chat |
| `Ctrl+Shift+R` | AI refactor |
| `Ctrl+P` | Quick open file |

## Tips sử dụng

<Note type="tip">
**Rules:** Tạo file `.antigravity/rules.md` để định nghĩa coding conventions cho project — AI sẽ tuân theo rules này khi generate code.
</Note>

<Note type="tip">
**Context:** Antigravity tự động đọc toàn bộ codebase để hiểu ngữ cảnh. Đảm bảo `.gitignore` được cấu hình đúng để loại bỏ files không cần thiết.
</Note>
