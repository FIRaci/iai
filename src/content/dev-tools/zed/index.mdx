## Tổng quan

<Note type="info">
**Zed** là code editor hiệu năng cao viết bằng Rust. Tập trung vào tốc độ, collaboration realtime, và trải nghiệm developer mượt mà. Hỗ trợ multiple cursors, inline AI, và team collaboration.
</Note>

## Cài đặt trên Windows 11

### Bước 1: Tải Zed

Truy cập [zed.dev](https://zed.dev) và tải bản Windows.

### Bước 2: Cài đặt

1. Chạy installer
2. Zed tự động cài đặt và thêm vào PATH
3. Mở Zed từ Start Menu

### Bước 3: Cài extensions

Mở Command Palette (`Ctrl+Shift+P`) → `zed: extensions` để cài extensions cho ngôn ngữ bạn dùng.

## Sử dụng cơ bản

### Tính năng chính

| Tính năng | Mô tả |
|-----------|-------|
| Hiệu năng cao | Viết bằng Rust, khởi động trong mili-giây |
| Collaboration | Chia sẻ session, code cùng team realtime |
| Inline AI | AI suggestions ngay trong editor |
| Multiple cursors | Chỉnh sửa nhiều vị trí cùng lúc |
| Terminal tích hợp | Terminal built-in |

### Keyboard shortcuts

| Phím tắt | Chức năng |
|----------|-----------|
| `Ctrl+P` | Quick open file |
| `Ctrl+Shift+P` | Command palette |
| `Alt+Click` | Multiple cursors |
| `Ctrl+D` | Duplicate line |
| `Ctrl+/` | Toggle comment |

## Tips sử dụng

<Note type="tip">
**Remote Development:** Zed hỗ trợ remote development qua SSH. Kết nối đến remote server và code như local.
</Note>

<Note type="tip">
**Performance:** Zed được tối ưu cho performance. Nếu bạn cảm thấy editor khác chậm, hãy thử Zed — đặc biệt với project lớn.
</Note>
