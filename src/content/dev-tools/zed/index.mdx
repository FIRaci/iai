## Tổng quan

Zed là code editor hiệu năng cao viết bằng Rust, được thiết kế cho collaboration realtime và tốc độ tối đa. Hỗ trợ AI inline completion (Claude, GPT-4), multiple cursors, terminal tích hợp, và Vim mode. Đặc biệt phù hợp với Windows developers muốn editor khởi động nhanh và xử lý project lớn mượt mà.

## Cài đặt

Tải installer từ zed.dev.

<CodeBlock language="powershell" code={`
# Visit https://zed.dev/download và tải bản Windows
# Run Zed-Setup-x.x.x.exe

# Hoặc cài qua winget:
winget install Zed.Zed

# Yêu cầu: Windows 11 + 8GB RAM khuyến nghị
`} />

## Sử dụng

Mở Zed và khám phá các tính năng chính.

<CodeBlock language="powershell" code={`
# Mở project từ terminal
zed .   # mở thư mục hiện tại
zed C:\Projects\my-app

# Các phím tắt quan trọng:
# Ctrl+P           - File finder
# Ctrl+Shift+P     - Command palette
# Ctrl+/           - Toggle comment
# Ctrl+D           - Duplicate line
# Alt+Click        - Multiple cursors
# Ctrl+G           - Go to line
# Ctrl+Shift+F     - Search in project

# AI Assistant (cần API key):
# Cấu hình trong Settings → AI:
# provider: anthropic / openai
# model: claude-sonnet-4-20250514 / gpt-4o

# Collaboration:
# Chia sẻ session: Click icon users ở status bar
`} />

## Xác minh cài đặt

## Lưu ý

Zed đang trong giai đoạn early release cho Windows — một số tính năng có thể chưa ổn định. AI Assistant yêu cầu API key riêng (Anthropic/OpenAI). Trên Windows 11, Zed hỗ trợ GPU rendering qua DirectX 12. Nếu gặp lỗi font rendering, cài Cascadia Code hoặc JetBrains Mono.
