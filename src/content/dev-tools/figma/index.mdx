## Tổng quan

Figma là design tool dựa trên web cho UI/UX design, prototyping, và design systems. Hỗ trợ real-time collaboration giống Google Docs, cho phép nhiều người cùng edit một file design. Có sẵn desktop app cho Windows 11.

## Cài đặt

<CodeBlock language="powershell" code={`winget install Figma.Figma
# Hoac tai tu https://www.figma.com/downloads/`} />

## Sử dụng

### Design co ban

<CodeBlock language="powershell" code={`
# Frame (Shift+A): Tao khung cho mobile/desktop
# Shape tools (R): Rectangle, Ellipse, Line
# Text (T): Them text layer
# Pen (P): Ve vector custom shapes
# Boolean groups: Union, Subtract, Intersect

# Components (Alt+2): Tao reusable component
# Auto Layout (Shift+A): Responsive layout
# Constraints: Pin to edges khi resize
`} />

### Prototyping

<CodeBlock language="powershell" code={`
# Chon frame -> tab Prototype
# Ket noi cac frame bang connector
# Them interactions:
# - On Click -> Navigate to (chuyen trang)
# - While Hover -> Open overlay (tooltip)
# - After Delay -> Auto-animate

# Present (Play button): Xem thu prototype
`} />

### Design System

<CodeBlock language="powershell" code={`
# Styles: Color, Text, Effect, Grid
# Components: Buttons, Inputs, Cards
# Variants: Chuyen doi trang thai (hover, active)
# Team Library: Chia se design system qua team

# Export assets:
# - Ctrl+Shift+E: Export selected
# - Dinh dang: PNG, SVG, JPG, PDF
`} />

## Luu y

<Note type="tip">
Dung Auto Layout cho moi component de dam bao responsive. Su dung Variants de quan ly trang thai button, input.
</Note>

<Note type="info">
Figma co che do offline cho desktop app. File duoc sync khi co internet tro lai. Plugin ecosystem rat phong phu cho icon, color palette, accessibility check.
</Note>
