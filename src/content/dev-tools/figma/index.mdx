## Tổng quan

<Note type="info">
**Figma** là công cụ thiết kế UI/UX dựa trên cloud, hỗ trợ collaboration realtime. Miễn phí cho cá nhân, với đầy đủ công cụ từ wireframe đến high-fidelity prototype.
</Note>

## Cài đặt trên Windows 11

### Phương pháp 1: Web App (khuyến nghị)

Truy cập [figma.com](https://figma.com) — không cần cài đặt, chạy trên trình duyệt.

### Phương pháp 2: Desktop App

1. Tải từ [figma.com/downloads](https://figma.com/downloads)
2. Cài đặt và đăng nhập

### Phương pháp 3: Figma Font Installer

Cài [Figma Font Installer](https://figma.com/downloads) để dùng local fonts trong thiết kế.

## Sử dụng cơ bản

### Thiết kế cơ bản — Tạo button component

1. **Tạo Frame:** Nhấn `F` → kéo khung 400x300px
2. **Vẽ Rectangle:** Nhấn `R` → vẽ hình chữ nhật
3. **Bo góc:** Properties panel → Corner radius: 8px
4. **Thêm text:** Nhấn `T` → click vào rectangle → gõ "Click me"
5. **Căn giữa:** Select cả rectangle và text → Align horizontal + vertical centers
6. **Auto Layout:** Select rectangle → Shift+A → điều chỉnh padding

## Tips sử dụng

<Note type="tip">
**Auto Layout:** Dùng Auto Layout để design responsive. Giống flexbox trong CSS — elements tự động adjust khi content thay đổi.
</Note>

<Note type="tip">
**Components:** Tạo reusable components cho buttons, cards, inputs — tiết kiệm thời gian. Update master component → tất cả instances tự động update.
</Note>

<Note type="tip">
**Plugins thiết yếu:**
- **Unsplash:** Insert ảnh miễn phí
- **Iconify:** 100,000+ icons
- **Content Reel:** Dummy data
- **Stark:** Accessibility checker
- **Auto Layout:** Tự động sắp xếp elements
</Note>

## Khắc phục sự cố

### Font missing errors

- Cài [Figma Font Installer](https://figma.com/downloads)
- Hoặc thay thế font trong Properties panel
- Dùng Google Fonts (có sẵn trong Figma)

### Plugin crashes

- Disable và enable lại plugin
- Kiểm tra plugin đã update chưa
- Thử plugin alternative từ Community

### Offline mode limitations

- Figma cần internet để sync
- Offline mode chỉ xem được files đã mở trước đó
- Không thể tạo file mới khi offline

### Performance với large files

- Tách file lớn thành nhiều pages
- Detach unused components
- Dùng **File → Save local copy** để backup
- Reduce image resolutions trong design

### Dev Mode không hoạt động

- Dev Mode yêu cầu paid plan (Professional trở lên)
- Free plan chỉ xem được inspect panel cơ bản
- Upgrade hoặc dùng plugin "Figma to Code" alternative
