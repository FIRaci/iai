## Tổng quan

Windows Terminal là ứng dụng terminal hiện đại của Microsoft, hỗ trợ nhiều tabs, GPU-accelerated rendering, và tùy biến giao diện. Tích hợp sẵn PowerShell, CMD, WSL, và có thể thêm các shell khác như Git Bash, PowerShell 7.

## Cài đặt

### Phương pháp 1: Winget (khuyến nghị)

<CodeBlock language="powershell" code={`winget install Microsoft.WindowsTerminal`} />

### Phương pháp 2: Microsoft Store

Tìm "Windows Terminal" trong Microsoft Store và cài đặt.

## Sử dụng

### Phím tắt cơ bản

| Phím tắt | Chức năng |
|----------|-----------|
| `Ctrl+Shift+1/2/3` | Mở tab theo profile |
| `Ctrl+Shift+D` | Mở tab mới |
| `Ctrl+Shift+W` | Đóng tab |
| `Ctrl+Shift+N` | Mở cửa sổ mới |
| `Ctrl+,` | Mở Settings |
| `Ctrl+Shift+F` | Tìm kiếm |
| `Ctrl+Shift+P` | Command palette |

### Cấu hình mẫu

<CodeBlock language="json" code={`{
  "profiles": {
    "defaults": {
      "fontFace": "CascadiaCode NF",
      "fontSize": 14,
      "opacity": 90,
      "useAcrylic": true
    }
  }
}`} />

## Xác minh cài đặt

Mở Windows Terminal từ Start Menu hoặc chạy `wt` từ PowerShell/CMD.
