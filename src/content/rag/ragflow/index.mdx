## Tổng quan

<Note type="info">
**RAGFlow** là RAG engine với document parsing sâu, hỗ trợ nhiều document formats và template-based retrieval.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`git clone https://github.com/infiniflow/ragflow
cd ragflow
docker compose up -d`} />

## Sử dụng

1. Mở `http://localhost`
2. Upload documents
3. Tạo knowledge base
4. Query qua API hoặc UI
