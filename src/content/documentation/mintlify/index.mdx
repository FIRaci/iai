## Tổng quan

<Note type="info">
**Mintlify** là modern documentation platform, tạo docs sites đẹp và dễ maintain từ Markdown/MDX files.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm i -g mintlify`} />

## Sử dụng

<CodeBlock language="powershell" code={`mintlify init
mintlify dev`} />

Mở `http://localhost:3000` để xem docs locally.
