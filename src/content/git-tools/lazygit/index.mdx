## Tổng quan

<Note type="info">
**Lazygit** là simple terminal UI cho git commands. Dễ dùng và nhanh hơn CLI git truyền thống.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install JesseDuffield.lazygit`} />

## Sử dụng

<CodeBlock language="powershell" code={`cd your-repo
lazygit`} />

Phím tắt phổ biến:
- `Space` để stage file
- `c` để commit
- `p` để push
- `P` để pull
