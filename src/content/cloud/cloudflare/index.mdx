## Tổng quan

<Note type="info">
**Cloudflare** là cloud platform cung cấp CDN, DDoS protection, DNS, và serverless compute (Workers). Hỗ trợ Workers AI, D1 database, R2 object storage, và KV storage cho full-stack serverless apps.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install -g wrangler
# Login
wrangler login`} />

## Sử dụng

<CodeBlock language="javascript" code={`// Cloudflare Worker
export default {
  async fetch(request, env) {
    return new Response("Hello from Cloudflare Workers!")
  }
}

// Deploy
// wrangler deploy`} />
