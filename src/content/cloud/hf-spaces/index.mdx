## Tổng quan

<Note type="info">
**Hugging Face Spaces** là platform cho hosting ML demos. Deploy Gradio, Streamlit, hoặc static apps dễ dàng.
</Note>

## Sử dụng

1. Tạo Space trên [huggingface.co/spaces](https://huggingface.co/spaces)
2. Chọn SDK (Gradio, Streamlit, Docker)
3. Push code hoặc upload files
4. App tự động deploy
