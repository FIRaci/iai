## Tổng quan

<Note type="info">
**Google Colab** là môi trường Jupyter Notebook chạy trên cloud, cung cấp GPU miễn phí (NVIDIA T4, V100) và TPU. Không cần cấu hình, mở browser là code được ngay. Phù hợp cho ML experimentation, data analysis, và education.
</Note>

## Tính năng chính

- **GPU/TPU miễn phí:** Truy cập NVIDIA T4, V100, A100 và TPU v2/v3 không mất phí
- **Tích hợp Drive:** Mount Google Drive để lưu dữ liệu và checkpoint
- **Form sharing:** Chia sẻ notebook dưới dạng .ipynb hoặc link GitHub
- **Pre-installed libs:** TensorFlow, PyTorch, JAX, Transformers có sẵn
- **Magic commands:** `%run`, `%load`, `%store` giúp quản lý code linh hoạt
