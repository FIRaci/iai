## Tổng quan

<Note type="info">
**Google Colab** là cloud-based Jupyter notebook environment với free GPU/TPU access. Perfect cho ML experiments.
</Note>

## Sử dụng

1. Truy cập [colab.research.google.com](https://colab.research.google.com)
2. Tạo notebook mới
3. Chọn runtime type: GPU/TPU
4. Bắt đầu code

## Tips

<Note type="tip">
**Free GPU:** Colab cung cấp free GPU (T4, P100) cho notebooks.
</Note>
