## Tổng quan

<Note type="info">
**Beautiful Soup** là thư viện Python cho web scraping, cung cấp API đơn giản để parse HTML/XML documents. Tích hợp với requests và lxml.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install beautiful-soup`} />
