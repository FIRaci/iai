## Tổng quan

<Note type="info">
**Selenium** là công cụ browser automation đa nền tảng, hỗ trợ nhiều ngôn ngữ lập trình. Dùng cho web scraping, automated testing và RPA.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install selenium`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
