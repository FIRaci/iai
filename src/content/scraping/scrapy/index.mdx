## Tổng quan

<Note type="info">
**Scrapy** là web scraping framework Python mạnh mẽ và nhanh chóng. Cung cấp selectors, middlewares, pipelines và built-in support cho xử lý requests và data export.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install scrapy`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
