## Tổng quan

<Note type="info">
**Crawlee** là web scraping và browser automation library cho Node.js, hỗ trợ cả HTTP requests và headless browser. Cung cấp built-in proxy management.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install crawlee`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
