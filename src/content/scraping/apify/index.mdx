## Tổng quan

<Note type="info">
**Apify** là nền tảng web scraping và automation cho developers. Cung cấp Apify SDK, Crawlee framework và cloud platform với proxy rotation.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install apify`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
