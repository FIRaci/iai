## Tổng quan

<Note type="info">
**Colly** là web scraping framework cho Go, nổi tiếng với hiệu suất cao và API đơn giản. Hỗ trợ parallel crawling và caching.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install colly`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
