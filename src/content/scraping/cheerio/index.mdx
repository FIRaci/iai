## Tổng quan

<Note type="info">
**Cheerio** là thư viện Node.js nhanh và linh hoạt cho HTML parsing, sử dụng cú pháp jQuery quen thuộc. Lý tưởng cho web scraping phía server.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install cheerio`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
