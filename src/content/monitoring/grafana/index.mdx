## Tổng quan

Grafana là công cụ phân tích và trực quan hóa dữ liệu metrics, logs, traces. Hỗ trợ hàng chục data sources khác nhau: Prometheus, Elasticsearch, InfluxDB, Loki, Graphite, PostgreSQL. Có thể tự host hoặc dùng Grafana Cloud với mobile app hỗ trợ xem dashboard trên điện thoại.

## Cài đặt

<CodeBlock language="powershell" code={`
# Docker (nhanh nhất)
docker run -d -p 3000:3000 --name grafana grafana/grafana

# Windows với winget
winget install Grafana.Grafana

# Cấu hình persistence
docker run -d -p 3000:3000 -v grafana-storage:/var/lib/grafana -e GF_SECURITY_ADMIN_PASSWORD=StrongPass123 grafana/grafana
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
# Mở trình duyệt
start http://localhost:3000
# Đăng nhập: admin / admin
# Đổi password ngay lần đầu
`} />

## Sử dụng

1. **Add data source:** Configuration -> Data Sources -> Add. Chọn Prometheus, nhập URL `http://localhost:9090`, click Save & Test.
2. **Tạo dashboard:** Dashboards -> New Dashboard. Thêm panel với queries từ data source. Dùng visualization types: graph, bar chart, heatmap, stat, table.
3. **Import dashboard:** Dashboards -> Import. Nhập ID từ grafana.com/dashboards (ví dụ: 1860 cho Node Exporter).
4. **Alert rules:** Alerting -> Alert rules. Tạo rule: `avg by(instance) (rate(node_cpu_seconds_total[5m])) > 0.8`. Nhận notification qua email, Slack, Telegram.
5. **API provisioning:** Tự động hóa cấu hình bằng API key hoặc service account.

<CodeBlock language="powershell" code={`
# Provision data source qua API
curl -X POST http://admin:admin@localhost:3000/api/datasources -H 'Content-Type: application/json' -d '{"name":"Prometheus","type":"prometheus","url":"http://localhost:9090","access":"proxy"}'
`} />

## Lưu ý

Grafana Cloud có free tier với 10k metrics. Dùng service account thay vì admin API key cho production. Có thể dùng Grafana Loki cho log aggregation thay vì Elasticsearch để tiết kiệm chi phí. Mobile app Grafana cho phép xem dashboard và nhận alert trên điện thoại.
