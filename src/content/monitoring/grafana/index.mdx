## Tổng quan

<Note type="info">
**Grafana** là visualization platform cho metrics và logs. Tạo dashboards từ Prometheus, Elasticsearch, và nhiều data sources.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -p 3000:3000 grafana/grafana`} />

## Sử dụng

1. Mở `http://localhost:3000` (admin/admin)
2. Add data source (Prometheus, Elasticsearch, v.v.)
3. Tạo dashboard với panels
