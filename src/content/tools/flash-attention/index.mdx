## Tổng quan

<Note type="info">
**Flash Attention** là fast và memory-efficient exact attention algorithm cho GPUs. Tăng tốc training và inference.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install flash-attn`} />

## Sử dụng

<CodeBlock language="python" code={`from flash_attn import flash_attn_func
q = torch.randn(2, 1024, 16, 64, device="cuda")
k = torch.randn(2, 1024, 16, 64, device="cuda")
v = torch.randn(2, 1024, 16, 64, device="cuda")
out = flash_attn_func(q, k, v)`} />
