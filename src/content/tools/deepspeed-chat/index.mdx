## Tổng quan

<Note type="info">
**DeepSpeed Chat** là framework cho RLHF training của LLMs. Implement InstructGPT-style training với DeepSpeed.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install deepspeed`} />

## Sử dụng

<CodeBlock language="powershell" code={`deepspeed --num_gpus=8 main.py --actor_model mistralai/Mistral-7B-v0.1`} />
