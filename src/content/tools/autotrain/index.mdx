## Tổng quan

<Note type="info">
**AutoTrain** từ Hugging Face là auto-ML tool cho fine-tuning models mà không cần code. GUI và CLI đều available.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install autotrain-advanced`} />

## Sử dụng

<CodeBlock language="powershell" code={`autotrain llm --model mistralai/Mistral-7B-v0.1 --data-path data/`} />
