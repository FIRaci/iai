## Tổng quan

Winget (Windows Package Manager CLI) là package manager chính thức của Microsoft, được tích hợp sẵn trên Windows 11. Cho phép cài đặt, cập nhật và gỡ bỏ ứng dụng từ command line, hỗ trợ hàng nghìn ứng dụng từ Microsoft Store và GitHub releases.

## Cài đặt

Winget có sẵn trên Windows 11 — không cần cài thêm. Bạn có thể kiểm tra bằng lệnh sau:

<CodeBlock language="powershell" code={`winget --version`} />

Nếu chưa có (Windows 10), cài qua App Installer từ Microsoft Store.

## Sử dụng

### Tìm kiếm và cài đặt

<CodeBlock language="powershell" code={`# Tìm kiếm package
winget search nodejs
winget search "visual studio code"

# Cài đặt package
winget install Microsoft.VisualStudioCode
winget install Git.Git
winget install OpenJS.NodeJS.LTS

# Cài với ID chính xác
winget install --id Google.Chrome -e`} />

### Quản lý packages

<CodeBlock language="powershell" code={`# Cập nhật package
winget upgrade Microsoft.VisualStudioCode

# Cập nhật tất cả
winget upgrade --all

# Gỡ package
winget uninstall Google.Chrome

# Liệt kê packages đã cài
winget list

# Xem thông tin package
winget show Microsoft.PowerShell`} />

### Export/Import cấu hình

<CodeBlock language="powershell" code={`# Export danh sách packages
winget export -o packages.json

# Import và cài tất cả
winget import -i packages.json`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`winget --version
# Kết quả mẫu: v1.10.x`} />

## Lưu ý

<Note type="tip">
Dùng `winget export` và `winget import` để đồng bộ cấu hình máy mới nhanh chóng. Lưu file `packages.json` vào git repository.
</Note>

<Note type="warning">
Một số ứng dụng cần quyền Administrator để cài qua winget. Winget sẽ tự động yêu cầu UAC nếu cần.
</Note>
