## Tổng quan

Postman là API platform cho phép thiết kế, kiểm thử, và document API. Hỗ trợ REST, GraphQL, gRPC, và WebSocket với giao diện đồ hoạ trực quan. Tích hợp collection runner, environment variables, và mock servers.

## Cài đặt

<CodeBlock language="powershell" code={`winget install Postman.Postman
# Hoac tai tu https://www.postman.com/downloads/`} />

## Sử dụng

### Gửi request co ban

<CodeBlock language="powershell" code={`
# Mo Postman va tao request moi
# Method: GET, POST, PUT, DELETE, PATCH
# URL: https://api.example.com/users

# Dinh nghia headers, params, body
# Body co the la JSON, XML, form-data, binary

# Sau do click Send de xem response
`} />

### Environment variables

<CodeBlock language="powershell" code={`
# Tao environment (VD: dev, staging, prod)
# Bien: base_url, api_key, token

# Su dung trong request: {{base_url}}/users
# Postman tu dong thay the bang gia tri

# Pre-request Script: tu dong lay token truoc khi chay
pm.sendRequest("https://auth.example.com/login", ...)
`} />

### Collection Runner

<CodeBlock language="powershell" code={`
# Chay toan bo collection tu CLI
newman run Collection.json -e Dev.postman_environment.json

# Xuat report HTML
newman run Collection.json -r htmlextra

# Chay voi data file
newman run Collection.json -d data.csv`} />

## Xác minh cài đặt

Dung environment variables de quan ly API keys va URLs giua cac moi truong. Export collection va environment de chia se voi team qua git.
</Note>

Khong commit real API keys vao collection. Dung Postman Vault hoac environment variables voi gia tri mac dinh la placeholder.
</Note>
