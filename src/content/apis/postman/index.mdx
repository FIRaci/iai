## Tổng quan

<Note type="info">
**Postman** là API platform cho building, testing, và documenting APIs. GUI dễ dùng và collaboration features.
</Note>

## Cài đặt

Tải từ [postman.com](https://www.postman.com/downloads)

## Sử dụng

1. Mở Postman
2. Tạo request mới
3. Nhập URL và method
4. Send request và xem response
