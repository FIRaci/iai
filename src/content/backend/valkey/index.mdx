## Tổng quan

<Note type="info">
**Valkey** là in-memory database mã nguồn mở, tương thích với Redis protocol. Valkey ra đời như fork của Redis sau khi Redis chuyển sang license không còn mã nguồn mở. Valkey hỗ trợ đầy đủ các tính năng: caching, pub/sub, streams, sorted sets, và Lua scripting.
</Note>

## Cài đặt

Chạy Valkey với Docker:

<CodeBlock language="powershell" code={`docker run -d --name valkey -p 6379:6379 valkey/valkey:8`} />

## Sử dụng với Node.js

<CodeBlock language="powershell" code={`npm install ioredis`} />

<CodeBlock language="javascript" code={`import Redis from "ioredis";
const client = new Redis("redis://localhost:6379");
await client.set("user:1", "hello", "EX", 3600);
const val = await client.get("user:1");
console.log(val); // hello`} />

## Xác minh

<CodeBlock language="powershell" code={`docker exec valkey valkey-cli PING`} />

## Lưu ý

<Note type="tip">
Valkey tương thích hoàn toàn với Redis clients (ioredis, redis-py, redis-cli), bạn chỉ cần thay đổi connection string. Valkey 8 có cải thiện hiệu suất đáng kể so với Redis 7.
</Note>
