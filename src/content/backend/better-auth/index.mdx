## Tổng quan

<Note type="info">
**Better Auth** là thư viện authentication hiện đại cho TypeScript, được thiết kế để dễ dàng mở rộng và tích hợp. Hỗ trợ email/password, OAuth, 2FA, Magic Links, WebAuthn, và session management tích hợp sẵn.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install better-auth`} />

## Sử dụng cơ bản

Khởi tạo Better Auth server với các providers:

<CodeBlock language="javascript" code={`
import { betterAuth } from 'better-auth';
import { emailPassword } from 'better-auth/plugins';

export const auth = betterAuth({
  plugins: [emailPassword()],
  social: {
    github: {
      clientId: process.env.GITHUB_CLIENT_ID,
      clientSecret: process.env.GITHUB_CLIENT_SECRET,
    },
  },
});
`} />

Sử dụng trong API route:

<CodeBlock language="javascript" code={`
import { auth } from '@/lib/auth';

export async function POST(req: Request) {
  const { email, password } = await req.json();
  const { user, session } = await auth.api.signIn({
    body: { email, password },
  });
  return Response.json(user);
}
`} />

## Xác minh

<CodeBlock language="powershell" code={`curl -X POST http://localhost:3000/api/auth/sign-in -H "Content-Type: application/json" -d '{"email":"test@test.com","password":"123456"}'`} />
