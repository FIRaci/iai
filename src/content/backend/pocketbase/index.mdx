## Tổng quan

<Note type="info">
**PocketBase** là backend all-in-one mã nguồn mở, sử dụng SQLite làm database và tích hợp sẵn admin UI. PocketBase cung cấp authentication (email, OAuth2), file storage, realtime subscriptions, và REST API tự động từ collections. Tất cả gói gọn trong một binary duy nhất.
</Note>

## Cài đặt

Tải binary từ GitHub releases:

<CodeBlock language="powershell" code={`curl -L https://github.com/pocketbase/pocketbase/releases/latest/download/pocketbase_0.26.1_windows_amd64.zip -o pocketbase.zip
Expand-Archive pocketbase.zip -DestinationPath ./pocketbase
.\pocketbase\pocketbase.exe serve`} />

## Tạo Collection và API

```bash
# API collections tự động sinh
GET /api/collections/posts/records
POST /api/collections/posts/records

# Authenticate
POST /api/collections/users/auth-with-password
```

## Xác minh

<CodeBlock language="powershell" code={`curl http://127.0.0.1:8090/api/health`} />

## Lưu ý

<Note type="tip">
PocketBase rất phù hợp cho MVP và side projects. Admin UI tại `/_/` cho phép quản lý collections, users, và files trực quan mà không cần code.
</Note>
