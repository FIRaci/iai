## Tổng quan

<Note type="info">
**MikroORM** là TypeScript ORM hỗ trợ SQL (PostgreSQL, MySQL, SQLite) và MongoDB. MikroORM sử dụng Data Mapper pattern với Identity Map cho hiệu suất cao và hỗ trợ migrations, embedded entities, và transactions.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install @mikro-orm/core @mikro-orm/postgresql`} />

## Sử dụng cơ bản

Định nghĩa entity:

<CodeBlock language="javascript" code={`import { Entity, Property, PrimaryKey } from "@mikro-orm/core";

@Entity()
export class User {
  @PrimaryKey()
  id!: number;

  @Property()
  name!: string;

  @Property()
  email!: string;
}`} />

CRUD operations:

<CodeBlock language="javascript" code={`const user = em.create(User, { name: "Alice", email: "alice@test.com" });
await em.flush();
const found = await em.findOne(User, { email: "alice@test.com" });
found.name = "Bob";
await em.flush();
await em.removeAndFlush(found);`} />

## Xác minh

<CodeBlock language="powershell" code={`npx mikro-orm schema:create -r`} />

## Lưu ý

<Note type="tip">
Sử dụng `em.fork()` để tạo EntityManager mới trong xử lý concurrent. Identity Map của MikroORM giúp tránh duplicate entities trong cùng một context.
</Note>
