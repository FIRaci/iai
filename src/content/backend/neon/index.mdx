## Tổng quan

<Note type="info">
**Neon** là serverless PostgreSQL platform với kiến trúc compute-storage separation. Neon cung cấp database branching (giống Git cho database), auto-scaling xuống 0 khi không dùng, và cold start nhanh. Neon tương thích hoàn toàn với PostgreSQL ecosystem.
</Note>

## Kết nối

Lấy connection string từ Neon Console và kết nối:

<CodeBlock language="powershell" code={`psql "postgresql://user:pass@ep-example-123.us-east-2.aws.neon.tech/neondb"`} />

Kết nối từ Node.js:

<CodeBlock language="javascript" code={`import { Pool } from "@neondatabase/serverless";
const pool = new Pool({ connectionString: process.env.DATABASE_URL });
const { rows } = await pool.query("SELECT NOW()");
console.log(rows[0].now);`} />

## Database Branching

Tạo branch từ Neon Console hoặc CLI:

<CodeBlock language="powershell" code={`neon branch create --name feature-branch`} />

## Xác minh

<CodeBlock language="powershell" code={`psql $DATABASE_URL -c "SELECT version();"`} />

</Note>
