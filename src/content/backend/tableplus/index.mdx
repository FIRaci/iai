## Tổng quan

<Note type="info">
**TablePlus** là công cụ GUI quản trị database hiện đại, hỗ trợ nhiều loại database: PostgreSQL, MySQL, Redis, SQLite, MongoDB, SQL Server, và hơn 10 loại khác. TablePlus có giao diện native, nhanh, hỗ trợ multiple tabs, code auto-complete, và native SSH tunneling.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install TablePlus`} />

Hoặc tải từ website: https://tableplus.com/download

## Kết nối đến database

1. Mở TablePlus, click Create New Connection
2. Chọn loại database (PostgreSQL, MySQL, v.v.)
3. Nhập thông tin kết nối: Host, Port, User, Password, Database
4. Click Test để kiểm tra, sau đó Save

## Phím tắt hữu ích

<CodeBlock language="powershell" code={`Ctrl+N  - Tab mới
Cmd+R   - Run query
Cmd+W   - Close tab
Cmd+T   - New query tab`} />

## Xác minh

<CodeBlock language="powershell" code={`tableplus --version`} />

## Lưu ý

<Note type="tip">
TablePlus hỗ trợ native SSH tunneling, không cần mở port database ra internet. Dùng Filter Bar để tìm kiếm dữ liệu nhanh trong bảng.
</Note>
