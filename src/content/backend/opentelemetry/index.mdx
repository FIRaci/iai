## Tổng quan

<Note type="info">
**OpenTelemetry** (OTel) là observability framework mã nguồn mở, cung cấp bộ công cụ thu thập traces, metrics, và logs từ ứng dụng. OpenTelemetry là CNCF incubating project, hỗ trợ đa ngôn ngữ và tích hợp với nhiều backends như Jaeger, Zipkin, Prometheus, và Grafana.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install @opentelemetry/api @opentelemetry/sdk-node @opentelemetry/instrumentation-http @opentelemetry/instrumentation-express`} />

## Sử dụng cơ bản

<CodeBlock language="javascript" code={`import { NodeSDK } from "@opentelemetry/sdk-node";
import { ConsoleSpanExporter } from "@opentelemetry/sdk-trace-node";
import { getNodeAutoInstrumentations } from "@opentelemetry/auto-instrumentations-node";

const sdk = new NodeSDK({
  traceExporter: new ConsoleSpanExporter(),
  instrumentations: [getNodeAutoInstrumentations()],
});

sdk.start();`} />

Tạo custom span:

<CodeBlock language="javascript" code={`import { trace } from "@opentelemetry/api";
const tracer = trace.getTracer("my-service");
const span = tracer.startSpan("process-order");
span.setAttribute("order.id", "123");
span.end();`} />

## Xác minh

## Lưu ý

<Note type="tip">
Bắt đầu với auto-instrumentation trước, sau đó thêm custom spans cho business logic quan trọng. Export traces với sampling rate phù hợp để kiểm soát chi phí.
</Note>
