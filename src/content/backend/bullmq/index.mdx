## Tổng quan

<Note type="info">
**BullMQ** là thư viện message queue mạnh mẽ cho Node.js, sử dụng Redis làm backend. Cho phép quản lý hàng đợi công việc, lên lịch tác vụ, xử lý jobs với độ ưu tiên, retry, rate limiting và real-time events.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install bullmq ioredis`} />

## Sử dụng cơ bản

Định nghĩa queue và worker:

<CodeBlock language="javascript" code={`
import { Queue, Worker } from 'bullmq';

const queue = new Queue('email-queue', {
  connection: { host: 'localhost', port: 6379 },
});

await queue.add('send-welcome', {
  email: 'user@example.com',
  template: 'welcome',
});

const worker = new Worker('email-queue', async (job) => {
  console.log('Processing job', job.id, job.data);
  await sendEmail(job.data);
}, {
  connection: { host: 'localhost', port: 6379 },
});
`} />

## Xác minh

<CodeBlock language="powershell" code={`node worker.js`} />
