## Tổng quan

<Note type="info">
**Fastify** là web framework siêu nhanh cho Node.js, tập trung vào hiệu suất và trải nghiệm developer. Với kiến trúc plugin mạnh mẽ, schema-based serialization, và logging tích hợp, Fastify là lựa chọn lý tưởng cho các ứng dụng cần tốc độ cao.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install fastify`} />

## Sử dụng cơ bản

Tạo server Fastify với route:

<CodeBlock language="javascript" code={`
import Fastify from 'fastify';
const fastify = Fastify({ logger: true });

fastify.get('/', async (request, reply) => {
  return { hello: 'world' };
});

fastify.post('/users', {
  schema: {
    body: {
      type: 'object',
      properties: {
        name: { type: 'string' },
        email: { type: 'string' },
      },
      required: ['name', 'email'],
    },
  },
}, async (request, reply) => {
  const { name, email } = request.body;
  reply.status(201).send({ id: 1, name, email });
});

const start = async () => {
  await fastify.listen({ port: 3000 });
};
start();
`} />

## Xác minh

<CodeBlock language="powershell" code={`curl http://localhost:3000`} />
