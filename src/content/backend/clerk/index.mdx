## Tổng quan

<Note type="info">
**Clerk** là nền tảng quản lý authentication và user hoàn chỉnh, cung cấp các component UI sẵn sàng như sign-in, sign-up, user profile và organization management. Tích hợp dễ dàng với Next.js, React và nhiều framework khác.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install @clerk/nextjs`} />

## Sử dụng cơ bản

Bọc ứng dụng với ClerkProvider:

<CodeBlock language="javascript" code={`
import { ClerkProvider, SignInButton, SignedIn, SignedOut } from '@clerk/nextjs';

export default function RootLayout({ children }) {
  return (
    <ClerkProvider>
      <SignedOut>
        <SignInButton />
      </SignedOut>
      <SignedIn>
        {children}
      </SignedIn>
    </ClerkProvider>
  );
}
`} />

Sử dụng middleware để bảo vệ routes:

<CodeBlock language="javascript" code={`
import { authMiddleware } from '@clerk/nextjs';

export default authMiddleware({
  publicRoutes: ['/', '/about'],
});

export const config = {
  matcher: ['/((?!.*\\..*|_next).*)'],
};
`} />

## Xác minh

<CodeBlock language="powershell" code={`curl http://localhost:3000/api/auth/status`} />
