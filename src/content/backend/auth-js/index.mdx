## Tổng quan

<Note type="info">
**Auth.js** (trước đây là NextAuth.js) là thư viện authentication mã nguồn mở cho Next.js và các framework JavaScript khác. Hỗ trợ đa dạng providers như Google, GitHub, Facebook, email magic links, và credentials-based auth.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install next-auth`} />

## Sử dụng cơ bản

Thêm API route cho authentication trong Next.js App Router:

<CodeBlock language="javascript" code={`
import NextAuth from 'next-auth';
import GitHub from 'next-auth/providers/github';

export const { handlers, auth, signIn, signOut } = NextAuth({
  providers: [GitHub],
});
`} />

Sử dụng trong layout:

<CodeBlock language="javascript" code={`
import { auth } from '@/auth';

export default async function Page() {
  const session = await auth();
  return <p>Welcome {session?.user?.name}</p>;
}
`} />

## Xác minh

<CodeBlock language="powershell" code={`curl http://localhost:3000/api/auth/session`} />
