## Tổng quan

<Note type="info">
**Bruno** là API client mã nguồn mở, hoạt động hoàn toàn offline và lưu trữ collection dưới dạng file văn bản thuần. Đây là giải pháp thay thế mã nguồn mở cho Postman, không yêu cầu đăng nhập tài khoản và tôn trọng quyền riêng tư dữ liệu của bạn.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install Bruno`} />

## Sử dụng cơ bản

Tạo request collection mới:

<CodeBlock language="text" code={`
bruno/
  collections/
    my-api/
      request-get-users.bru
      request-create-user.bru
  bruno.json
`} />

Ví dụ file request `.bru`:

<CodeBlock language="text" code={`
meta {
  name: Get Users
  method: GET
  url: https://api.example.com/users
}

headers {
  Content-Type: application/json
  Authorization: Bearer {{token}}
}
`} />

## Xác minh

<CodeBlock language="powershell" code={`bruno --version`} />
