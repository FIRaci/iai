## Tổng quan

<Note type="info">
**Hono** là web framework cực nhẹ và nhanh cho JavaScript/TypeScript, tương thích với nhiều runtime như Cloudflare Workers, Deno, Bun, và Node.js. Với kích thước chỉ ~17KB, Hono lý tưởng cho các ứng dụng chạy ở edge và serverless.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm create hono@latest my-app`} />

## Sử dụng cơ bản

Tạo ứng dụng Hono cơ bản:

<CodeBlock language="javascript" code={`
import { Hono } from 'hono';
import { cors } from 'hono/cors';
import { logger } from 'hono/logger';

const app = new Hono();

app.use('*', cors());
app.use('*', logger());

app.get('/', (c) => c.json({ message: 'Hello Hono!' }));

app.post('/users', async (c) => {
  const body = await c.req.json();
  return c.json({ id: 1, ...body }, 201);
});

export default app;
`} />

Chạy trên Deno:

<CodeBlock language="powershell" code={`deno run -A src/index.ts`} />

## Xác minh

</Note>
