## Tổng quan

<Note type="info">
**RabbitMQ** là message broker phổ biến nhất, hỗ trợ nhiều giao thức messaging (AMQP, MQTT, STOMP). RabbitMQ dùng exchanges và queues để routing messages, với các patterns: direct, topic, fanout, và headers. RabbitMQ phù hợp cho task queues, event-driven architecture, và microservices communication.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install RabbitMQ`} />

Hoặc chạy với Docker:

<CodeBlock language="powershell" code={`docker run -d --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:4-management`} />

## Sử dụng với Node.js

<CodeBlock language="javascript" code={`import amqp from "amqplib";
const conn = await amqp.connect("amqp://localhost");
const ch = await conn.createChannel();
const q = await ch.assertQueue("tasks");

// Consumer
ch.consume(q.queue, (msg) => {
  console.log(msg.content.toString());
  ch.ack(msg);
});

// Publisher
ch.sendToQueue(q.queue, Buffer.from("Hello"));`} />

## Lưu ý

<Note type="tip">
Dùng direct exchange cho task routing, fanout exchange cho broadcast, topic exchange cho pattern-based routing. Nhớ `ch.ack(msg)` để tránh message không được xác nhận.
</Note>
