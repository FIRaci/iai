## Tổng quan

Supabase là open-source Backend-as-a-Service (BaaS) thay thế Firebase với PostgreSQL, authentication, real-time subscriptions, storage, và Edge Functions. Cung cấp REST API và GraphQL tự động từ schema database.

## Cài đặt

Sử dụng Supabase CLI cho local development:

<CodeBlock language="powershell" code={`# Cài đặt Supabase CLI qua npm
npm install -g supabase

# Kiểm tra cài đặt
supabase --version

# Khởi tạo Supabase trong project
supabase init`} />

Hoặc dùng Docker:

<CodeBlock language="powershell" code={`# Clone Supabase local
supabase start

# Dừng local Supabase
supabase stop`} />

## Sử dụng

### Kết nối từ ứng dụng

<CodeBlock language="powershell" code={`npm install @supabase/supabase-js`} />

### Client cơ bản

<CodeBlock language="powershell" code={`import { createClient } from '@supabase/supabase-js'

const supabase = createClient(
  process.env.SUPABASE_URL!,
  process.env.SUPABASE_ANON_KEY!
)

// Query data
const { data, error } = await supabase
  .from('articles')
  .select('*')
  .eq('published', true)
  .order('created_at', { ascending: false })

// Insert data
const { data, error } = await supabase
  .from('articles')
  .insert({ title: 'New Post', content: '...' })
  .select()`} />

### Local Development

<CodeBlock language="powershell" code={`# Start local Supabase
supabase start

# Lấy URLs và keys cho project
supabase status

# Tạo migration mới
supabase migration new create_profiles

# Apply migrations
supabase db push

# Xem Studio UI
# http://localhost:54323`} />

## Lưu ý

Supabase Local cần Docker Desktop để chạy. Free plan gồm 2 database projects, 500MB database, và 50MB file storage. Dùng Row Level Security (RLS) để bảo vệ data — mặc định tất cả tables đều private.
