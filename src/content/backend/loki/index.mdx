## Tổng quan

<Note type="info">
**Grafana Loki** là hệ thống log aggregation được thiết kế bởi Grafana Labs. Khác với Elasticsearch, Loki không index nội dung log mà chỉ index metadata (labels), giúp tiết kiệm chi phí lưu trữ và băng thông. Loki tích hợp chặt chẽ với Grafana và Prometheus.
</Note>

## Cài đặt

Chạy Loki với Docker:

<CodeBlock language="powershell" code={`docker run -d --name=loki -p 3100:3100 grafana/loki:latest`} />

Cài đặt Promtail (log collector):

<CodeBlock language="powershell" code={`docker run -d --name=promtail -v /var/log:/var/log grafana/promtail:latest -config.file=/etc/promtail/promtail.yaml`} />

## Truy vấn với LogQL

<CodeBlock language="powershell" code={`{job="nginx"} |= "error"
{job="nginx"} |= "error" != "timeout"
{job="api"} |= "500" | json | status >= 500`} />

Ví dụ truy vấn qua curl:

<CodeBlock language="powershell" code={`curl -G -s "http://localhost:3100/loki/api/v1/query_range" --data-urlencode "query={job=\\"nginx\\"} |= \\"error\\"" --data-urlencode "start=$(date -d '1 hour ago' +%s)" --data-urlencode "end=$(date +%s)"`} />

## Xác minh

<CodeBlock language="powershell" code={`curl http://localhost:3100/ready`} />

## Lưu ý

<Note type="tip">
Sử dụng LogQL label matchers để lọc logs nhanh hơn. Cấu trúc label tốt ngay từ đầu sẽ giúp truy vấn hiệu quả hơn nhiều lần.
</Note>
