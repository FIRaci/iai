## Tổng quan

<Note type="info">
**TypeORM** là ORM mã nguồn mở cho TypeScript và JavaScript, hỗ trợ SQL databases (PostgreSQL, MySQL, SQLite, MS SQL Server). TypeORM hỗ trợ cả Active Record và Data Mapper patterns, với tính năng migrations, relations, eager/lazy loading, và query builder.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install typeorm reflect-metadata @types/node
npm install pg`} />

## Entity và Relations

<CodeBlock language="javascript" code={`import { Entity, PrimaryGeneratedColumn, Column, OneToMany } from "typeorm";

@Entity()
export class User {
  @PrimaryGeneratedColumn()
  id: number;

  @Column()
  name: string;

  @OneToMany(() => Post, (post) => post.user)
  posts: Post[];
}`} />

## Query Builder

<CodeBlock language="javascript" code={`const users = await dataSource
  .getRepository(User)
  .createQueryBuilder("user")
  .leftJoinAndSelect("user.posts", "post")
  .where("user.name LIKE :name", { name: "%Alice%" })
  .orderBy("user.id", "DESC")
  .getMany();`} />

## Xác minh

<CodeBlock language="powershell" code={`npx typeorm migration:run -d ormconfig.ts`} />

## Lưu ý

<Note type="tip">
Dùng Data Mapper pattern (Repository pattern) cho ứng dụng phức tạp, Active Record cho đơn giản. Chạy migration trước khi deploy để tránh schema mismatch.
</Note>
