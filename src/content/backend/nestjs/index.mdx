## Tổng quan

<Note type="info">
**NestJS** là progressive Node.js framework cho xây dựng server-side applications hiệu quả. NestJS sử dụng TypeScript, kết hợp concepts từ Angular (modules, decorators, dependency injection) với flexibility của Node.js. Hỗ trợ GraphQL, WebSockets, microservices, và nhiều database integrations.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm i -g @nestjs/cli
nest new my-project`} />

## Module và Controller

<CodeBlock language="javascript" code={`import { Module } from "@nestjs/common";
import { AppController } from "./app.controller";

@Module({
  controllers: [AppController],
  providers: [AppService],
})
export class AppModule {}`} />

<CodeBlock language="javascript" code={`import { Controller, Get } from "@nestjs/common";

@Controller("users")
export class UsersController {
  @Get()
  findAll() {
    return [{ id: 1, name: "Alice" }];
  }
}`} />

## Chạy ứng dụng

<CodeBlock language="powershell" code={`npm run start:dev`} />

## Xác minh

<CodeBlock language="powershell" code={`curl http://localhost:3000/users`} />

<Note type="tip">
Sử dụng `nest g resource` để generate CRUD resources nhanh chóng. NestJS hỗ trợ OpenAPI/Swagger tự động qua @nestjs/swagger.
</Note>
