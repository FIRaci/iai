## Tổng quan

<Note type="info">
**Hoppscotch** (trước đây là Postwoman) là API testing tool mã nguồn mở, chạy hoàn toàn trên trình duyệt. Hỗ trợ nhiều phương thức HTTP, WebSocket, SSE, GraphQL, và MQTT với giao diện hiện đại, nhẹ và không cần cài đặt.
</Note>

## Cài đặt

Có thể dùng bản online hoặc cài đặt local:

<CodeBlock language="powershell" code={`winget install hoppscotch`} />

Hoặc chạy Docker:

<CodeBlock language="powershell" code={`docker run -p 3000:3000 hoppscotch/hoppscotch`} />

## Sử dụng cơ bản

Sau khi mở Hoppscotch, tạo request:

<CodeBlock language="text" code={`
Method: GET
URL: https://api.example.com/users
Headers:
  Content-Type: application/json
  Authorization: Bearer token

Click Send để nhận response.
`} />

Ví dụ GraphQL request:

<CodeBlock language="graphql" code={`
query {
  users {
    id
    name
    email
  }
}
`} />

## Xác minh

<CodeBlock language="powershell" code={`curl http://localhost:3000`} />
