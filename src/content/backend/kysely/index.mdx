## Tổng quan

<Note type="info">
**Kysely** là SQL query builder TypeScript với kiểu an toàn tuyệt đối (type-safe). Không cần ORM phức tạp, Kysely cho phép bạn viết SQL thuần nhưng vẫn có autocomplete, type checking và prevent SQL injection ngay từ compile time.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install kysely`} />

## Sử dụng cơ bản

Kết nối database và thực hiện query:

<CodeBlock language="javascript" code={`
import { Kysely, PostgresDialect } from 'kysely';
import { Pool } from 'pg';

interface Database {
  users: {
    id: string;
    name: string;
    email: string;
    created_at: Date;
  };
}

const db = new Kysely<Database>({
  dialect: new PostgresDialect({
    pool: new Pool({
      connectionString: 'postgres://user:pass@localhost:5432/mydb',
    }),
  }),
});

const user = await db
  .insertInto('users')
  .values({ name: 'Alice', email: 'alice@test.com' })
  .returningAll()
  .executeTakeFirst();

const users = await db
  .selectFrom('users')
  .select(['id', 'name', 'email'])
  .where('email', 'like', '%@test.com')
  .orderBy('created_at', 'desc')
  .execute();
`} />

## Xác minh

<CodeBlock language="powershell" code={`node queries.js`} />

</Note>
