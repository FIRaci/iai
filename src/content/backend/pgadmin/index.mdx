## Tổng quan

<Note type="info">
**pgAdmin** là công cụ quản trị PostgreSQL dạng web GUI phổ biến nhất. pgAdmin hỗ trợ query editor với syntax highlighting, server monitoring, backup/restore, và graphical explain plans. pgAdmin chạy như web application, có thể deploy local hoặc trên server.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`winget install pgadmin`} />

Hoặc chạy với Docker:

<CodeBlock language="powershell" code={`docker run -d --name pgadmin -p 5050:80 -e PGADMIN_DEFAULT_EMAIL=admin@test.com -e PGADMIN_DEFAULT_PASSWORD=admin dpage/pgadmin4`} />

## Kết nối đến server

Mở trình duyệt tại `http://localhost:5050`, đăng nhập, sau đó:

- Right-click Servers > Register > Server
- Name: Nhập tên kết nối
- Connection tab: Host, Port, Database, Username, Password

## Xác minh

<CodeBlock language="powershell" code={`curl http://localhost:5050/misc/ping`} />

## Lưu ý

<Note type="tip">
Dùng Query Tool (Alt+Shift+Q) để chạy SQL queries. Explain (EXPLAIN ANALYZE) giúp debug performance issues trực quan qua đồ thị.
</Note>
