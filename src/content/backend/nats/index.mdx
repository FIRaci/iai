## Tổng quan

<Note type="info">
**NATS** là messaging system nhẹ, hiệu suất cao được viết bằng Go. NATS hỗ trợ pub/sub, request-reply, và load-balanced queue groups. Với độ trễ dưới millisecond và throughput hàng triệu messages/giây, NATS phù hợp cho IoT, microservices, và edge computing.
</Note>

## Cài đặt

Chạy NATS server với Docker:

<CodeBlock language="powershell" code={`docker run -d --name nats -p 4222:4222 nats:latest`} />

Cài đặt NATS client cho Node.js:

<CodeBlock language="powershell" code={`npm install nats`} />

## Pub/Sub cơ bản

<CodeBlock language="javascript" code={`import { connect } from "nats";
const nc = await connect({ servers: "localhost:4222" });

// Subscriber
const sub = nc.subscribe("orders.created");
(async () => {
  for await (const msg of sub) {
    console.log(JSON.parse(msg.data));
  }
})();

// Publisher
nc.publish("orders.created", JSON.stringify({ id: 1, item: "book" }));`} />
