## Tổng quan

<Note type="info">
**Lucia** là thư viện authentication cho TypeScript/JavaScript, hỗ trợ nhiều framework như Next.js, SvelteKit, Astro, và Hono. Lucia cung cấp session management, OAuth integration, và database adapter cho PostgreSQL, MySQL, SQLite, MongoDB.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install lucia`} />

Cài đặt adapter cho database:

<CodeBlock language="powershell" code={`npm install @lucia-auth/adapter-postgresql`} />

## Sử dụng cơ bản

<CodeBlock language="javascript" code={`import { Lucia } from "lucia";
import { pg } from "@lucia-auth/adapter-postgresql";

const lucia = new Lucia(new pg(pool), {
  sessionCookie: {
    attributes: { secure: true }
  }
});

// Create session
const session = await lucia.createSession(userId, {});
// Validate session
const { user, session } = await lucia.validateSession(sessionId);`} />
