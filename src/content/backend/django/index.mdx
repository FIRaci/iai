## Tổng quan

Django là web framework Python "batteries-included" với ORM, admin panel, authentication, và routing tích hợp sẵn. Django theo mô hình MTV (Model-Template-View) và phù hợp cho các ứng dụng web từ nhỏ đến enterprise.

## Cài đặt

Cài đặt qua pip trong virtual environment:

<CodeBlock language="powershell" code={`# Tạo virtual environment
python -m venv venv
.\venv\Scripts\Activate

# Cài đặt Django
pip install django

# Kiểm tra cài đặt
django-admin --version`} />

## Sử dụng

### Tạo project Django

<CodeBlock language="powershell" code={`# Khởi tạo project
django-admin startproject mysite
cd mysite

# Run migrations đầu tiên
python manage.py migrate

# Chạy development server
python manage.py runserver`} />

### Tạo app

<CodeBlock language="powershell" code={`# Tạo app mới
python manage.py startapp blog

# Cấu trúc app
blog/
├── migrations/
├── admin.py
├── apps.py
├── models.py
├── views.py
├── urls.py
└── tests.py`} />

### Model và Migration

<CodeBlock language="powershell" code={`# blog/models.py
from django.db import models

class Article(models.Model):
    title = models.CharField(max_length=200)
    content = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.title

# Tạo và chạy migration
python manage.py makemigrations
python manage.py migrate`} />

### Tạo superuser

<CodeBlock language="powershell" code={`python manage.py createsuperuser`} />

## Lưu ý

Luôn dùng virtual environment cho mỗi project Django. Nhớ thêm app mới vào `INSTALLED_APPS` trong `settings.py`. Set `DEBUG=False` trong production. Dùng `python manage.py check --deploy` để kiểm tra security settings.
