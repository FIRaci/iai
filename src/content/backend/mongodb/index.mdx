## Tổng quan

<Note type="info">
**MongoDB** là NoSQL database phổ biến nhất, lưu trữ dữ liệu dạng JSON-like documents (BSON). MongoDB hỗ trợ indexing, aggregation pipeline, replication, và sharding, phù hợp với ứng dụng cần schema linh hoạt và scale ngang.
</Note>

## Cài đặt

Cài đặt MongoDB Community Server:

<CodeBlock language="powershell" code={`winget install MongoDB.Server`} />

Hoặc chạy với Docker:

<CodeBlock language="powershell" code={`docker run -d --name mongodb -p 27017:27017 mongo:7`} />

## CRUD cơ bản với Node.js

<CodeBlock language="javascript" code={`import { MongoClient } from "mongodb";
const client = new MongoClient("mongodb://localhost:27017");
await client.connect();
const db = client.db("myapp");
const users = db.collection("users");

await users.insertOne({ name: "Alice", age: 30 });
const result = await users.findOne({ name: "Alice" });
await users.updateOne({ name: "Alice" }, { $set: { age: 31 } });
await users.deleteOne({ name: "Alice" });`} />

## Xác minh

</Note>
