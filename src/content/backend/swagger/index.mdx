## Tổng quan

<Note type="info">
**Swagger** là bộ công cụ mã nguồn mở cho RESTful API documentation và development. Swagger UI render OpenAPI spec thành web page tương tác, cho phép test API endpoints trực tiếp từ trình duyệt. Swagger Editor hỗ trợ viết và validate OpenAPI spec real-time.
</Note>

## Cài đặt

Chạy Swagger UI với Docker:

<CodeBlock language="powershell" code={`docker run -d --name swagger-ui -p 8080:8080 -e SWAGGER_JSON=/spec/openapi.yaml -v ${PWD}:/spec swaggerapi/swagger-ui`} />

Cài đặt Swagger Editor:

<CodeBlock language="powershell" code={`docker run -d --name swagger-editor -p 8081:8080 swaggerapi/swagger-editor`} />

## Tích hợp với Express

<CodeBlock language="javascript" code={`import swaggerUi from "swagger-ui-express";
import swaggerSpec from "./openapi.json" with { type: "json" };

app.use("/api-docs", swaggerUi.serve, swaggerUi.setup(swaggerSpec));
console.log("Swagger UI at http://localhost:3000/api-docs");`} />

## Xác minh

<CodeBlock language="powershell" code={`curl http://localhost:8080`} />

<Note type="tip">
Dùng Swagger UI để developers và stakeholders có thể test API mà không cần curl. Kết hợp với OpenAPI code generation để đồng bộ code và spec.
</Note>
