## Tổng quan

<Note type="info">
**Apache Kafka** là nền tảng distributed event streaming mã nguồn mở, được phát triển bởi LinkedIn và Apache Software Foundation. Kafka cho phép publish, subscribe, lưu trữ và xử lý luồng dữ liệu real-time với độ tin cậy cao, thường dùng trong microservices và data pipeline.
</Note>

## Cài đặt

Cài Kafka qua Docker:

<CodeBlock language="powershell" code={`docker run -d -p 9092:9092 ^
  --name broker ^
  apache/kafka:latest`} />

Hoặc dùng winget:

<CodeBlock language="powershell" code={`winget install Apache.Kafka`} />

## Sử dụng cơ bản

Produce và consume messages với Node.js:

<CodeBlock language="javascript" code={`
import { Kafka } from 'kafkajs';

const kafka = new Kafka({
  clientId: 'my-app',
  brokers: ['localhost:9092'],
});

const producer = kafka.producer();
await producer.connect();
await producer.send({
  topic: 'orders',
  messages: [{ value: 'New order created' }],
});

const consumer = kafka.consumer({ groupId: 'order-group' });
await consumer.connect();
await consumer.subscribe({ topic: 'orders' });
await consumer.run({
  eachMessage: async ({ message }) => {
    console.log(message.value.toString());
  },
});
`} />
