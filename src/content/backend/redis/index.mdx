## Tổng quan

<Note type="info">
**Redis** là kho dữ liệu in-memory mã nguồn mở, hoạt động như database, cache, và message broker. Hỗ trợ cấu trúc dữ liệu phong phú: strings, hashes, lists, sets, sorted sets với TTL tự động.
</Note>

## Cài đặt

### Cài đặt qua winget

<CodeBlock language="powershell" code={`winget install Redis.Redis`} />

### Cài đặt qua Chocolatey (thay thế)

<CodeBlock language="powershell" code={`choco install redis-64`} />

### Kiểm tra cài đặt

<CodeBlock language="powershell" code={`redis-cli --version`} />

### Khởi động Redis server

<CodeBlock language="powershell" code={`# Khởi động service
Start-Service -Name "Redis"

# Kiểm tra status
Get-Service -Name "Redis"`} />

## Sử dụng

### Kết nối với Redis CLI

<CodeBlock language="powershell" code={`redis-cli`} />

### Các lệnh cơ bản

<CodeBlock language="bash" code={`# SET & GET
SET mykey "Hello Redis"
GET mykey

# Với TTL (hết hạn sau 60 giây)
SET session:123 "user_data" EX 60

# Làm việc với lists
LPUSH tasks "task1"
LPUSH tasks "task2"
LRANGE tasks 0 -1

# Làm việc với hashes
HSET user:1 name "Alice" age "30"
HGETALL user:1`} />

### Kết nối từ ứng dụng Node.js

<CodeBlock language="javascript" code={`import { createClient } from 'redis';

const client = createClient();
await client.connect();

await client.set('key', 'value');
const value = await client.get('key');
console.log(value); // 'value'`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`# Kiểm tra version
redis-cli --version

# Ping Redis server
redis-cli ping
# Kết quả: PONG

# Kiểm tra service
Get-Service -Name "Redis" | Format-Table Status, Name`} />

## Lưu ý

<Note type="warning">
Redis lưu dữ liệu trong RAM. Dùng lệnh `SAVE` hoặc `BGSAVE` để persist xuống disk. Cấu hình trong file `redis.conf`: `save 900 1` (save mỗi 15 phút nếu có ít nhất 1 key thay đổi).
</Note>

<Note type="tip">
Dùng RedisInsight (GUI tool miễn phí của Redis) để visualize và debug dữ liệu.
</Note>
