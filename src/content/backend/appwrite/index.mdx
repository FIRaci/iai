## Tổng quan

<Note type="info">
**Appwrite** là nền tảng Backend-as-a-Service (BaaS) mã nguồn mở, cung cấp APIs sẵn sàng cho authentication, database, storage, serverless functions và nhiều hơn nữa. Triển khai qua Docker giúp bạn tự chủ hoàn toàn dữ liệu.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`docker run -it --rm ^
  --volume //var/run/docker.sock:/var/run/docker.sock ^
  --volume \"%CD%/appwrite\":/usr/src/code/appwrite.console.json ^
  --env APPWRITE_OPEN_SOURCE_KEY=your-key ^
  appwrite/appwrite:latest`} />

## Sử dụng cơ bản

Khởi tạo Appwrite SDK và tạo user:

<CodeBlock language="javascript" code={`
import { Client, Account } from 'appwrite';

const client = new Client()
  .setEndpoint('http://localhost/v1')
  .setProject('your-project-id');

const account = new Account(client);
const user = await account.create(
  'unique()',
  'user@example.com',
  'securepassword',
  'User Name'
);
console.log(user.$id);
`} />

<CodeBlock language="powershell" code={`npm install appwrite`} />
