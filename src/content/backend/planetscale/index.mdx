## Tổng quan

<Note type="info">
**PlanetScale** là serverless MySQL-compatible database platform sử dụng Vitess (sharding engine). PlanetScale cung cấp database branching với deploy requests workflow, giúp quản lý schema changes an toàn. PlanetScale có free tier và hỗ trợ connection pooling, điểm phục vụ trên toàn cầu.
</Note>

## Kết nối

Cài đặt PlanetScale CLI:

<CodeBlock language="powershell" code={`npm i -g @planetscale/database`} />

Kết nối từ ứng dụng:

<CodeBlock language="javascript" code={`import { connect } from "@planetscale/database";
const conn = connect({
  url: process.env.DATABASE_URL,
});
const results = await conn.execute("SELECT 1 AS test");
console.log(results.rows);`} />

## Database Branches

- **main branch**: Production database
- **feature branch**: Schema changes, không ảnh hưởng production

<CodeBlock language="powershell" code={`pscale branch create my-db feature-branch
pscale deploy-request create my-db feature-branch`} />

## Lưu ý

<Note type="tip">
PlanetScale không hỗ trợ foreign key constraints. Dùng deploy requests để review schema changes trước khi merge vào production branch.
</Note>
