## Tổng quan

<Note type="info">
**Pico SDK** là software development kit cho Raspberry Pi Pico với chip RP2040. Cung cấp C SDK với đầy đủ thư viện cho GPIO, I2C, SPI, UART và PIO.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install pico-sdk`} />
