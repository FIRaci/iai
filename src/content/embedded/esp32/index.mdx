## Tổng quan

<Note type="info">
**ESP32** là vi điều khiển tích hợp WiFi và Bluetooth LE, lý tưởng cho IoT projects. Mạnh mẽ hơn Arduino với dual-core processor.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install esp32`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
