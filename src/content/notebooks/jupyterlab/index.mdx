## Tổng quan

<Note type="info">
**JupyterLab** là next-generation web-based UI cho Jupyter notebooks. Hỗ trợ code, data, visualizations trong một environment.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install jupyterlab`} />

## Sử dụng

<CodeBlock language="powershell" code={`jupyter lab`} />

Mở `http://localhost:8888` để sử dụng.
