## Tổng quan

<Note type="info">
**JetBrains Junie** là AI coding agent từ JetBrains. Không chỉ suggest code — Junie có thể tự động thực hiện tasks phức tạp: refactor, fix bugs, viết tests.
</Note>

## Cài đặt

Cài extension **Junie** từ JetBrains Marketplace trong IDE.

## Sử dụng

1. Mở IntelliJ IDEA / PyCharm / WebStorm
2. Cài Junie plugin
3. Sign in với JetBrains account
4. Giao task cho Junie qua chat panel

## Tasks Junie có thể làm

- **Refactoring:** Tự động refactor code theo best practices
- **Bug fixing:** Phân tích và fix bugs
- **Test generation:** Viết unit tests
- **Code review:** Review code và suggest improvements

## Tips

<Note type="tip">
**IDE integration:** Junie hiểu project structure, dependencies, và code conventions của bạn.
</Note>

<Note type="tip">
**Language support:** Hỗ trợ Java, Kotlin, Python, JavaScript, TypeScript, Go, và nhiều ngôn ngữ khác.
</Note>
