## Tổng quan

<Note type="info">
**AWS Kiro** là AI-powered IDE từ Amazon, tích hợp sâu với AWS services. Giúp developers viết, test, và deploy code trên AWS nhanh hơn.
</Note>

## Cài đặt

Tải từ [aws.amazon.com/kiro](https://aws.amazon.com/kiro)

## Sử dụng

1. Mở Kiro IDE
2. Sign in với AWS account
3. Tạo project mới hoặc mở existing
4. Sử dụng AI assistant để generate code

## Tính năng chính

- **AWS-aware:** Hiểu AWS services và best practices
- **Code generation:** Tạo code từ natural language
- **Debug assistance:** AI giúp tìm và fix bugs
- **Deployment:** Deploy trực tiếp lên AWS từ IDE

## Tips

<Note type="tip">
**IAM integration:** Tự động gợi ý IAM policies phù hợp.
</Note>

<Note type="warning">
Yêu cầu AWS account. Có thể incur charges khi deploy.
</Note>
