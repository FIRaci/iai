## Tổng quan

<Note type="info">
**Amazon Q Developer** là AI coding assistant từ AWS. Bao gồm IDE extension, CLI tool, và console integration. Hỗ trợ code generation, debugging, và AWS optimization.
</Note>

## Cài đặt

### VS Code Extension

Cài **Amazon Q** từ VS Code Marketplace.

### CLI

<CodeBlock language="powershell" code={`# Cài qua AWS CLI
aws qdeveloper --help

# Hoặc tải từ AWS Console`} />

## Sử dụng

### Trong IDE

1. Mở VS Code / JetBrains IDE
2. Sign in với AWS Builder ID
3. Sử dụng chat panel để hỏi về code
4. Accept AI suggestions

### CLI

<CodeBlock language="powershell" code={`# Transform code
amazon-q transform

# Generate documentation
amazon-q doc generate`} />

## Tính năng

- **Code transformation:** Upgrade Java versions tự động
- **Security scanning:** Phát hiện vulnerabilities
- **Code explanation:** Giải thích code bằng natural language
- **Unit test generation:** Tự động viết tests
