## Tổng quan

<Note type="info">
**Microsoft AI Dev Gallery** là collection của Microsoft chứa các AI-powered developer tools và extensions. Bao gồm GitHub Copilot, Azure AI services, PowerToys AI, và nhiều tools khác.
</Note>

## Truy cập

Truy cập [developer.microsoft.com/ai](https://developer.microsoft.com/ai)

## Tools trong Gallery

### GitHub Copilot
AI pair programmer trong VS Code và JetBrains IDEs.

### Azure AI Services
Cloud AI services: vision, speech, language, decision.

### PowerToys
Utility suite với AI-enhanced features:
- **PowerToys Run:** Quick launcher với AI suggestions
- **Text Extractor:** OCR từ screen captures
- **Color Picker:** AI color matching

### Visual Studio IntelliCode
AI-assisted code completion trong Visual Studio.

### Windows Copilot
AI assistant built vào Windows 11.

## Sử dụng

### Cài đặt từ Gallery

1. Truy cập Microsoft AI Dev Gallery
2. Chọn tool bạn cần
3. Download và cài đặt
4. Sign in với Microsoft account

### VS Code + Copilot

<CodeBlock language="powershell" code={`# Cài Copilot extension
code --install-extension GitHub.copilot

# Hoặc qua VS Code Extensions view:
# Ctrl+Shift+X → search "Copilot" → Install`} />

## Tips

<Note type="tip">
**Microsoft ecosystem:** Nếu dùng Azure + VS Code + Windows, các tools này integrate rất tốt với nhau.
</Note>
