## Tổng quan

<Note type="info">
**GitHub Copilot Workspace** là cloud-based development environment từ GitHub. Khác với Copilot extension — Workspace là môi trường dev hoàn chỉnh trong browser với AI built-in.
</Note>

## Sử dụng

1. Truy cập [github.com/copilot/workspace](https://github.com/copilot/workspace)
2. Chọn repository
3. Tạo workspace mới
4. Code với AI assistance trong browser

## Khác biệt với Copilot thông thường

| Feature | Copilot Extension | Copilot Workspace |
|---------|-------------------|-------------------|
| Environment | Local IDE | Cloud browser |
| Setup | Cần cài IDE | Zero setup |
| Collaboration | Individual | Shared workspace |
| AI | Code completion | Full task planning |

## Tips

<Note type="tip">
**GitHub integration:** Truy cập trực tiếp vào repos, issues, và PRs.
</Note>

<Note type="tip">
**Task-oriented:** Mô tả task bằng natural language, Workspace tự plan và implement.
</Note>
