## Tổng quan

<Note type="info">
**Chrome DevTools AI** là các tính năng AI được tích hợp vào Chrome DevTools. Giúp developers debug, optimize, và develop web apps nhanh hơn với AI assistance.
</Note>

## Tính năng AI trong DevTools

### Issues Panel
AI tự động phát hiện và gợi ý fix cho:
- Console errors
- Network failures
- Performance bottlenecks
- Accessibility issues

### Performance Panel
AI phân tích performance traces và gợi ý optimizations.

### Lighthouse
AI-powered audits cho:
- Performance
- Accessibility
- SEO
- Best practices

### Elements Panel
AI suggest CSS improvements và layout fixes.

## Sử dụng

### Mở DevTools

<CodeBlock language="powershell" code={`# Phím tắt
F12
# Hoặc
Ctrl + Shift + I
# Hoặc
Right-click → Inspect`} />

### Sử dụng Issues Panel

1. Mở DevTools
2. Tab **Issues**
3. Xem AI-detected problems
4. Click vào issue để xem suggested fix

### Lighthouse Audit

1. Tab **Lighthouse**
2. Chọn categories to audit
3. Click **Analyze page load**
4. Xem AI-generated recommendations

## Tips

<Note type="tip">
**Experiment flags:** Một số AI features cần bật qua `chrome://flags`.
</Note>

<Note type="tip">
**Canary channel:** Chrome Canary có AI features mới nhất trước khi release stable.
</Note>
