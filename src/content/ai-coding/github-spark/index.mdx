## Tổng quan

<Note type="info">
**GitHub Spark** là AI-powered app builder từ GitHub. Cho phép tạo ứng dụng web nhỏ từ mô tả natural language — không cần coding experience.
</Note>

## Sử dụng

1. Truy cập [githubnext.com/projects/spark](https://githubnext.com/projects/spark)
2. Mô tả app bạn muốn tạo
3. AI generate app trong browser
4. Preview và deploy

## Use cases

- **Internal tools:** Dashboard, form, data viewer
- **Prototypes:** Quick proof-of-concept apps
- **Learning:** Hiểu cách web apps hoạt động
- **Automation:** Simple workflow tools

## Tips

<Note type="tip">
**No-code friendly:** Perfect cho non-technical users muốn tạo apps nhanh.
</Note>

<Note type="tip">
**GitHub hosting:** Apps được host trên GitHub infrastructure.
</Note>
