## Tổng quan

Continue là open-source AI coding assistant cho VS Code và JetBrains IDE. Hỗ trợ đa dạng LLM backends: Ollama (local), OpenAI, Anthropic, Google Gemini, và nhiều providers khác. Có chat trong IDE, tab autocomplete, và inline code editing.

## Cài đặt

Cài trực tiếp từ VS Code Marketplace hoặc dùng CLI.

<CodeBlock language="powershell" code={`
# Cài từ CLI (VS Code)
code --install-extension continue.continue

# Hoặc cài thủ công từ marketplace
# Mở VS Code > Extensions > Search 'Continue'
`} />

## Sử dụng

1. Mở VS Code, cài Continue extension
2. Mở Continue panel (Ctrl+Shift+J hoặc icon bên trái)
3. Cấu hình LLM provider trong `~/.continue/config.json`
4. Dùng Tab autocomplete khi gõ code
5. Bôi đen code + Ctrl+L để gửi vào chat
6. Dùng Ctrl+I để inline edit code trực tiếp

<CodeBlock language="json" code={`
{
  "models": [
    {
      "title": "Ollama",
      "provider": "ollama",
      "model": "codellama:7b"
    }
  ],
  "tabAutocompleteModel": {
    "title": "Starcoder2",
    "provider": "ollama",
    "model": "starcoder2:3b"
  }
}
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`
# Kiểm tra extension đã cài
code --list-extensions | findstr continue

# Mở Continue panel
# Ctrl+Shift+P > Continue: Toggle Continue
`} />

## Lưu ý

Tab autocomplete yêu cầu model local nhẹ như Starcoder2 3B hoặc DeepSeek Coder. Nếu dùng Ollama, đảm bảo Ollama đang chạy nền. Trên Windows, nên dùng Continue với WSL nếu model yêu cầu GPU. Phím tắt Ctrl+Shift+J mở/tắt Continue panel nhanh nhất.
