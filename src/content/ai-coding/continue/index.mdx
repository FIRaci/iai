## Tổng quan

<Note type="info">
**Continue** là open-source AI coding assistant cho VS Code và JetBrains. Hỗ trợ nhiều LLM backends: Ollama, OpenAI, Anthropic.
</Note>

## Cài đặt

Cài extension từ VS Code Marketplace: **Continue**

## Sử dụng

1. Mở VS Code
2. Cài Continue extension
3. Cấu hình LLM provider trong `config.json`
4. Bắt đầu chat và code completion
