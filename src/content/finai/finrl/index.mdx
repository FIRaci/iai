## Tổng quan

<Note type="info">
**FinRL** là framework cho financial reinforcement learning, kết hợp deep RL với financial data. Hỗ trợ stock trading và portfolio management.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install finrl`} />
