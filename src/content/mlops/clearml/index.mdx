## Tổng quan

<Note type="info">
**ClearML** là end-to-end MLOps platform: experiment tracking, orchestration, và model serving. Open-source và self-hostable.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`pip install clearml`} />

## Sử dụng

<CodeBlock language="python" code={`from clearml import Task
task = Task.init(project_name="my-project", task_name="training")
task.connect({"learning_rate": 0.01, "epochs": 10})
# Training code...
task.close()`} />
