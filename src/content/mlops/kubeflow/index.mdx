## Tổng quan

Kubeflow là ML platform chạy trên Kubernetes, giúp xây dựng và triển khai end-to-end ML pipelines. Tích hợp với Jupyter Notebooks, hỗ trợ distributed training với TFJob và PyTorchJob, hyperparameter tuning với Katib, và model serving với KServe.

## Cài đặt

Có thể cài trên cluster thật hoặc local với minikube/kind.

<CodeBlock language="powershell" code={`
# Option 1: Local với minikube
minikube start --cpus 4 --memory 8192
kubectl apply -k github.com/kubeflow/manifests.git

# Option 2: Local với kind
kind create cluster --name kubeflow --config kind-config.yaml
kubectl apply -k github.com/kubeflow/manifests.git

# Option 3: Triển khai trực tiếp
kubectl apply -k github.com/kubeflow/manifests.git
kubectl get pods -n kubeflow
`} />

## Xác minh cài đặt

<CodeBlock language="powershell" code={`

## Sử dụng

1. **Central Dashboard:** Giao diện trung tâm quản lý pipelines, notebooks, experiments. Mở http://localhost:8080 sau khi port-forward.
2. **Notebook Server:** Tạo Jupyter Notebook server với custom image và resource limits từ Central Dashboard.
3. **Pipelines:** Xây dựng ML pipeline bằng DSL Python, tải lên và chạy trên Kubeflow.
4. **Katib:** Tuning hyperparameters tự động với các thuật toán như Bayesian optimization.
5. **KServe:** Deploy model lên production với auto-scaling và canary deployments.

## Lưu ý

Kubeflow yêu cầu tối thiểu 8GB RAM cho local test. Nên dùng managed Kubernetes (EKS, GKE, AKS) cho production. Pipeline caching giúp tái sử dụng kết quả nếu input không đổi. Central Dashboard hỗ trợ multi-user với RBAC thông qua Istio.
