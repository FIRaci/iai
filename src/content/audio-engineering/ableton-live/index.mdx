## Tổng quan

<Note type="info">
**Ableton Live** là DAW và live performance tool cho music production. Nổi tiếng với session view và warping engine.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install ableton-live`} />

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux

- Hiệu suất cao và khả năng mở rộng linh hoạt
