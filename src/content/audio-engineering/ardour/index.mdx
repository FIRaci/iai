## Tổng quan

<Note type="info">
**Ardour** là DAW mã nguồn mở chuyên nghiệp cho recording, editing và mixing. Hỗ trợ multi-track và automation.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install ardour`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
