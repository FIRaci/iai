## Tổng quan

<Note type="info">
**LMMS** là phần mềm music production mã nguồn mở, thay thế miễn phí cho FL Studio. Hỗ trợ VST plugins và built-in synthesizers.
</Note>

## Cài đặt

<CodeBlock language="powershell" code={`npm install lmms`} />

## Tính năng chính

- Tích hợp đa nền tảng, hỗ trợ Windows, macOS và Linux
- Cộng đồng lớn mạnh với nhiều tài nguyên học tập
- Hiệu suất cao và khả năng mở rộng linh hoạt
- Tương thích với các công cụ và workflow phổ biến
